# 캔버스 배포는 어떻게 하나요?

<hr />
<br />

1. 캔버스 배포는 배포관리 메뉴의 `유효성 검사`를 통해 배포 전 검사를 진행하고 이후 검사 성공시 `배포하기`를 클릭하여 CSP에 배포 할 수 있습니다.
2. 캔버스 상위 메뉴 `배포관리 > 유효성 검사`를 선택합니다.
3. 유효성 검사 모달에서 자격증명을 선택하고 `유효성 검사하기` 버튼을 클릭합니다.
4. 좌측 유효성 검사 메뉴에서 배포 테스트 진행 및 결과를 확인합니다.
5. 배포테스트 완료 이후 배포하고자 하는 `환경을 선택`하고 `배포하기`를 클릭하여 CSP에 배포할 수 있습니다.

:::tip

캔버스에 그린 클라우드 인프라를 CSP에 배포하기 위해서는 자격증명이 등록되어 있어야 합니다.

:::
