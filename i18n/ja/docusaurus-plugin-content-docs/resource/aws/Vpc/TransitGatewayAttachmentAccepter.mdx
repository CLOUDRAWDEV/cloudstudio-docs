## 리소스 개요

타 계정의 리소스 공유에 대한 허용여부를 결정하는 리소스

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### 연결 리소스

-   [**TransitGatewayAttachment**](https://docs.cstudio.app/docs/resource/aws/Vpc/TransitGatewayAttachment)

## 리소스 설정 값

-   `is_manual` : 연결할 Transit gateway Attachment ID를 직접 입력할지 여부 - `false`,`true`
-   `linked_transit_gateway_attachment_name` : 공유를 허용할 다른 Transit Gateway Attachment의 Name
