## 리소스 개요

서버 간 네트워크 접근 제어 및 관리를 할 수 있는 IP/Port 기반 필터링 방화벽 서비스입니다.

## 연관 리소스

### 상위 리소스

-   [**VPC**](https://docs.cstudio.app/docs/resource/ncloud/Networking/Vpc)

### 연결 리소스

-   [**Network Interface**](https://docs.cstudio.app/docs/resource/ncloud/Networking/NetworkInterface)
-   [**Auto Scaling**](https://docs.cstudio.app/docs/resource/ncloud/Compute/AutoScaling)

## 리소스 설정 값

-   `included_vpc_name` : ACG를 생성할 VPC 이름
-   `acg_description` : ACG에 대한 설명
-   `inbound_rule`
    -   `name` : inbound rule에 대한 이름
    -   `protocol` : 인바운드 트래픽의 프로토콜 - `TCP`, `UDP`, `ICMP`
    -   `rule_apply_method` : 인바운드 규칙 적용 방법 - `ip_block`, `source_name`
    -   `ip_block` : 인바운드 트래픽 목적지, CDIR 형식의 IP 대역
    -   `port_range` : 인바운드 규칙을 적용할 포트 범위, 1 ~ 65535의 범위(범위 설정시: 100-200)

-   `outbound_rule`
    -   `name` : outbound rule에 대한 이름
    -   `protocol` : 아웃바운드 트래픽의 프로토콜 - `TCP`, `UDP`, `ICMP`
    -   `rule_apply_method` : 아웃바운드 규칙 적용 방법 - `ip_block`, `source_name`
    -   `ip_block` : 인바운드 트래픽 목적지, CDIR 형식의 IP 대역
    -   `port_range` : 인바운드 규칙을 적용할 포트 범위, 1 ~ 65535의 범위(범위 설정시: 100-200)
