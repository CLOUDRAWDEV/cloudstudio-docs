## 아이디 찾기

1. 클라우드 스튜디오 로그인 페이지에서 **[아이디 찾기](https://cstudio.app/forgot/id)**를 클릭합니다.
2. 이메일 주소를 입력한 후 코드 전송을 클릭합니다. 예) `cloudstudio@cloudraw.kr`
3. 입력한 이메일 주소를 통해 전송된 `여섯 자리의 인증번호`를 확인합니다.
4. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 `확인`을 클릭합니다.
5. `다음` 버튼을 클릭합니다.
6. 이메일 주소로 전송된 아이디를 확인합니다.

## 비밀번호 찾기

1. 클라우드 스튜디오 로그인 페이지에서 **[비밀번호 찾기](https://cstudio.app/forgot/pw)**를 클릭합니다.
2. 이메일 주소를 입력한 후 코드 전송을 클릭합니다. 예) `cloudstudio@cloudraw.kr`
3. 입력한 이메일 주소를 통해 전송된 `여섯 자리의 인증번호`를 확인합니다.
4. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 `확인`을 클릭합니다.
5. `비밀번호 재설정`버튼을 클릭합니다.
6. 새 비밀번호를 입력합니다.
7. 확인을 위해 다시 한 번 새 비밀번호를 입력합니다.
8. `비밀번호 재설정` 버튼을 눌러 완료합니다.

:::info
비밀번호는 아래의 조건을 만족해야 합니다.

-   영문 대소문자, 숫자, 특수문자만을 사용
-   시작은 영문 소문자
-   8자리 이상 20자리 이내

:::
