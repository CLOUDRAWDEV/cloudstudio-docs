クレデンシャル管理ページでは、クレデンシャル報を表示して削除することができます。
登録したクレデンシャルを削除するには `以下の手順`に従っていただきます。

<div className={styles.imgWrap}>
    <img src={deleteCredential} />
</div>

<br />
<br />

1. `マイページ > クレデンシャル管理`をクリックします。
2. 設定で `三点リーダーアイコン`をクリックします。
3. `削除する`をクリックします。
4. 登録した`クレデンシャル名を入力`の後、削除するボタンをクリックして削除を完了します。

<br />

:::caution

クレデンシャルを削除するためには、事前に必ず OTP が登録されていなければなりません。

:::

<br />

:::tip

登録したクレデンシャルをもっと早く探すためにクレデンシャル検索機能を利用しましょう。

:::
