## リソース概要

ACM Certificate リソースを使用すると、Amazon Certificate Manager で認証書を要請及び管理することができます。

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

## 関連リソース

## リソース設定値

-   `enabled_import` : 証明書インポート
-   `certificate_body` : インポートする証明書の PEM-encoded
-   `certificate_private_key` : インポートする証明書の PEM-encoded プライベートキー
-   `certificate_chain` : インポートする証明書の PEM-encoded 証明書 chain
