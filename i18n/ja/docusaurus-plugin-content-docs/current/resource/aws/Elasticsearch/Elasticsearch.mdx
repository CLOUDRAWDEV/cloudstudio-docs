## リソース概要

AWS Elastic Search は分散検索及び分析エンジンで、迅速な検索及びデータ分析のために構築された検索サービスです。
Elastic Search は Apache Lucene 検索エンジンの上に構築されており、迅速かつ拡張可能なオープンソースソリューションです。
Elastic Search はデータ収集、ログ分析、モニタリングなど様々な用途で使用されます。
Elastic Search は Amazon VPC 内で実行することができ、AWS Lambda、Kinesis などと統合して使用することができます。
Elastic Search は AWS が提供するセキュリティ及びアクセス制御機能を提供し、Amazon CloudWatch と統合してモニタリング及びロギング機能を提供します。

## リソース設定値

-   `elasticsearch_version` : Elasticsearch Domain のバージョン
-   `availability_zone_count` : availability zone の数 - `2`,`3`
-   `tls_security_policy` : HTTPS endpoint に適用すべき TLS セキュリティポリシーの名前 - `Policy-Min-TLS-1-2-2019-07`, `Policy-Min-TLS-1-0-2019-07`
-   `data_node`
    -   `instance_type` : クラスター内のデータ node のインスタンスタイプ
    -   `instance_count` : クラスターのインスタンス数 - `1`~`80`
    -   `ebs_enabled` : EBS ボリュームがドメインのデータ node に接続されているかどうかの可否 - `true`, `false`
    -   `ebs_volume_type` : データ node に接続された EBS ボリュームタイプ - `standard`, `gp2`, `io1`
    -   `ebs_volume_size` : データ node に接続された EBS ボリュームのサイズ(GiB) - `1`~`16384`
-   `master_node`
    -   `master_node_enabled` : クラスターの専用メイン node を有効にする - `true`, `false`
    -   `master_instance_type` : クラスター内の専用メイン node のインスタンスタイプ - `r5.large.search`
    -   `master_node_count ` : クラスターの専用メイン node の数 - `3`, `5`
-   `custom_endpoint`
    -   `custom_endpoint_enabled` : ユーザーカスタムエンドポイントを有効にする - `true`, `false`
-   `network`
    -   `network_type` : ネットワークアクセスのタイプ - `vpc_access`, `public_access`
    -   `included_subnet_names` : 生成する Elasticsearch ドメインエンドポイントの VPC サブネット ID リスト
    -   `security_group_names` : Elasticsearch ドメインエンドポイントに適用する VPC Security Group ID のリスト
-   `fine_grained_access`
    -   `fine_grained_access_enabled` : 細分化されたアクセス制御を有効にする - `true`, `false`
    -   `master_user_type` : マスターユーザータイプ - `iam_arn`, `master_user`
    -   `iam_arn` : 基本ユーザー向け IAM ARN
-   `encryption`
    -   `enforce_https` : HTTPS リクエストの可否 - `true`, `false`
    -   `node_to_node_encryption_enabled` : node 間暗号化活性化の可否 - `true`, `false`
    -   `data_at_rest_encryption_enabled` : 保存データの暗号化活性化の可否 - `true`, `false`
-   `kms_key_name` : データの暗号化に使用する KMS キー名
