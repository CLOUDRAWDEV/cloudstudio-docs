## リソース概要

Amazon Dynamo DB は完璧に管理される NoSQL データベースサービスで、円滑な拡張性とともに迅速かつ予測可能な性能を提供します。
Amazon Dynamo DB を使用して、必要な量のデータを保存または取得できるデータベーステーブルを作成し、どのレベルの要求トラフィックも処理できます。
Amazon Dynamo DB はテーブルのデータとトラフィックを十分な数のサーバーに自動分散し、お客様が指定した要求容量と保存されたデータ量を処理しながらも一貫して速い性能を維持します。

## リソース設定値

-   `billing_mode` : DynamoDB の読み取り·書き込み処理量に対する料金賦課方式と容量管理方式 - `PROVISIONED`,`PAY_PER_REQUEST`
-   `read_capacity` : DynamoDB Table の読み取り単位数
-   `write_capacity` : DynamoDB Table の書き込み単位数
-   `hash_key` : ハッシュ（パーティション）キーとして使用するプロパティ
-   `hash_key_type` : ハッシュ（パーティション）キーとして使用するプロパティ - `S`, `N`, `B`
-   `range_key` :範囲（並び替え）キーで使用するプロパティ
-   `range_key_type` : 範囲（並び替え）キーで使用するプロパティ - `S`, `N`, `B`
-   `point_in_time_recovery_enabled` : 特定の時点リカバリオプションを有効にする - `true`, `false`
-   `server_side_encryption` : Server Side Encryption の名前
    -   `encryption_enabled` : AWS 管理型 KMS 顧客マスターキーを使用して保存データ暗号化を有効にするかどうかの可否 - `true`, `false`
