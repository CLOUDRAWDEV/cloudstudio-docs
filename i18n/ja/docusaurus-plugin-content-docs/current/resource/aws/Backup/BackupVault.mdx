## リソース概要

AWS Backup で Backup Vault はバックアップを保存して構成するコンテナです。
Backup Vault を生成するとき、このボルトにある一部のバックアップを暗号化する AWS KMS 暗号化キーを指定する必要があります。

## リソース設定値

-   `kms_key_name` : バックアップを保護するために使用される server-side 暗号キー
-   `lock_configuration` : AWS Backup vault lock 構成リソース
    -   `lock_configuration.enabled_lock` : vault lock 使用の可否 - `false`,`true`
        -   `lock_configuration.enabled_lock.lock_mode` : vault lock タイプ - `Governance`,`Compliance`
        -   `lock_configuration.enabled_lock.min_retention_days` : vault リカバリ地点を維持する最小保存期間 - `1~36500`
        -   `lock_configuration.enabled_lock.max_retention_days` : vault リカバリ地点を維持する最大保存期間 - `1~36500`
