## リソース概要

AWS WAF IPSet は Amazon Web Services の Web Application Firewall で使用する IP アドレスのリストです。
これにより、ユーザは定義した IP アドレスへのアクセス規則を生成し、管理することができます。
IPSet を活用すると、特定の IP アドレスからの要請を許可または遮断することで、ウェブアプリケーションのセキュリティを強化することができます。

## リソース設定値

-   `scope` : AWS CloudFront 配布用か、それともリージョンアプリケーション用かを指定 - `REGIONAL`, `CLOUDFRONT`
-   `ip_address_version` : IPV4 または IPV6 指定 - `IPV4`, `IPV6`
-   `addresses` : CIDR 表記法で 1 つ以上の IP アドレスまたは IP アドレスブロックを指定する文字列配列を入力
