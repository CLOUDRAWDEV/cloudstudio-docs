## リソース概要

AWS Service Catalog を使用すると、IT 管理者が承認された製品で構成されたポートフォリオを作成·管理してエンドユーザーに配布することができ、
エンド ユーザーは、パーソナライズされたポータルから必要な製品にアクセスできます。
一般的な製品には、AWS リソースを使用してデプロイされるサーバー、データベース、ウェブサイト、またはアプリケーションが含まれます。(例:Amazon EC2 インスタンスまたは Amazon RDS データベース)
どのユーザーが特定の製品にアクセスできるかを制御し、組織のビジネス標準のコンプライアンスを適用し、製品のライフサイクルを管理し、ユーザーが自信を持って製品を検索して開始できるようにすることができます。

## リソース設定値

-   `description` : ポートフォリオについての説明
-   `provider_name` : ポートフォリオを所有する個人または組織の名前
