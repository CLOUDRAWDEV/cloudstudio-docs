## リソース概要

AWS リソースとオンプレミスネットワークのリソースに安全にアクセスできるようにする管理型クライアントベースの VPN サービス

## 関連リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)
-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Vpc)

## リソース設定値

-   `description` : クライアント VPN エンドポイントの簡単な説明
-   `client_cidr_block` : クライアント I アドレスが割り当てられる IP アドレス範囲
-   `server_acm_certificate_name` : サーバ証明書の名前
-   `authentication` : 使用する認証方式
-   `authentication.option_types` : 使用する認証方式 - `certificate-authentication`,`directory-service-authentication`,`federated-authentication`
-   `connection_logging.enalbed` : クライアント接続に対するログの詳細を使用 - `false`,`true`
-   `transport_protocol` : TLS セッションで使用されるトランスポートプロトコル - `tcp`,`udp`
-   `enabled_split_tunnel` : スプリットトンネル活性化 - `false`,`true`
-   `vpn_port` : VPN ポート
-   `enabled_self_service_portal` : セルフサービスポータル活性化 - `false`,`true`
-   `session_timeout_hours` : セッションのタイムアウト時間 - `8`,`10`,`12`,`24`
-   `included_vpc_name` : VPC 名
-   `linked_security_group_names` : エンドポイントに適用するセキュリティグループ名
-   `linked_subnet_names` : 接続する Subnet 名
