## リソース概要

VPC とオンプレミスネットワークを接続するために使用できるサービス

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Vpc)
-   [**TransitGateway**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/TransitGateway)
-   [**TransitGatewayAttachmentAccepter**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/TransitGatewayAttachmentAccepter)

## リソース設定値

-   `linked_tranit_gateway_name` : VPC と接続する Transit Gateway 名
-   `is_manual` : 接続する VPC ID を直接入力するかどうかの可否 - `false`,`true`
-   `linked_vpc_name` : Transit Gateway を接続する VPC 名
-   `subnet_names` : Transit Gateway を接続する Subnet 名
