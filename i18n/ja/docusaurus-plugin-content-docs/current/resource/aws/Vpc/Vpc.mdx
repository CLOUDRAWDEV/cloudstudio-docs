## リソース概要

Amazon Virtual Private Cloud(Amazon VPC)を使用すると、リソースの配置、接続およびセキュリティを含む仮想ネットワーキング環境を完全に制御することができます。
AWS サービスコンソールで VPC を設定して開始してください。 次に、Amazon Elastic Compute Cloud(EC2)および Amazon Relational Database Service(RDS)インスタンスなどのリソースを VPC に追加します。
最後に VPC がアカウント、可用領域または AWS リージョンで相互に通信する方法を定義します。以下の例で、ネットワークトラフィックは各リージョン内の 2 つの VPC 間で共有されます。
