## リソース概要

Vpn Gateway は、Site-to-Site VPN 接続の Amazon 側にある VPN 集線器です。 仮想プライベートゲートウェイを生成して、Site-to-Site VPN 接続を生成する VPC に接続します。

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/)

-   [**Vpc Customer Gateway**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/CustomerGateway)

## リソース設定値

-   `linked_vpc_name` : VPN Gateway が含まれる VPC 名
-   `vpn_connection` : VPN Connection 名
    -   `linked_customer_gateway` : VPN Gateway と接続する Customer Gateway 名
    -   `static_routes_only` : ゲートウェイ間ルーティング時、static ルーティングのみを使用するかどうかを指定 - `true`, `false`
