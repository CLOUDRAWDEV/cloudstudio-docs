## リソース概要

Subnet は VPC の IP アドレス範囲です。特定のサブネットから EC2 インスタンスなどの AWS リソースを生成できます。

## リソース設定値

-   `included_vpc_name` : Subnet をデプロイする VPC 名
-   `cidr_block` : Subnet の IP 帯域(CIDR)
-   `availability_zone` : Subnet をデプロイする Availability Zone
-   `pip_enable` : Public IP 割り当て
