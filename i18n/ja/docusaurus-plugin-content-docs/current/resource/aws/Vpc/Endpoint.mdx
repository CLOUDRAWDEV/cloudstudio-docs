## リソース概要

お客様がサポートする AWS サービスおよび VPC エンドポイントサービスにプライベートで接続できるサービス

## リソース設定値

-   `service_range` : Endpoint サービスを使用するサービスのカテゴリ - `awsService`,`endpointService`
-   `aws_service_name` : Endpoint サービスを使用するサービスの名前
-   `vpc_endpoint_type` : Endpoint サービスを使用する他の Endpoint サービスの名前 - `gateway`, `interface`, `gatewayloadbalancer`
-   `included_vpc_name` : The name of VPC to use endpoint
-   `route_table_names` : endpoint と接続する 1 つ以上のパステーブル ID
-   `policy_access` : サービスアクセスを制御する endpoint に接続するポリシーアクセス範囲 - `Full`, `Custom`
