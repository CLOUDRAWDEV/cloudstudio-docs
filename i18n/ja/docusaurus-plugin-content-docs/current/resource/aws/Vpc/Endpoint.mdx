## 리소스 개요

고객이 지원하는 AWS 서비스 및 VPC 엔드포인트 서비스에 프라이빗하게 연결할 수 있는 서비스

## 연관 리소스

### 상위 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)

### 하위 리소스

-   [**Endpoint Service**](https://docs.cstudio.app/docs/resource/aws/Vpc/EndpointService)
-   [**Security Group**](https://docs.cstudio.app/docs/resource/aws/Vpc/SecurityGroup)
-   [**Subnet**](https://docs.cstudio.app/docs/resource/aws/Vpc/Subnet)

## 리소스 설정 값

-   `service_range` : Endpoint 서비스를 사용할 서비스의 범주 - `awsService`,`endpointService`
-   `aws_service_name` : Endpoint 서비스를 사용할 AWS 서비스의 이름
-   `vpc_endpoint_type` : Endpoint 서비스를 사용할 다른 endpoint 서비스의 이름 - `gateway`, `interface`, `gatewayloadbalancer`
-   `included_vpc_name` : 엔드포인트를 생성할 VPC 이름
-   `route_table_names` : endpoin와 연결할 하나 이상의 경로 테이블 ID
-   `policy_access` : 서비스 액세스를 제어하는 endpoint의 연결할 정책 액세스 범위 - `Full`, `Custom`
