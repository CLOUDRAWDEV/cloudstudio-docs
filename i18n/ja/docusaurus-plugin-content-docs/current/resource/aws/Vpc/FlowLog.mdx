## リソース概要

VPC FlowLog は VPC のネットワークインタフェースから送信され、受信される IP トラフィックに関する情報を収集できる機能です。
FlowLog データは Amazon CloudWatch Logs, Amazon S3 または Amazon Kinesis Data Firehose に掲示されることがあります。
FlowLog を生成すると構成したロググループ、バケット、または転送ストリームの FlowLog レコードを検索して表示できます。

## リソース設定値

-   `linked_log_destination_type` : ログデータを保存するリポジトリタイプ
-   `linked_log_destination_name` : ログデータを保存するリポジトリ名
-   `traffic_type` : 収集するトラフィックのタイプ
-   `included_vpc_name` : Flow Log を接続する VPC 名
