## リソース概要

AWS Transit Gateway は、中央ハブを通じて Amazon Virtual Private Cloud(VPC)とオンプレミスネットワークを接続します。
この接続は複雑なピアリング関係を排除してネットワークを簡素化します。Transit Gateway は高度に拡張可能なクラウドルーターとして機能します。

## リソース設定値

-   `bgp_asn` : ゲートウェイの BGP(Border Gateway Protocol)ASN(自律システム番号)
-   `ip_address` : ゲートウェイのインターネットルーティング可能な外部インターフェイスの固定 IP アドレス
