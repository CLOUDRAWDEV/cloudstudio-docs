## リソース概要

NAT Gateway は NAT(ネットワークアドレス変換)サービスです。
プライベートサブネットのインスタンスが VPC 外部のサービスに接続できますが、外部サービスでこのようなインスタンスとの接続を開始できないように NAT Gateway を使用することができます。

## リソース設定値

-   `included_subnet_name` : NAT Gateway を接続する Subnet 名
-   `connectivity_type` : NAT ゲートウェイの接続タイプ - `public`, `private`
-   `eip_name` : EIP 名
