## リソース概要

組織内の AWS アカウントおよびリソース全般にわたって保護を構成できる AWS サービス

## 関連リソース

### 上位リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Vpc)

### 接続リソース

-   [**Network Firewall Policy**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/NetworkFirewallPolicy)
-   [**Subnet**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Subnet)

## リソース設定値

-   `description` : Firewall ポリシーの説明
-   `included_vpc_name` : Firewall が配置される VPC 名
-   `linked_subnet_names` : Firewall エンドポイント専用のパブリックサブネット
-   `linked_network_firewall_policy_name` : 接続する Firewall ポリシー名
-   `enabled_delete_protection` : Firewall が削除されないように保護を活性化 - `false`,`true`
-   `enabled_subnet_change_protection` : サブネット接続が変更されないように保護を活性化 - `false`,`true`
-   `encryption_configuration.enabled_custom_configuration` : AWS 管理型キーを使用してデータを暗号化 - `false`,`true`
