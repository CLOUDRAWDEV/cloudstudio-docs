## リソース概要

Transit Gateway は仮想プライベートクラウド(VPC)とオンプレミスネットワークを相互接続するために使用できるネットワーク伝送ハブです。
クラウドインフラが世界的に拡張されるにつれて、リージョン間のピアリングは AWS グローバルインフラを使用して Transit Gateway を一緒に接続します。
データは自動的に暗号化され、パブリック インターネット経由では送信されません。

## リソース設定値

-   `dns_support` : 接続された VPC に対する DNS システム確認活性化 - `enable`, `disable`
-   `vpn_ecmp_support` : 接続された VPN の ECMP(マルチパス)ルーティングのサポート - `enable`, `disable`
-   `default_route_table_association` : Attachment をデフォルトのルーティングテーブルと自動接続 - `enable`, `disable`
-   `default_route_table_propagation` : デフォルトのルーティングテーブルで Attachment を自動伝播 - `enable`, `disable`
-   `multicast_support` : マルチキャストドメインの生成を有効にする - `enable`, `disable`
-   `transit_gateway_cidr_blocks` : Transit Gateway Cidr ブロック
-   `vpc_attachment`
