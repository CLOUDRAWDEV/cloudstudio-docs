## リソース概要

他のアカウントのリソース共有を許可するかどうかを決定するリソース

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**TransitGatewayAttachment**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/TransitGatewayAttachment)

## リソース設定値

-   `is_manual` : 接続する Transit gateway Attachment ID を直接入力するかどうかの可否 - `false`,`true`
-   `linked_transit_gateway_attachment_name` : 共有を許可する他の Transit Gateway Attachment の名前
