## リソース概要

고객이 지원하는 AWS 서비스 및 VPC endpoint 서비스에 private하게 연결할 수 있는 서비스

## 関連リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 下位リソース

-   [**LB**](https://docs.cstudio.app/ja/docs/resource/aws/Lb/Lb)
-   [**Endpoint**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Endpoint)

## リソース設定値

-   `acceptance_required` : サービスへの VPC エンドポイント接続リクエストをサービスオーナーが受け入れる必要があるかどうかの可否 - `false`,`true`
-   `loadbalancer_type` : Endpoint service で接続する load balancer タイプ - `network`, `gateway`
-   `linked_lb_names` : エンドポイントサービスに対する 1 つ以上のリンクされた Lb リソース名
-   `manual_lb_arns` : エンドポイントサービスの 1 つ以上のロードバランシングデバイスの Amazon Resource Names(ARNs)
-   `supported_ip_address_types` : サポートされている IP アドレス タイプ - `ipv4`
-   `activate_private_dns_name` : サービスのプライベート DNS 名
