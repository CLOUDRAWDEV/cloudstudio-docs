## リソース概要

Amazon Vpc と送受信する DNS クエリに対するエンドポイントサービス

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Vpc)
-   [**SecurityGroup**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/SecurityGroup)
-   [**ResolverRule**](https://docs.cstudio.app/ja/docs/resource/aws/Route53/ResolverRule)

## リソース設定値

-   `direction` : DNS クエリ方向 - `INBOUND`, `OUTBOUND`
-   `linked_vpc_name` : エンドポイントと接続する VPC 名
-   `linked_security_group_names` : エンドポイントのセキュリティグループ名
-   `ip_address` : IP アドレスの名前
