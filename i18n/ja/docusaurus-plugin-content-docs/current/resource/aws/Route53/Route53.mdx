## リソース概要

Amazon Route 53 は可用性と拡張性に優れたドメイン名システム(DNS)ウェブサービスです。

### 上位リソース

-   [**Base**](https://docs.cstudio.app/ja/docs/resource/aws/Base/)

### 接続リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Vpc)

## リソース設定値

-   `linked_vpc_names` : Route53 Zone を接続する VPC 名
-   `record` : Route53 Zone に生成する Record の名前
    -   `type` : Record のタイプ - `A`, `CAA`, `CNAME`, `DS`, `NAPTR`, `NS`, `PTR`, `SOA`, `SPF`, `SRV`, `TXT`
    -   `ttl` : Record の ttl
    -   `records` : レコードの文字列リスト
