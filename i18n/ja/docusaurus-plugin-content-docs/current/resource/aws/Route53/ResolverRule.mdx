## リソース概要

Resolver が指定されたドメイン名のクエリをネットワークに転送するときに使用するルール

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Vpc)
-   [**ResolverEndpoint**](https://docs.cstudio.app/ja/docs/resource/aws/Route53/ResolverEndpoint)

## リソース設定値

-   `rule_type` : ルールタイプ - `Forward`, `system`
-   `domain_name` : DNS クエリーが転送されるドメイン名
-   `linked_vpc_names` : ルールを使用する VPC 名
-   `linked_resolver_endpoint_name` : Resolver が使用するアウトバウンドエンドポイントの名前
-   `target_ip` : Target IP アドレス
-   `target_ip.name`
-   `target_ip.ip` : Target IP アドレスの IP
-   `target_ip.port` : Target IP アドレスのポート - `1~65535`
