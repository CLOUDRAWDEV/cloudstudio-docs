## リソース概要

AWS Direct Connect を使用すると標準イーサネット光ファイバケーブルを介して内部ネットワークを AWS Direct Connect の位置に接続することができます。
ケーブルの一端をユーザのルータに接続し、もう一端を AWS Direct Connect ルータに接続します。 この接続により、次のものを作成できます。
仮想インタフェースパブリック AWS ネットワーク経路でインターネットサービスプロバイダを迂回してサービス(例:Amazon S3)または Amazon VPC に転送します。
元の要求 ping に対する AWS Direct Connect の位置は、次へのアクセスを提供します。
AWS 該当項目が接続された地域でパブリックリージョンの単一接続を使用するか、AWS GovCloud(US)パブリックに対するアクセス AWS その他すべてのパブリックリージョンのサービスです。

## リソース設定値

-   `bandwidth` : Direct Connection の Bandwidth - `50Mbps`,`100Mbps`,`200Mbps`,`300Mbps`,`400Mbps`,`500Mbps`,`1Gbps`,`2Gbps`,`5Gbps`,`10Gbps`,`100Gbps`
-   `location` : 接続がある AWS Direct Connect 位置
