## リソース概要

Direct Connect 接続を通じて AWS リージョンの VPC へのアクセスをサポートする DxGateway

## 関連リソース

### 下位リソース

-   [**Vpn Gateway**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/VpnGateway)

## リソース設定値

-   `side_asn` : Amazon 側 ASN（自律システム番号） - `64512 ~ 4294967294`
