## リソース概要

Amazon Elastic Container Registry(Amazon ECR)は安全で拡張可能で信頼できる AWS 管理型コンテナイメージレジストリサービスです。
Amazon ECR は AWS IAM を使用してリソースベースの権限を持つプライベートリポジトリをサポートします。
したがって、指定されたユーザーまたは Amazon EC2 インスタンスがコンテナリポジトリ及びイメージにアクセスできます。
希望する CLI を使用して Docker イメージ、 Open Container Initiative(OCI)イメージ及び OCI 互換アーティファクトをプッシュ、プル及び管理することができます。

## リソース設定値

-   `image_tag_mutability` : リポジトリのタグ変更可能設定 - `IMMUTABLE`, `MUTABLE`
-   `scan_on_push` : イメージがリポジトリにプッシュされてからスキャンされるかどうかを設定 - `true`, `false`
-   `encryption_type` : リポジトリに使用する暗号化タイプ - `KMS`, `AES256`
-   `kms_key_name` : 使用する KMS Key の ARN 名
-   `target_principal_arn` : Policy を適用する対象の ARN
