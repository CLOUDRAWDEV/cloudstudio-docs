## リソース概要

AWS Directory Service は他の AWS サービスで Microsoft Active Directory(AD)を使用できるいくつかの方法を提供します。
ディレクトリはユーザー、グループ、およびデバイスに関する情報を保存し、管理者はこれを使用して情報およびリソースへのアクセスを管理します。
AWS Directory Service はクラウドで既存の Microsoft AD または LDAP(Lightweight Directory Access Protocol)認識アプリケーションを使用しようとするお客様に多様なディレクトリ選択オプションを提供します。
また、ユーザー、グループ、デバイス、およびアクセス権を管理するためにディレクトリが必要な開発者にも同じ選択オプションを提供します。

## リソース設定値

-   `password` : ディレクトリ管理者またはコネクタユーザーのパスワード
-   `type` : `SimpleAD`,`MicrosoftAD`
-   `size` : ディレクトリのサイズ - `Small`,`Large`
-   `included_vpc_name` : ディレクトリがある VPC の名前
-   `included_subnet_names` : ディレクトリサーバーのサブネット名(サブネットは 2 つの異なる AZ にデプロイされる必要があります。)
