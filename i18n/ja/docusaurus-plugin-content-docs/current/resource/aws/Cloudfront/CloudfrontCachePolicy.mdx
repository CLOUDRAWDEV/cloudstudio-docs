## リソース概要

Cloudfront Cache Policy は Cloudfront デプロイのキャッシュ動作に接続して CloudFront のユーザ指定を設定するキャッシュポリシーです。

## 関連リソース

### 接続リソース

-   [**CloudfrontDistribution**](https://docs.cstudio.app/docs/resource/aws/Cloudfront/CloudfrontDistribution)

## リソース設定値

-   `min_ttl` : オブジェクトが CloudFront キャッシュに残っている必要がある最小時間(秒)
-   `max_ttl` : オブジェクトが CloudFront キャッシュに維持される最大時間(秒)
-   `default_ttl` : オブジェクトが CloudFront キャッシュに残っている必要がある基本時間(秒)
-   `cookie_behavior` : クッキーをオリジンサーバーに渡すかどうかの可否 - `whitelist`,`all`,`none`,`allExcept`
-   `header_behavior` : 指定されたヘッダーの値に基づいてオブジェクトをキャッシングするかどうかの可否 - `whitelist`,`none`
-   `query_string_behavior` : クエリ文字列パラメータ値に基づいてコンテンツの別のバージョンをキャッシングするかどうかの可否 - `whitelist`,`all`,`none`,`allExcept`
