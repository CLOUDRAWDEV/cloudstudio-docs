## リソース概要

AWS Elasticache User は Elasticache クラスタを使用するために必要な IAM ユーザーです。
Elasticache User は Elasticache クラスタを生成、修正及び削除する権限を持ちます。
Elasticache User は AWS IAM で生成され、一般的に Elasticache を使用するアプリケーションまたは開発者に割り当てられます。
Elasticache User は Elasticache クラスタに対するセキュリティ及びアクセス制御を担当し、これによりアプリケーションとデータを保護することができます。
Elasticache User は AWS アカウントで他のリソースと一緒に使用することができます。

## リソース設定値

-   `user_id` : ユーザーの ID、英文字、ハイフン(-)使用可能、文字で始まり、ハイフンで終わるか、ハイフンを二つ連続で含めることはできない、最大 40 文字
-   `access_string` : ユーザーに適用する権限を入力した string
-   `authentication_type` : ユーザーがクラスターにアクセスするための認証方式
