## リソース概要

AWS SNS でコミュニケーションチャンネルの役割をする論理的アクセスポイント

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**Metric**](https://docs.cstudio.app/ja/docs/resource/aws/Cloudwatch/Metric)
-   [**Config**](https://docs.cstudio.app/ja/docs/resource/aws/Config/Config)

## リソース設定値

-   `display_name` : SNS Topic の表示名
-   `type` : SNS Topic のタイプ - `FIFO`, `Standard`
-   `kms_key_name` : SNS Topic に使用する KMS Key の名前
