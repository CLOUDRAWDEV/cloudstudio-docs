## リソース概要

指定したプロトコルとポート番号を使用して EC2 インスタンスのような 1 つ以上の登録された対象にリクエストをルーティングするサービス

-   [**Ec2 Instance**](https://docs.cstudio.app/ja/docs/resource/aws/Ec2/Instance)

## リソース設定値

-   `target_type` : LB ターゲットの種類 - `Instance`, `Ip`, `Lambda`, `Alb`
-   `protocol` : トラフィックへのルーティングに使用するプロトコル - `HTTP`,`HTTPS`,`TCP`,`TCP_UDP`,`TLS`
-   `port` : トラフィックを受信するポート - `1~65535`
-   `included_vpc_name` : ターゲットグループを生成する VPC の名前
-   `linked_instance_names` : Target Instance の名前リスト
