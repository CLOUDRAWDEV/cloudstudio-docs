## リソース概要

AWS CloudWatch Metric はモニタリング対象リソースから収集された測定値を表す名前と値のペアです。
例えば、EC2 インスタンスから CPU 使用率、ネットワーク入力および出力などの測定値を収集できます。
これらの測定値は CloudWatch Metric として保存され、ダッシュボード、通知および分析に使用されます。
また、ユーザー定義 Metric を生成してアプリケーションやサービスから収集されたデータをモニタリングすることもできます。
CloudWatch Metric は AWS が提供する様々なサービスと統合されて使用されます。

## リソース設定値

-   `comparison_operator` : 指定された統計及びスレッショルドを比較する時に使用する算術演算
-   `evaluation_periods` : データが指定されたスレッショルドと比較される期間の数
-   `metric_name` : アラームに関連されている測定項目の名前
-   `namespace` : アラームに関連されている指標のネームスペース
-   `period` : 指定された統計が適用される期間(秒)
-   `statistic` : アラームの関連測定項目に適用する統計
-   `threshold` : 指定された統計が比較される値
-   `alarm_description` : アラームについての説明
-   `insufficient_data_actions` : アラームが他の状態から insufficient_data 状態に切り替わるときに実行するジョブリスト
