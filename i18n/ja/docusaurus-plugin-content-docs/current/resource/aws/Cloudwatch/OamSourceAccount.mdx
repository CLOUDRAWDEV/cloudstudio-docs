## リソース概要

AWS CloudWatch Oam Source Accounts は、中央モニタリングアカウントと共有されるソースアカウントを構成します。

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**Oam Monitoring Account**](https://docs.cstudio.app/ja/docs/resource/aws/Cloudwatch/OamMonitoringAccount)

## リソース設定値

-   `is_manual` : 接続する Oam Source Account Arn を直接入力するかどうかの可否 - `false`,`true`
-   `linked_oam_monitoring_account_name` : 接続する Oam Monitoring Account の名前
-   `resource_types` : モニタリングアカウントと共有するデータ タイプ
    -   `AWS::Logs::LogGroup`
    -   `AWS::Cloudwatch::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
-   `label_template` : ソースアカウントを識別するのに役立つラベル
