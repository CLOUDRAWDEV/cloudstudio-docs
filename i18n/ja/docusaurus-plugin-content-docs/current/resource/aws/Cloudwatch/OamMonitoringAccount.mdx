## リソース概要

AWS CloudWatch Oam Monitoring Accounts は中央モニタリングアカウントを設定し、複数のアカウントのアプリケーションをモニタリングするサービスです。

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/aws/Base/Region)

### 接続リソース

-   [**Oam Source Account**](https://docs.cstudio.app/ja/docs/resource/aws/Cloudwatch/OamSourceAccount)

## リソース設定値

-   `oam_source_accounts` : モニタリングアカウントとデータを共有するアカウントのアカウント、組織パスまたは組織 ID
-   `resource_types` : 共有しようとしているデータタイプ
    -   `AWS::Logs::LogGroup`
    -   `AWS::Cloudwatch::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
