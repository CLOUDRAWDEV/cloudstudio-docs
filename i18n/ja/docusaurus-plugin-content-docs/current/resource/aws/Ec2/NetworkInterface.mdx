## リソース概要

VPC で仮想ネットワークカードを表す論理的ネットワーキングコンポーネント

## 関連リソース

### 上位リソース

-   [**Subnet**](https://docs.cstudio.app/ja/docs/resource/aws/Vpc/Subnet)

### 接続リソース

-   [**Eip**](https://docs.cstudio.app/ja/docs/resource/aws/Ec2/Eip)

## リソース設定値

-   `private_ips` : Private IP のリスト
-   `linked_security_group_names` : ネットワークインターフェイスに使用する Security Group の名前
-   `linked_eip_names` : 使用する Elastic Ip の名前
