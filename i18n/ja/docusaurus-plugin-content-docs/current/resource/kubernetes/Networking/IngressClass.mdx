## 리소스 개요

인그레스가 사용할 인그레스 컨트롤러를 선택하기 위한 쿠버네티스 리소스

## 연관 리소스

### 상위 리소스

-   [**Cluster**](https://docs.cstudio.app/docs/resource/kubernetes/Base/Cluster)



### 연결 리소스

-   [**Ingress**](https://docs.cstudio.app/docs/resource/kubernetes/Networking/Ingress)



## 리소스 설정 값

-   `service_name` : IngressClass 리소스 이름
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `controller` : 인그레스클래스를 처리해야 하는 컨트롤러 이름
-   `parameters` : 참조되는 리소스의 정보
    -   `name` : 참조되는 리소스 이름
    -   `kind` : 참조되는 리소스 유형
    -   `namespace` : 참조되는 리소스의 네임스페이스
    -   `scope` : 리소스의 범위를 나타냄 - 클러스터 또는 네임스페이스
    -   `api_group` : 참조되는 리소스의 API 그룹
