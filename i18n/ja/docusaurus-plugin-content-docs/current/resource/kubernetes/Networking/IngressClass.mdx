## リソース概要

Ingress が使用するイングレスコントローラーを選択するためのクバネティスリソース

## 関連リソース

### 上位リソース

-   [**Cluster**](https://docs.cstudio.app/ja/docs/resource/kubernetes/Base/Cluster)

### 接続リソース

-   [**Ingress**](https://docs.cstudio.app/ja/docs/resource/kubernetes/Networking/Ingress)

## リソース設定値

-   `service_name` : IngressClass リソース名
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `controller` : IngressClass を処理する必要があるコントローラ名
-   `parameters` : 参照されるリソースの情報
    -   `name` : 参照されるリソース名
    -   `kind` : 参照されるリソース タイプ
    -   `namespace` : 参照されるリソースのネームスペース
    -   `scope` : リソースの範囲を示します - クラスターまたはネームスペース
    -   `api_group` : 参照されるリソースに関するグループ
