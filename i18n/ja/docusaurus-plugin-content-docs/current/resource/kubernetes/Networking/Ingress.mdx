## リソース概要

インバウンド接続がバックエンドで定義されたエンドポイントに到達できるようにするルールに関するクバネティスリソース

## リソース設定値

-   `service_name` : Ingress リソース名
-   `namespace` : Ingress リソースがデプロイされるネームスペース
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `tls` : TLS を適用
    -   `hosts` : TLS 認証書に含まれるホストのリスト
    -   `secret_name` : TLS 認証書に関連付けられたクバネティスシークレットの名前を指定
-   `ingress_class_name` : 接続するイングレスクラスの名前を指定
-   `host` : ルーティング ルールを適用するホストネーム アドレスを指定
-   `backend` : ルーティングするバックエンド情報を入力
    -   `name` : Routing rule の名前
        -   `path` : バックエンドに接続するパス
        -   `path_type` : 経路一致の解析を決定 - `Prefix`, `ImplementationSpecific`, `Exact`
        -   `port` : サービスのポート番号 - `1~65535`
