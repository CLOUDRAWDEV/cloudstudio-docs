## 리소스 개요

Hashicorp-Vault를 설치하기 위한 헬름 차트입니다.

## 연관 리소스

### 상위 리소스

-   [**Cluster**](https://docs.cstudio.app/docs/resource/kubernetes/Base/Cluster)
-   [**Namespace**](https://docs.cstudio.app/docs/resource/kubernetes/Core/Namespace)

### 연결 리소스




## 리소스 설정 값

-   `service_name` : Hashicorp-vault 헬름 차트 릴리즈 이름
-   `namespace` : Hashicorp-vault 헬름 차트가 배포될 네임스페이스
-   `version` : 사용할 Hashicorp-vault 헬름 차트 버전
-   `set` : Hashicorp-vault 기본 차트의 값을 동적으로 변경
