## リソース概要

Hashicorp-Vault をインストールするための Helm Chart です。

### 接続リソース

## リソース設定値

-   `service_name` : Hashicorp-vault Helm Chart リリース名
-   `namespace` : Hashicorp Vault Helm Chart リソースがデプロイされるネームスペース
-   `version` : Hashicorp Vault Helm Chart のバージョンを指定
-   `set` : Hashicorp Vault デフォルトのチャートの値を動的に変更
