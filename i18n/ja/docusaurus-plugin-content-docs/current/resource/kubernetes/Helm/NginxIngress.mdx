## リソース概要

Nginx Ingress Controller をインストールするための Helm Chart です。

## リソース設定値

-   `service_name` : Nginx Ingress Controller Helm Chart リリース名
-   `namespace` : Nginx Ingress Controller Helm Chart リソースがデプロイされるネームスペース
-   `version` : Nginx Ingress Controller Helm Chart のバージョンを指定
-   `replica_count` : Nginx Ingress Controller Deployment のレプリカの数
-   `external_traffic_policy` : 外部トラフィックへの応答に関するポリシーを設定 - `Cluster`,`Local`
-   `ingress_class_name` : イングレスをどのイングレス コントローラーが管理するかを指定するリソースであるイングレス クラスの名前を指定
-   `set` : Nginx イングレスコントローラデフォルトのチャートの値を動的に変更
