## リソース概要

Cert Manager をインストールするための Helm Chart です。

### 接続リソース

## リソース設定値

-   `service_name` : Cert Manager Helm Chart リリース名
-   `namespace` : Cert Manager Helm Chart がデプロイされるネームスペース
-   `version` : 使用する Cert Manager Helm Chart バージョン
-   `installCRDs` : Cert Manager 関連 Custom Resource Definition をインストールするかどうかをチェック
