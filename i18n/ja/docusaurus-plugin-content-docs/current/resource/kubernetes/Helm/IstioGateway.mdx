## リソース設定値

-   `service_name` : Istio gateway Helm Chart リリース名
-   `namespace` : Istio gateway Helm Chart リソースがデプロイされるネームスペース
-   `version` : Istio gateway Helm Chart のバージョンを指定
-   `set` : Istio gateway デフォルトのチャートの値を動的に変更
