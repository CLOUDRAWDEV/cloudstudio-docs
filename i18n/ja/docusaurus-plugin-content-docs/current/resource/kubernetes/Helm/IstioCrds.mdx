## リソース概要

istio 用 Custom Resource Definitions Helm Chart です。

### 接続リソース

## リソース設定値

-   `service_name` : Istio CRDs Helm Chart リリース名
-   `namespace` : Istio CRDs Helm Chart リソースがデプロイされるネームスペース - `istio-system`に指定
-   `version` : Istio CRDs Helm Chart のバージョンを指定
-   `set` : Istio CRDs デフォルトのチャートの値を動的に変更
