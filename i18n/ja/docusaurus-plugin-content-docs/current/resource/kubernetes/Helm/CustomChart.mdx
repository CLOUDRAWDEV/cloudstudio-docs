## リソース概要

デフォルトのヘルムチャートをユーザーが希望する値に設定して配布できるリソースです。

### 接続リソース

## リソース設定値

-   `service_name` : Helm Custom Chart リリース名
-   `namespace` : カスタム Helm Chart リソースがデプロイされるネームスペース
-   `repository` : ヘルムチャートを保存および共有するためのリポジトリ(カスタムするチャートが保存されているリポジトリを指定)
-   `chart` : カスタムするヘルムチャート名を指定
-   `version` : カスタムするヘルム チャートのバージョンを指定
-   `set` : デフォルトのチャートの値を動的に変更
