## リソース概要

指定された名前空間にのみ適用される役割を生成するクバネティスリソース

## 関連リソース

### 上位リソース

-   [**Cluster**](https://docs.cstudio.app/ja/docs/resource/kubernetes/Base/Cluster)
-   [**Namespace**](https://docs.cstudio.app/ja/docs/resource/kubernetes/Core/Namespace)

### 接続リソース

-   [**Role**](https://docs.cstudio.app/ja/docs/resource/kubernetes/Rbac/Role)

## リソース設定値

-   `service_name` : RoleBinding リソース名
-   `namespace` : RoleBinding リソースがデプロイされるネームスペース
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `role_name` : 主体をバインディングする Role 名を指定
-   `subject` : Role を付与する主体を指定
    -   `name` : バインディングする主体の名前を入力
    -   `namespace` : バインディングする主体のネームスペースを入力
    -   `api_group` : 承認決定を下す API Group - `rbac.authorization.k8s.io`,`User`,`Group`
    -   `kind` : バインディングする主体のタイプを選択 - `ServiceAccount`,`User`,`Group`
