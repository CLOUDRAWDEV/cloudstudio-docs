## 리소스 개요

클러스터 수준과 모든 네임스페이스에서 적용되는 역할을 생성하는 쿠버네티스 리소스

## 연관 리소스

### 상위 리소스

-   [**Cluster**](https://docs.cstudio.app/docs/resource/kubernetes/Base/Cluster)


### 연결 리소스

-   [**ClusterRolebinding**](https://docs.cstudio.app/docs/resource/kubernetes/Rbac/ClusterRolebinding)


## 리소스 설정 값

-   `service_name` : ClusterRole 리소스 이름
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `rule` : 해당 clusterrole에 대한 정책역할
    -   `name` : 정책역할 이름
    -   `api_groups` : 작업을 허용할 리소스가 포함된 API Group
    -   `resources` : 이 규칙이 적용되는 리소스 목록
    -   `resource_names` : 해당 리소스의 개별 인스턴스로 선택할 경우 지정 - 빈 값일 경우 해당 리소스 전체를 포함
    -   `non_resource_urls` : 리소스가 아니라 url형식으로 rule을 지정할 때 사용 - 헬스체크용 url 지정 가능
    -   `verbs` : Rule에 포함된 모든 리소스에 적용되는 동사 목록
