## リソース概要

クラスター レベルとすべてのネームスペースで適用される役割を生成するクバネティス リソース

## リソース設定値

-   `service_name` : ClusterRole リソース名
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `rule` : 当該 Cluster Role に対するポリシーの役割
    -   `name` : rule 名
    -   `api_groups` : 作業を許可するリソースを含む API Group
    -   `resources` : このルールが適用されるリソースのリスト
    -   `resource_names` : 対応するリソースの個別のインスタンスとして選択する場合は指定 - 空き値の場合は、対応するリソース全体を含めます
    -   `non_resource_urls` : リソースではなく、url 形式でルールを指定するときに使用 - ヘルスチェック用の url を指定できます
    -   `verbs` : Rule に含まれるすべてのリソースに適用される動詞リスト
