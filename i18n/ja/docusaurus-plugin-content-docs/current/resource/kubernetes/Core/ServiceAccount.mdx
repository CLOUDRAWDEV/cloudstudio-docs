## リソース概要

クバネティスクラスター内でアプリケーションとサービスがクラスター リソースにアクセスして使用できるようにするアカウントです。

## リソース設定値

-   `service_name` : ServiceAccount リソース名
-   `namespace` : ServiceAccount リソースがデプロイされるネームスペース
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `secret` : このサービスアカウントを参照するファッドで使用することが許可されている secrets リスト
-   `image_pull_secrets` : このサービスアカウントを参照するファッドのイメージを取得するために使用する同一のネームスペースの secrets リスト
