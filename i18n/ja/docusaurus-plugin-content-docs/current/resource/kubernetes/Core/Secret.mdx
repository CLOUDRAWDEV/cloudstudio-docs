## リソース概要

個別の属性や、ファイル全体、JSON Blob のような大まかな項目に重要な情報を保存するために使用されるクバネティスリソースです。

## リソース設定値

-   `service_name` : Secret リソース名
-   `namespace` : Secret リソースがデプロイされるネームスペース
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `type` : Secret のタイプ - `Opaque`,`kubernetes.io/basic-auth`,`kubernetes.io/service-account-token`,`kubernetes.io/dockerconfigjson`
-   `immutable` : 秘密に保存されているデータが更新されないようにします(オブジェクトメタデータだけを変更できます) - `false`,`true`

-   `data` : secret に入る data
    -   `data.name` : secret に入る data - `Opaque`,`kubernetes.io/asic-auth`
        -   `data.secret` : secret に入る data 値 - `Opaque`,`kubernetes.io/basic-auth`
    -   `data.name` : private docker registry の名前 - `kubernetes.io/dockerconfigjson`
        -   `data.username` : private docker registry 認証ユーザー名 - `kubernetes.io/dockerconfigjson`
        -   `data.password` : private docker registry 認証ユーザーパスワード - `kubernetes.io/dockerconfigjson`
