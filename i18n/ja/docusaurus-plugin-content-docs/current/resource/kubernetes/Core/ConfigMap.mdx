## リソース概要

アプリケーション設定情報を分離して保存し、パッド内のコンテナに渡すために使用されるクバネティス リソースです。

## リソース設定値

-   `service_name` : ConfigMap リソース名
-   `namespace` : コンフィグマップリソースがデプロイされるネームスペース
-   `annotations` : クバネティスシステムで必要な情報を表示するために使用する key-value
-   `labels` : オブジェクトを区切るためにユーザーが任意に希望する値を指定
-   `data` : key-value ペアで一般的なデータ - 設定ファイル、環境変数などのテキスト形式の構成情報を保存するために使用
    -   `data.name` : data の key 値
    -   `data.value` : data の value 値
-   `binary_data` : 特定のキーのバイナリデータ-バイナリファイルまたはその他のバイナリ形式のデータを保存するために使用
    -   `data.name` : binary data の key 値
    -   `data.value` : binary data の value 値
