## リソース概要

Azure Policy Set Definition はより簡単な管理と割り当てのために一緒にグループ化された 1 つ以上のポリシー定義のコレクションです。複数のリソース タイプ、リソース グループ、またはサブスクリプションに適用できるポリシー セットを作成できます。
Policy Set Definition を使用すると、各ポリシーを個別に適用することなく、単一の作業でリソースまたはリソースグループに Policy Set を適用することができます。これにより、ポリシー管理を簡素化し、すべてのリソースが組織の標準およびポリシーに準拠していることを確認できます。
Policy Set Definition は JSON 形式で定義することができ、媒介変数、メタデータ、およびポリシー定義を含めることができます。 また、他の Policy Set 定義またはポリシー定義を参照して、複数のリソースおよびリソースグループに適用できる複雑なポリシーを作成できます。

## リソース設定値

-   `type` : ポリシーのタイプ - `Custom`,`BuiltIn`

-   `display_name` : ポリシー定義の表示名

-   `included_definition_name` : ポリシー セット定義に含まれるポリシー定義名
