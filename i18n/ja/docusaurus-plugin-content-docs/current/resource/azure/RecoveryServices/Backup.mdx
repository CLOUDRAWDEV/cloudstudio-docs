## リソース概要

Azure Backup はクラウドでデータとアプリケーションをバックアップして保護できる Microsoft のクラウドベースのバックアップソリューションです。
Azure Backup を使用するとオンプレミスサーバー、Azure で実行される仮想マシンおよびその他のクラウドサービスからデータをバックアップすることができます。
Azure Backup は柔軟なバックアップスケジュールと保存ポリシーを提供するので、バックアップを保存する頻度と期間をカスタマイズすることができます。
また、増分バックアップ、暗号化、データ圧縮などの機能を提供し、バックアップリポジトリの使用を最適化し、バックアップ時間を短縮します。

## リソース設定値

-   `sku` : Recovery Services Vault を使用するための価格設定階層(SKU) - `Standard`,`RS0`

-   `soft_delete_enabled` : Vault の Soft delete 活性化 - `true`,`false`

-   `vm_backup_policy`

    -   `backup_frequency` : 項目がバックアップされる頻度 - `Daily`,`Weekly`
    -   `backup_time` : 項目がバックアップされる時間
    -   `timezone` : バックアップ時間の時間帯
    -   `retention_count` : 毎日のバックアップ保存数 - `7`~`9999`

-   `protected_vm`

-   `share_backup_policy`

    -   `backup_frequency` : 項目がバックアップされる頻度 - `Daily`,`Weekly`
    -   `backup_time` : 項目がバックアップされる時間
    -   `timezone` : バックアップ時間の時間帯
    -   `daily_retention_count` : 毎日のバックアップ保存数 - `1`~`200`
    -   `enabled_weekly_backup` : 週間バックアップを有効にする

-   `protected_share`

    -   `source_storage_account_name` : バックアップするファイル共有の Storage アカウント名
    -   `source_share_name` : バックアップするファイル共有名
    -   `backup_policy_name` : バックアップに適用するバックアップポリシー名

-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
