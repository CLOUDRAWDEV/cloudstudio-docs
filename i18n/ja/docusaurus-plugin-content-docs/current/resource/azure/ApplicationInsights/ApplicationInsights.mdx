## リソース概要

Azure Application Insights は開発者がライブアプリケーションをモニタリングして分析するのに役立つ Microsoft が提供するクラウドベースのサービスです。
これにより、開発者はアプリケーションの問題を検出および診断し、アプリケーションのパフォーマンスに関するリアルタイムの洞察を得ることができます。
Application Insights を使用して開発者は、1 秒あたりのリクエスト数、応答時間、エラー率を含むアプリケーションのパフォーマンス及び使用状況を追跡することができます。
さらに、アプリケーションが使用する依存性と外部サービスに対する可視性を得ることができます。これにより、問題を迅速かつ効率的に解決できます。

## リソース設定値

-   `type` : 生成するアプリケーション Insight のプラットフォームタイプ - `ios`,`java`,`Node.js`,`other`,`MobileCenter`,`phone`,`store`,`web`
-   `retention_in_days` : データ保管日数 - `30`,`60`,`90`,`120`,`180`,`270`,`365`,`550`,`730`
-   `linked_workspace_name` : Application Insights が収集したデータを保存する Log Analytics Workspace の名前
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
