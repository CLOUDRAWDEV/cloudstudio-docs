## リソース概要

Log Analytics は、Azure Portal で Azure Monitor ログストアのデータに対してログクエリを編集して実行するために使用されるツールです。
レコードセットを返す簡単なクエリを作成し、Log Analytics 機能を使用してソート、フィルタリング、分析することができます。
または、統計分析を行い、チャートで結果を視覚化して特定のトレンドを把握する高度なクエリを作成することもできます。
クエリ結果を対話型で処理するか、クエリ警告または統合文書ログのような他の Azure Monitor 機能と一緒に使用するか、クエリ作成及びテストに Log Analytics が使用されます。

## リソース設定値

-   `retention_in_days` : Workspace のデータ保管日数 - `30`~ `730`

-   `internet_query_enabled` : 公共インターネットによるクエリの活性化 - `true`, `false`

-   `internet_ingestion_enabled` : 公共インターネットによるデータ収集の活性化 - `true`, `false`

-   `linked_automation_account_names` : Workspace と接続する読み取り可能なリソース名のリスト

-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
