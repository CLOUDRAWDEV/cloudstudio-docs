## リソース概要

Azure Security Center は、組織がハイブリッドクラウドワークロードにおけるセキュリティ脅威を防止、
検出、対応できるように、Microsoft が提供するクラウドベースのサービスです。Microsoft Azure およびオンプレミス環境でホストされているすべてのリソースの
セキュリティ状況を可視化するための中央管理型ダッシュボードを提供します。Azure Security Center は、環境を継続的に監視し、セキュリティ推奨事項、脆弱性評価、
脅威インテリジェンス、セキュリティアラートを提供することで、脅威から保護し、潜在的な脆弱性を特定するのに役立ちます。

## リソース設定値

-   `linked_log_analytics_workspace` : データを保存する Log Analytics Workspace の名前

-   `assessment_policy`

    -   `display_name`: Security Center Assessment の表示名
    -   `severity`: Security center assessment の重要度レベル
    -   `description`: Security center assessment の説明

-   `assessment`

-   `linked_advanced_threat_protection` : Security Center Advanced Threat Protection の名前
