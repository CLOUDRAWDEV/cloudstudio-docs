## リソース概要

Azure Redis Cache はオープンソースの Redis Cache エンジンをベースとする完全に管理されたメモリ内のデータストアです。
Microsoft Azure クラウドプラットフォームでサービスとして提供され、拡張性に優れ、応答性に優れたアプリケーションを構築するために使用されます。
Azure Redis Cache は頻繁かつ迅速にアクセスしなければならないセッションデータ、ユーザのデフォルト設定、およびアプリケーションの状態などのデータを保存·検索するのに使用できます。
文字列、ハッシュ、リスト、セット、およびソートされたセットなどのデータ構造をサポートします。
Azure Redis Cache はデータの持続性、高可用性、自動サイズ調整などの機能を提供するため、迅速かつ安定的なデータアクセスが必要なアプリケーションをビルドするために広く使用されます。
Azure App Service、Azure Functions、および Azure Logic Apps などの他の Azure サービスと統合することもできます。

## リソース設定値

-   `sku_name` : Redis キャッシュの価格設定階層名(SKU) - `Basic`,`Standard`,`Premium`,`Enterprise`

-   `cache_type` : Redis キャッシュの価格設定グループ及びデプロイのための Redis Cache のサイズ - `azureRedisCacheSku`

-   `version` : Redis バージョン - `6`

-   `enable_non_ssl_port` : Redis に non-SSL port 適用の可否 - `true`,`false`

-   `minimum_tls_version` : Redis の TLS 最小バージョン - `1.0`,`1.1`,`1.2`

-   `public_network_access_enabled` : パブリック IP アドレスを使用して Redis キャッシュへのアクセスを許容 - `true`,`false`

-   `redis_firewall_rule`

    -   `start_ip` : 開始 IP アドレス範囲
    -   `end_ip` : 終了 IP アドレス範囲
    -   `included_redis_cache_name` : ファイアウォール ルールを適用する Redis キャッシュ名

-- `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
