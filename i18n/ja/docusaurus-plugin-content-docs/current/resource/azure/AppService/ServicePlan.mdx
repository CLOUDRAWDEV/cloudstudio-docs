## リソース概要

App Service Plan은 App Service에서 실행되는 앱의 성능과 용량을 결정하는 요소입니다.
서비스 플랜은 컴퓨팅 리소스를 관리하는 단위로, CPU, 메모리, 인스턴스 수 등을 설정할 수 있으며, 이 리소스는 여러 앱에서 공유할 수 있습니다.
