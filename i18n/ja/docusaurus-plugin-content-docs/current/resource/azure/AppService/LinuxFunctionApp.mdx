## リソース概要

Microsoft Azure が提供するサーバレスコンピューティングプラットフォームで、開発者がクラウドで関数という小さなコード断片を実行できるようにします。
これにより、開発者はインフラ管理を気にすることなく、拡張可能で柔軟なプラットフォーム上でイベント基盤のアプリケーションを構築及びデプロイすることができます。

## リソース設定値

-   `kind` : App Service に適用するオペレーティングシステムタイプ`Linux`
-   `sku_tier` : App Service 料金プランの価格設定階層
-   `delegated_subnet_name` : 仮想ネットワーク委任のための Subnet 名
-   `function_app` : アプリサービスの名前
    -   `storage_account_name` : Function App の使用のための汎用 Storage Account 名
    -   `https_only` : 暗号化及び認証されたセキュリティ接続のため Function App に HTTPS によるアクセスのみ許容 - `true`, `false`
    -   `detailed_error_messages_enabled` : HTTP エラー発生時、エラーページに詳しいエラーメッセージを含む - `true`, `false`
    -   `auth_settings_enabled` : Function App の権限活性化 - `true`, `false`
    -   `http2_enabled` : セキュリティのために最新バージョンの HTTP 活性化 - `true`, `false`
