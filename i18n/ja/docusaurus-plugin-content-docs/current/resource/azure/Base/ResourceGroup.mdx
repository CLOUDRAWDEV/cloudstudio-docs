## リソース概要

リソースグループは Azure ソリューションに関連するリソースを保持するコンテナです。
リソース グループにはソリューションのすべてのリソースまたはグループで管理しようとしているリソースのみを含めることができます。
ユーザーの組織に最適な内容に基づいて、リソースをリソース グループにどのように割り当てるかを決定します。
一般的に簡単にデプロイ、更新、グループとして削除できるように、同じリソース グループに対して同じライフサイクルを共有するリソースを追加します。
