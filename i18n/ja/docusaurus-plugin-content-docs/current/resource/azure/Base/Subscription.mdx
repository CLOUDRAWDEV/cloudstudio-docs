## リソース概要

Azure サブスクリプションはお客様が Azure クラウドコンピューティングプラットフォーム及びそのサービスにアクセスして使用できるようにする支払い計画です。
ユーザーに仮想マシン、ストレージ、データベース、その他のクラウドベースのサービスなど、さまざまなサービスへのアクセスを提供します。
Azure サブスクリプションははそれぞれ異なる機能と価格計画がある様々な階層で使用でき、ユーザーはビジネス要件に応じて必要なリソースレベルを選択することができます。
