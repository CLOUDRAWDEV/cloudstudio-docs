## リソース概要

Azure プライベート DNS は、ユーザ指定の DNS ソリューションを追加しなくても、仮想ネットワークのドメイン名を管理し、確認できる安定的で信頼できる DNS サービスを提供します。
プライベート DNS 領域を使用すると、現在 Azure で提供されている名前の代わりに、ユーザ固有のユーザ指定ドメイン名を使用することができます。
プライベート DNS 領域に含まれているレコードはインターネットで確認できません。 プライベート DNS 領域に対する DNS 確認は、接続された仮想ネットワークでのみ動作します。

## リソース設定値

-   `private_dns_a_record`

    -   `ttl` : DNS Record の Time To Live(秒) - `0` ~ `2147483647`
    -   `records` : IPv4 アドレスのリスト

-   `private_dns_aaaa_record`

    -   `ttl` : DNS Record の Time To Live(秒) - `0` ~ `2147483647`
    -   `records` : IPv6 アドレスのリスト

-   `private_dns_cname_record`

    -   `ttl` : DNS Record の Time To Live(秒) - `0` ~ `2147483647`
    -   `record` : サブドメインを別のレコードにリンクするためのエイリアス

-   `private_dns_ptr_record`

    -   `ttl` : DNS Record の Time To Live(秒) - `0` ~ `2147483647`
    -   `records` : ポインタレコードタイプのドメイン名のリスト

-   `private_dns_txt_record`

    -   `ttl` : DNS Record の Time To Live(秒) - `0` ~ `2147483647`
    -   `records` : テキストタイプのレコード一リスト

-   `vnet_link`

    -   `linked_vnet_name` : DNS Record의 Time To Live(초)

-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
