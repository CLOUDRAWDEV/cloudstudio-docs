## リソース概要

で提供する管理型ネットワーキングサービスで、ソース IP アドレスを非表示にした状態で VNet（仮想ネットワーク）内のリソースがインターネットにアクセスできるようにします。
Azure NAT Gateway を使用して、個人の IP アドレスをインターネットに露出することなく、VNet 内のリソースにインターネット接続を提供することができます。これにより、インターネットからリソースに直接アクセスするのを防ぎ、セキュリティを改善するのに役立ちます。
また、VNet 内のリソースにアクセスするために必要なパブリック IP アドレスの数を減らすのに役立ちます。

## リソース設定値

-   `linked_subnet_name` : Nat Gateway を接続する Subnet の名前

-   `linked_public_ip_name` : Nat Gateway に接続する Public Ip の名前

-   `idle_timeout_in_minutes` : NatGateway でアイドル状態の流れを除去する時間制限(分) - `4`~ `120`

-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
