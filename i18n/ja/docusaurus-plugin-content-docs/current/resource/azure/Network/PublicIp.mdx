## リソース概要

Azure Public IP（共用 IP）はインターネットから直接アクセスできる固有の IP アドレスです。
Azure リソース、例えば仮想マシン、Load Balancer、VPN Gateway などのようにインターネットで露出されるリソースに使用されます。
これにより、Azure リソースに対するインターネットトラフィックを送信することができ、インターネットからそのリソースにアクセスすることができます。

## リソース設定値

-   `sku` : Public Ip の価格設定階層 - `true`, `false`
-   `allocation_method` : Public Ip アドレス割り当て方式 - `Static`, `Dynamic`
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
