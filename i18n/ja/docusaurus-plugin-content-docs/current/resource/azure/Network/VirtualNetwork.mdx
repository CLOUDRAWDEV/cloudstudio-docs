## 리소스 개요

Azure 가상 네트워크(VNet)는 Microsoft Azure 클라우드 컴퓨팅 플랫폼에서 제공하는 네트워킹 기능입니다.
이를 통해 가상 컴퓨터와 같은 Azure 리소스를 서로, 인터넷 및 온-프레미스 네트워크에 안전하게 연결하는 데 사용할 수 있는 논리적으로 격리된 네트워크 환경을 클라우드에 만들 수 있습니다.
Azure Virtual Network를 사용하면 여러 서브넷을 만들고 NSG(네트워크 보안 그룹)를 정의하여 리소스에 대한 인바운드 및 아웃바운드 트래픽을 제어할 수 있습니다.
Azure VPN Gateway 또는 Azure ExpressRoute를 사용하여 가상 네트워크를 온-프레미스 네트워크에 안전하게 연결할 수도 있습니다.

## 리소스 설정 값

-   `vnet_address_space` : Virtual Network가 사용될 IP 주소 공간, 1개 이상의 주소 지원 가능
-   `enable_ddos_protection` : Virtual Network에 DDos protection plan 허용 -`true`, `false`
-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
