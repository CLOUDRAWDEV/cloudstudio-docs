## リソース概要

Azure Bastion はブラウザと Azure Portal を使用したり、ローカルコンピュータに既にインストールされている基本 SSH または RDP クライアントを通じて仮想マシンに接続できるように配布するサービスです。
Azure Bastion サービスは仮想ネットワーク内部でプロビジョニングする完全なプラットフォーム管理型 PaaS サービスです。TLS を通じて Azure Portal で直接仮想マシンに対する安全で円滑な RDP/SSH 接続を提供します。
Azure Bastion を通じて接続すると、仮想マシンに共用 IP アドレス、エージェント、または特殊クライアントソフトウェアを必要としません。

## リソース設定値

-   `included_subnet_name` : Bastion Host が位置する Subnet の名前

-   `sku` : Bastion Host の SKU - `Basic`, `Standard`

-   `linked_public_ip_name` : Bastion Host に接続する Public Ip の名前

-   `ip_config_name` : IP Configuration の名前

-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
