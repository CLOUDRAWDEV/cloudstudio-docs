## リソース概要

Azure Public IP Prefix(공용 IP 접두사)는 여러 개의 연속된 공용 IP 주소 범위를 제공하는 리소스입니다.
이는 특정 시나리오에서 필요한 대규모 공용 IP 주소를 사용하는데 유용합니다.
Public IP Prefix를 사용하면 Azure 가상 머신, Load Balancer, Application Gateway 등과 같은
인터넷으로 노출되는 리소스에 대해 대량의 공용 IP 주소를 쉽게 관리할 수 있습니다.

## リソース設定値

-   `prefix_length` : `28`~`31`
-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
