## リソース概要

Azure Public IP Prefix(共用 IP プレフィックス)は、複数の連続した共用 IP アドレス範囲を提供するリソースです。
これは特定のシナリオで必要な大規模なパブリック IP アドレスを使用するのに役立ちます。
Public IP Prefix を使用すると、Azure 仮想マシン、Load Balancer、Application Gateway などのような
インターネットで露出するリソースに対して大量のパブリック IP アドレスを簡単に管理できます。

## リソース設定値

-   `prefix_length` : Public Ip prefix ビット数 - `28`~`31`
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
