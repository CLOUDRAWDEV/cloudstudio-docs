## リソース概要

Azure DDoS Protection は、DDoS(Distributed Denial of Service)攻撃から Azure リソースを保護するのに役立つ Microsoft Azure が提供するサービスです。
Azure DDoS Protection は基本、標準、プレミアムの 3 つの保護階層を提供します。
基本階層は、一般的なネットワーク階層攻撃に対する保護を提供しますが、標準およびプレミアム階層は、より高度なアプリケーション階層攻撃に対する追加保護を提供します。
Azure DDoS Protection プランは、お客様が Azure 購読の保護された共用 IP アドレス数に応じて DDoS Protection サービス費用を支払うことができる価格設定プランです。
この計画は顧客が保護サービスに対する追加費用をかけずに DDoS 攻撃から Azure リソースを保護できる費用効率的な方法を提供します。
