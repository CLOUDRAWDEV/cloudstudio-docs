## リソース概要

Azure Virtual Network Gateway は、Azure の Virtual Network とオン·プレミスインフラまたはその他の Azure Virtual Network との間に安全なクロス·プレミス接続を設定できるようにする Microsoft Azure が提供するサービスです。
オンプレミスインフラと Azure Virtual Network 間の架け橋の役割をしてデータセンターをクラウドに拡張し、ハイブリッドソリューションを作ることができます。
Azure Virtual Network Gateway はサイト間 VPN と支店間 VPN という 2 つのタイプの VPN 接続を提供します。
サイト間 VPN は、オン·プレミスネットワークを Azure Virtual Network に接続するために使用されます。 支店とサイト間の VPN は、個々のコンピュータを Azure 仮想ネットワークに接続するために使用されます。
Azure Virtual Network Gateway は、BGP(Border Gateway Protocol)、静的ルーティング、および動的ルーティングをはじめとする様々なルーティングタイプもサポートします。 また、高可用性、アクティブ-アクティブ構成、複数の仮想ネットワーク ゲートウェイ サポートなどの機能も提供します。

## リソース設定値

-   `included_subnet_name` : Virtual Network Gateway を通じてトラフィックを送受信する Subnet の名前
-   `gateway_type` : Virtual Network Gateway のタイプ -`Vpn`, `ExpressRoute`
-   `vpn_type` : VPN 接続の種類を設定するための VPN 形式 -`RouteBased`, `PolicyBased`
-   `sku` : Virtual Network Gateway の価格設定階層(SKU)
-   `generation` : Virtual Network Gateway の世代 - `Generation1`, `Generation2`, `None`
-   `enable_active_active_mode` : `true`, `false`
-   `enable_configure_bgp` : `true`, `false`
-   `ip_configuration` : IP configuration 名
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
