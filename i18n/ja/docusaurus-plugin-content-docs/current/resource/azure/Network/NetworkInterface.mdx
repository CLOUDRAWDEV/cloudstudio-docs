## リソース設定値

-   `included_subnet_name` : Network Interface が含まれている Subnet 名
-   `ip_configuration`
    -   `is_primary` : 基本 Ip Configuration を有効にする - `true`, `false`
    -   `private_ip_address_allocation` : プライベート IP アドレスの割り当て方法
    -   `enabled_public_ip` : Public Ip 使用の可否 - `true`, `false`
    -   `linked_public_ip_name` : 使用する Public Ip の名前
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
