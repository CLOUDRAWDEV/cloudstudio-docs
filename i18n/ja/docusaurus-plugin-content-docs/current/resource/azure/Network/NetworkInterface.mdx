## 리소스 설정 값

-   `included_subnet_name` : Network Interface가 포함된 Subnet 이름
-   `ip_configuration`
    -   `is_primary` : 기본 ip configuration 활성화 - `true`, `false`
    -   `private_ip_address_allocation` : 개인 ip 주소 할당 방식
    -   `enabled_public_ip` : Public ip 사용 여부 - `true`, `false`
    -   `linked_public_ip_name` : 사용할 Public ip 이름
-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
