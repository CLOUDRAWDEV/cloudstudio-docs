## リソース概要

Azure Windows Server Virtual Machine は、Microsoft の Azure クラウドコンピューティングプラットフォームで Windows Server オペレーティングシステムを実行する仮想マシン(VM)です。
Azure で Windows Server 仮想マシンを実行することによって、ユーザは好む Linux デプロイを使用しながら、Microsoft クラウドプラットフォームの拡張性、安定性、およびグローバル範囲を活用することができます。

## リソース設定値

-   `included_vnet_name` : Subnet が含まれている Virtual Network 名
-   `size` : ワークロード実行のための Virtual Machine のサイズ設定
-   `source_image`
    -   `enabled_custom_image` : ユーザーイメージ ID の使用有無 - `true`, `false`
    -   `publisher` : VM 生成のために使用するイメージのプロバイダー
    -   `Offer` : VM 生成のために使用するイメージの ID
    -   `sku` : VM 生成のために使用するイメージの価格設定階層
-   `computer_name` : Virtual Machine の host_name
-   `admin_username` : VM に対する管理者ユーザー名
-   `admin_password` : VM に対する管理者パスワード
-   `on_disk_caching` : 内部 OS Disk 使用のための caching タイプ
-   `storage_account_type` : 内部 OS Disk をバックアップするためのストレージタイプ
-   `disk_size_gb` : 内部 OS Disk のサイズ(GB) - `1`~`32767`
-   `disk_attachment` : ディスク接続名
-   `linked_network_interface_names` : Network Interface 名
-   `primary_network_interface_name` : Virtual Machine が接続された Network Interface 名
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
