## リソース概要

AvailabilitySet は重複性と可用性を提供するために Azure でアプリケーションがビルドされる方法を理解できるようにする VM の論理的グループです。
高可用性アプリケーションを提供し、99.95% Azure SLA を満たすために、可用性セット内に 2 つ以上の VM を作成することをお勧めします。
AvailabilitySet 自体に対する費用はないので作成する各 VM インスタンスに対してのみ料金を支払います。

## リソース設定値

-   `update_domain_count` : 仮想マシン グループを同時に再起動するための更新ドメインの数 - `1` ~ `20`
-   `fault_domain_count` : Cognitive Service 使用のための価格設定階層 - `1` ~ `3`
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
