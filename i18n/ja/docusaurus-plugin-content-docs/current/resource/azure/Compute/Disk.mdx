## リソース概要

Azure Virtual Machines(仮想マシン)はクラウドで実行される仮想サーバーです。
仮想マシンには 1 つ以上のディスクが含まれ、これらのディスクは仮想マシンのオペレーティング システム（OS）とデータを保存するために使用されます。
ディスクは仮想マシンの永続的なストレージとして機能します。

## リソース設定値

-   `storage_type` : ディスクをバックアップするためのストレージタイプ
-   `size_gb` : ディスクのサイズ(GB) - `1` ~ `32767`
-   `enabled_public_network_access` : ディスクにパブリックネットワークへのアクセスを許容 - `true`, `false`
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
