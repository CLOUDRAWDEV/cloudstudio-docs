## リソース概要

Azure SQL Database は広く使用されている Microsoft SQL Server エンジンを基盤とする Microsoft Azure が提供する完全に管理された関係型データベースサービスです。
ハードウェアまたはソフトウェアインフラを管理する必要なく、クラウドで SQL Server データベースを作成して管理することができます。
Azure SQL データベースを使用すると、自動パッチ、バックアップ、高可用性、および災害復旧などの機能を活用できます。
また、インテリジェントなパフォーマンス監視、自動調整、高度なセキュリティ機能などの機能を提供することで、データベースのパフォーマンスを最適化し、重要なデータを保護できます。

## リソース設定値

-   `included_mssql_server_name` : 데이터베이스가 생성될 Microsoft SQL 서버 이름

-   `sku_name` : 서버의 관리자 사용자 이름
-   `max_size_gb` : 데이터베이스의 최대 용량(GB) - `1` ~ `1024`
-   `collation` : Microsoft SQL 데이터베이스의 데이터 정렬 방식 - `azureMssqlCol`
-   `license_type` : 데이터베이스에 적용할 라이센스 유형 - `LicenseIncluded` , `BasePrice`
-   `read_scale` : 읽기 전용 복제본 중 하나의 컴퓨팅 용량을 사용하여 읽기 전용 워크로드를 오프로드하기 위한 기능 활성화 - `true` , `false`
-   `zone_redundant` : 데이터베이스 유지 관리 작업을 위한 영역 중복성 활성화 - `true` , `false`

-   `auditing_policy` : 보안 경고를 받을 이메일 주소 목록

    -   `database_name` : 감사 정책을 적용할 database 이름
    -   `enable_storage_destination` : Storage 계정 활성화 여부 - `true` ~ `false`
    -   `enable_log_analytics_destination` : Log analytics 활성화 여부 - `true` , `false`

-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
