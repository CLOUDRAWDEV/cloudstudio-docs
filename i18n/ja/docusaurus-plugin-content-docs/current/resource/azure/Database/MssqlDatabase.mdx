## リソース概要

Azure SQL Database は広く使用されているオープンソース Microsoft SQL Server エンジンを基盤とする Microsoft Azure が提供する完全に管理された関係型データベースサービスです。
ハードウェアまたはソフトウェアインフラを管理する必要なく、クラウドで SQL Server データベースを作成して管理することができます。
Azure SQL データベースを使用すると、自動パッチ、バックアップ、高可用性、および災害復旧などの機能を活用できます。
また、インテリジェントなパフォーマンス監視、自動調整、高度なセキュリティ機能などの機能を提供することで、データベースのパフォーマンスを最適化し、重要なデータを保護できます。

## リソース設定値

-   `included_mssql_server_name` : データベースが生成される Microsoft SQL サーバー名

-   `sku_name` : Microsoft SQL データベースの価格設定階層
-   `max_size_gb` : データベースの最大容量(GB) - `1` ~ `1024`
-   `collation` : Microsoft SQL データベースのデータ整列方式 - `azureMssqlCol`
-   `license_type` : データベースに適用するライセンスタイプ - `LicenseIncluded` , `BasePrice`
-   `read_scale` : 読み取り専用レプリカの 1 つのコンピューティング容量を使用して、読み取り専用ワークロードをオフロードするための機能を有効にする - `true` , `false`
-   `zone_redundant` : データベースメンテナンス操作のゾーン冗長性を有効にする - `true` , `false`

-   `auditing_policy` : 監査ポリシーの名前

    -   `database_name` : 監査ポリシーを適用する database 名
    -   `enable_storage_destination` : Storage アカウント活性化の可否 - `true` ~ `false`
    -   `enable_log_analytics_destination` : Log analytics 活性化の可否 - `true` , `false`

-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
