## リソース概要

Azure Cognitive Services는 개발자에게 기계 학습 모델을 처음부터 빌드하고 학습시킬 필요 없이 애플리케이션에 지능형 기능을 추가할 수 있도록 미리 빌드된 API를 제공하는 클라우드 기반 서비스 제품군입니다.
이러한 서비스에는 자연어 처리, 음성 인식, 이미지 및 비디오 분석 등이 포함됩니다.  
Azure Cognitive Services를 통해 개발자는 고급 기능을 애플리케이션에 쉽고 빠르게 통합하여 기능을 향상하고 더 나은 사용자 환경을 제공할 수 있습니다.
Azure Cognitive Services를 사용하여 개발자는 기계 학습 또는 AI에 대한 전문 지식이 없어도 애플리케이션에 강력한 AI 기능을 추가할 수 있습니다.

## リソース設定値

-   `kind` : 생성할 Cognitive Service 유형

-   `sku_name` : Cognitive Service 사용을 위한 가격 책정 계층

-   `custom_subdomain_name` : 토큰 기반의 권한 사용을 위한 서브 도메인 이름
