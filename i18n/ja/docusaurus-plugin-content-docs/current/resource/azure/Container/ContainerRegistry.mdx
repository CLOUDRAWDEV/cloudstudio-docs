## リソース概要

Azure Container Registry はオープンソース Docker Registry 2.0 を基盤とする管理型レジストリサービスです。
Azure コンテナレジストリを作成·維持·管理してコンテナイメージと関連アーティファクトを保存·管理します。

## リソース設定値

-   `sku` : Container Registry の性能及び拡張性のための SKU タイプ - `Basic`, `Standard`, `Premium`

-   `admin_enabled` : 管理者活性化 - `true`, `false`

-   `public_network_access_enabled` : Public ネットワークへのアクセスを許容 - `true`, `false`

-   `network_rule_set` : SKU タイプが Premium の場合に生成可能
    -   `default_action` : Rule と一致しないリクエストに対する動作 - `Deny`, `Allow`
    -   `ip_ranges` : Rule と一致するリクエストの IP アドレス IP 주소 CIDR
-   `tag` : リソースをカテゴリに分類するためのタグ - 最大 512 文字, `key`:`value`
