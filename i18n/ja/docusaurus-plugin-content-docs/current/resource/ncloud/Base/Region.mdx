## リソース概要

NAVER Cloud リソースをデプロイする Region 指定するリソースです。 韓国(KR)、日本(JPN)、シンガポール(SGN)に対応しています。

## 関連リソース

### 上位リソース

-   [**Base**](https://docs.cstudio.app/ja/docs/resource/ncloud/Base/)

### 下位リソース

-   [**Network Vpc**](https://docs.cstudio.app/ja/docs/resource/ncloud/Network/Vpc)
-   [**Network Vpc Peering**](https://docs.cstudio.app/ja/docs/resource/ncloud/Network/VpcPeering)

## リソース設定値

-   `region` : リソースをデプロイする NAVER Cloud Region
