## 리소스 개요

네이버 클라우드 리소스를 배포 할 region 지정하는 리소스입니다. 한국(KR), 일본(JPN), 싱가포르(SGN)을 지원합니다.

## 연관 리소스

### 상위 리소스

-   [**Base**](https://docs.cstudio.app/docs/resource/ncloud/Base/)

### 하위 리소스

-   [**Network Vpc**](https://docs.cstudio.app/docs/resource/ncloud/Network/Vpc)
-   [**Network Vpc Peering**](https://docs.cstudio.app/docs/resource/ncloud/Network/VpcPeering)

## 리소스 설정 값

-   `region` : 배포지역
