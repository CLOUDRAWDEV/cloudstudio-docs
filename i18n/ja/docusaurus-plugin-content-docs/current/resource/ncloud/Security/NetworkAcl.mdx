## リソース概要

サブネットに対するネットワーク セキュリティ管理ができるサービスです。

## 関連リソース

### 上位リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/ncloud/Networking/Vpc)

## リソース設定値

-   `included_vpc_name` : ACL を作成する VPC の名前
-   `nacl_description` : ACL の説明
-   `inbound_rule`
    -   `priority` : インバウンドルールの優先度、0 ~ 199
    -   `protocol` : インバウンドトラフィックのプロトコル - `TCP`, `UDP`, `ICMP`
    -   `rule_action` : インバウンドトラフィックの許可または拒否 - `ALLOW`, `DROP`
    -   `ip_block` : インバウンドトラフィックのソース、CIDR ブロック
    -   `port_range` : インバウンドルールに適用するポート範囲、1 から 65535
-   `outbound_rule` :
    -   `priority` : アウトバウンドルールの優先度、0 ~ 199
    -   `protocol` : アウトバウンドトラフィックのプロトコル - `TCP`, `UDP`, `ICMP`
    -   `rule_action` : アウトバウンドトラフィックの許可または拒否 - `ALLOW`, `DROP`
    -   `ip_block` : アウトバウンドトラフィックの宛先、CIDR ブロック
    -   `port_range` : アウトバウンドルールに適用するポート範囲、1 から 65535
