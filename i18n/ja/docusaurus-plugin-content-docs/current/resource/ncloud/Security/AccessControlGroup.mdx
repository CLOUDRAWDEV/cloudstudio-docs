## リソース概要

サーバー間ネットワークアクセス制御および管理ができる IP/Port ベースのフィルタリングファイアウォールサービスです。

## リソース設定値

-   `included_vpc_name` : ACG を作成する VPC の名前
-   `acg_description` : ACG の説明
-   `inbound_rule`

    -   `name` : インバウンドルールの名前
    -   `protocol` : インバウンドトラフィックのプロトコル - `TCP`, `UDP`, `ICMP`
    -   `rule_apply_method` : インバウンドルールを適用する方法 - `ip_block`, `source_name`
    -   `ip_block` : インバウンドトラフィックの宛先、CIDR ブロック
    -   `port_range` : 適用するポート範囲、1 から 65535

-   `outbound_rule`
    -   `name` : アウトバウンドルールの名前
    -   `protocol` : アウトバウンドトラフィックのプロトコル - `TCP`, `UDP`, `ICMP`
    -   `rule_apply_method` : アウトバウンドルールを適用する方法 - `ip_block`, `source_name`
    -   `ip_block` : アウトバウンドトラフィックの宛先、CIDR ブロック
    -   `port_range` : 適用するポート範囲、1 から 65535
