## リソース概要

一つの共用 IP で複数のサーバーと接続し、外部からの直接アクセスを防ぐサービスです。
複数台の非公認 IP を持つサーバーが外部サーバーと通信できるように接続し、外部からの直接アクセスを防ぎ、セキュリティを維持します。

## リソース設定値

-   `included_vpc_name` : Nat Gateway を作成する VPC の名前
-   `zone` : Nat Gateway が位置する物理的地域です - `KR-1`, `KR-2`
