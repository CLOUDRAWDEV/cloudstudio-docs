## 리소스 개요

하나의 공용 IP로 여러 서버와 연결하고 외부로 부터 직접 접근을 막는 서비스입니다. 여러 대의 비공인 IP를 가진 서버가 외부 서버와 통신할 수 있도록 연결하며 외부로부터의 직접 접근을 막아 보안을 유지합니다.

## 연관 리소스

### 상위 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/ncloud/Networking/Vpc)

## 리소스 설정 값

-   `included_vpc_name` : Nat gateway를 생성할 VPC 이름
-   `zone` : Nat gateway가 위치할 물리적 지역 - `KR-1`, `KR-2`
