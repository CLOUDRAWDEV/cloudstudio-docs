## リソース概要

VPC 間通信のための VPC ピアリングを設定するリソースです。

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/ncloud/Base/Region)

### 接続リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/ncloud/Networking/Vpc)

## リソース設定値

-   `linked_vpc_name_01` : リクエストを送信する VPC の名前
-   `linked_vpc_name_02` : リクエストを受信する VPC の名前
