## リソース概要

VPC 内のリソースのネットワーク経路を定義するサービスです。

## 関連リソース

### 上位リソース

-   [**Vpc**](https://docs.cstudio.app/ja/docs/resource/ncloud/Networking/Vpc)

### 接続リソース

-   [**Subnet**](https://docs.cstudio.app/ja/docs/resource/ncloud/Networking/Subnet)

## リソース設定値

-   `included_vpc_name` : 接続する VPC の名前
-   `supported_subnet_type` : サポートされるサブネットの設定、公用またはプライベート - `PUBLIC`, `PRIVATE`
-   `route_table_description` : ルートテーブルの説明
-   `linked_subnet_names` : 接続するサブネット名のリスト
-   `route`
    -   `target_type` : 宛先タイプ、Nat ゲートウェイまたは VPC ピアリング - `NATGW`, `VPCPEERING`, `VGW`
    -   `target_name` : 宛先の名前
    -   `destination_cidr_block` : CIDR 形式の宛先 IP 範囲
