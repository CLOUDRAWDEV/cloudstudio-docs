## リソース概要

サーバーに割り当てることができる Network Interface を生成するサービスです。 Network Interface が適用されたサーバは、Subnet を使用して独立したネットワーク通信が可能です。

-   [**Server**](https://docs.cstudio.app/ja/docs/resource/ncloud/Compute/Server)

## リソース設定値

-   `included_subnet_name` : ネットワークインターフェースが含まれるサブネットの名前"
-   `linked_acg_names` : NetworkIntercface を適用する ACG の名前のリスト
