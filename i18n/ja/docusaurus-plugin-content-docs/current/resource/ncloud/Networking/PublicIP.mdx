## リソース概要

サーバーをサービス目的で利用したり、生成したサーバーに接続するのに役立つサービスです。

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/ncloud/Base/Region)

### 接続リソース

-   [**Server**](https://docs.cstudio.app/ja/docs/resource/ncloud/Compute/Server)

## リソース設定値

-   `target_server_name` : 公用 IP を適用するサーバーインスタンスの名前
