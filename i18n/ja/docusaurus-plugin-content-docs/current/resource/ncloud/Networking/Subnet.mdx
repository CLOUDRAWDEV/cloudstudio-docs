## リソース概要

ネットワーク用途別の細分化が可能な VPC 下位ネットワークです。

## リソース設定値

-   `subnet_cidr` : VPC アドレス範囲内のサブネット IP アドレス範囲
-   `zone` : サブネットが配置される物理的なゾーン - `KR-1`, `KR-2`
-   `subnet_type` : インターネット接続、公用またはプライベート - `PUBLIC`, `PRIVATE`
-   `usage_type` : ロードバランサーの使用可否の設定 - `GEN`, `BM`
-   `included_vpc_name` : サブネットが配置される VPC の名前
-   `linked_nacl_name` : ブネットへのアクセスを制御する Network access control list
