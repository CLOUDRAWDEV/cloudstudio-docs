## リソース概要

SourceDeploy は新しく作成されたりアップデートされたソースを自動的にサーバにデプロイして適用するデプロイ自動化サービスです。
SourceDeploy を使用して事前に設定されたユーザーベースのコマンドを使用して、ソースのデプロイ、実行、および検証を自動化することができ、デプロイ中のサービス中断時間を最小限に抑えることができます。
