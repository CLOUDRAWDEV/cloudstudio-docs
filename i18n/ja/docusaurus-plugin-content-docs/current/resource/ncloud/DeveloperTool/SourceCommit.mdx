## 리소스 개요

SourceCommit은 소스 코드와 다양한 파일들을 안전하게 저장하기 위한 프라이빗 Git 리포지토리 서비스입니다. SourceCommit에서 리포지토리를 생성해 팀 프로젝트를 효율적으로 관리하고 소스 코드 등의 데이터를 손쉽게 저장할 수 있습니다.
