## リソース概要

リアルタイムストリーミングデータパイプライン及びアプリケーションビルドのためのオープンソースプラットフォームである Apache Kafka クラスターを簡単にデプロイ、保護、運営、拡張できるサービスです。

## リソース設定値

-   `kafka_version_code` : CDSS で使用する Kafka バージョン
-   `linked_config_group_name` : 使用する CDSS config group の名前
-   `os_image` : 使用する OS イメージタイプ
-   `cmak` : CMAK(Cluster Manager for Apache Kafka)の構成情報
    -   `user_name` : CMAK アクセス ID
    -   `user_password` : CMAK アクセスパスワード
-   `manager_node` : CMAK を通じて Apache Kafka クラスターを管理する Node 構成情報
    -   `manager_node_subnet_name` : マネージャノードが位置する subnet 名
    -   `manager_node_product_name` : マネージャ ノード ハードウェア製品コード
-   `broker_node` : 実際のデータが保存されるサーバー ノードの構成情報
    -   `broker_node_subnet_name` : ブローカーノードが位置する subnet 名
    -   `broker_node_product_code` : ブローカー ノード ハードウェア製品コード
    -   `broker_node_count` : ブローカー Node 数
    -   `broker_node_storage_size` : ブローカー Node の保存容量 - `100~2000`
