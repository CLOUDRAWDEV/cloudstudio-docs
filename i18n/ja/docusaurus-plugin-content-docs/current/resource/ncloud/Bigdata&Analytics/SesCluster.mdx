## リソース概要

elasticsearch 及び OpenSearch クラスタを便利に管理できる環境を提供し、
様々なサーバタイプ及びストレージ容量を選択してクラスタを構成し、簡単に配布、運営、及び拡張できるリソースです。

## リソース設定値

-   `os_image_code` : 使用する OS タイプ
-   `search_engine` : Search EngineNode の主要項目
    -   `version_code` : 使用する Search Engine バージョンコード
    -   `user_name` : Search Engine ユーザー名
    -   `user_password` : Search Engine ユーザーパスワード
    -   `dashboard_port` : Search Engine ダッシュボードのポート (1024 ~ 65535)
-   `manager_node` : manager node 主要項目
    -   `enable_dual_manager` : manager node 二重化可否
    -   `manager_node_product_code` : 使用する manager node 製品コード
    -   `manager_node_subnet_name` : manager node を位置させる subnet の名前
-   `data_node` : データ Node の主要項目
    -   `data_node_product_code` : 使用するデータ Node 製品コード
    -   `data_node_subnet_name` : data node を位置させる subnet の名前
    -   `data_node_count` : データ Node の数 (3 ~10)
    -   `data_node_storage_size` : データノードストレージ容量(100~2000)
-   `enable_master_node` : master node を使用しているかどうかの可否
-   `login_key_name` : 暗号化用のパブリック ログイン キー名
