## リソース概要

Init Script はあらかじめ作成しておいたスクリプトをサーバー生成時に自動的に実行させることができる機能です。
同じ用途のサーバーを複数台一括生成したり、同じ環境のサーバーを周期的に生成したり、用途別にサーバーの初期環境管理が必要な場合など、様々な場合に活用できます。

## リソース設定値

-   `OS_type` : 使用するサーバーインスタンスの種類 - `LNX`, `WND`
-   `Content` : サーバーが生成される時に自動的に実行するスクリプト内容、例) #!/bin/bash yum install -y httpd && systemctl start httpd
