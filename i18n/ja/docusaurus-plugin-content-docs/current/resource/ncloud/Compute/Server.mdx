## リソース概要

実際のハードウェアを管理しなくても、仮想化の柔軟性を提供するコンピューティング リソースです。
クラウド環境にサーバーを生成し、時間とコストの面で効率的でインフラ運用が負担になったり、一時的に多くのサーバーリソースが必要な場合に適しています。

## リソース設定値

-   `included_subnet_name` : サーバーが位置する Subnet の名前
-   `server_image_product_code` : 使用するオペレーティングシステムイメージコード
-   `server_product_code` : サーバーのハードウェア構成用のコード
-   `login_key_name` : 使用する Login key の名前
-   `init_script` : 使用する Init script の名前
-   `is_encrypted_base_block_storage_volume` : ブロックストレージを暗号化するかどうかの可否
-   `linked_network_interface_name` : サーバーに割り当てるネットワーク インターフェイス
