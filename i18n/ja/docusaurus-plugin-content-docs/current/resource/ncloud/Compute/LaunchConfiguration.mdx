## リソース概要

Auto Scaling 遂行中にサーバー拡張が必要な時に自動的に生成される仮想サーバーのスペックを構成するテンプレートです。

## 関連リソース

### 上位リソース

-   [**Region**](https://docs.cstudio.app/ja/docs/resource/ncloud/Base/Region)

## リソース設定値

-   `server_image_product_code` : 使用するオペレーティングシステムイメージコード(CNTOS:CentOS, UBNTU:Ubuntu, WND:Windows / MYSQL,MARIA:Database / SVR:Server, EN:English Edition)
-   `server_product_code` : サーバのハードウェア構成のためのコード(HICPU:High CPU、STAND:Standard、HIMEM:High Memory/C コア数、M メモリ(GB))
-   `login_key_name` : サーバー インスタンスへの接続と暗号化、復号化に必要なリソース
-   `init_script`
    -   `content` : サーバーが生成されたときに自動的に実行するスクリプトの内容
-   `is_encrypted_volume` : Basic block storage が暗号化されているかどうか
