## リソース概要

アプリケーションのコンテナ化、デプロイ、管理を自動化するオープンソースプラットフォームであるクバネティスサービスです。
コンテナのデプロイ、管理、拡張など、コンテナを使用するために必要な作業を自動化し、業務効率を高めます。

## リソース設定値

-   `zone` : クラスターを生成する物理的地域 - `KR-1`, `KR-2`
-   `included_vpc_name` : クラスターを生成する VPC 名
-   `hypervisor_code` : ハイパーバイザーコードクラスターバージョン - `XEN`
-   `version` : クラスターバージョン - `1.26.10-nks.1`, `1.27.9-nks.1`
-   `maximum_number_of_nodes` : クラスターの最大 Node 数の設定、10 または 50
-   `enable_public_subnet_network` : パブリックサブネットと接続するかどうかの可否
-   `subnet_names` : クラスターを構成するサブネット名のリスト、最大 3 つ
-   `lb_private_subnet_name` : クラスターと連動するプライベート ロード バランサーが位置するプライベート サブネット名
-   `lb_pubic_subnet_name` : クラスターと連動するパブリック ロード バランサーが位置するパブリック サブネット名
-   `login_key_name` : 使用する Login key の名前
-   `enable_audit_log` : audit log 機能を使用しているかどうかの可否
-   `node_pool` : Node pool 名
    -   `node_count` : クラスターの Node 数
    -   `software_code` : Node のサーバーイメージコード
    -   `product_code` : Node のハードウェア構成用のコード
    -   `node_pool_subnet_name` : Node pool が作成されるサブネット名
-   `enable_autoscale` : オートスケール機能が使用されているかどうかの可否
