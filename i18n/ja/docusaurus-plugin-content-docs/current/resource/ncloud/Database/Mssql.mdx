## リソース概要

Cloud DB for MSSQL は MSSQL データベースを簡単に構築し、自動的に管理することができます。
NAVER Cloud の最適化設定を通じて安定的に運営し、障害発生時に自動復旧する完全管理型クラウドデータベースサービスです。

## リソース設定値

-   `enable_high_availability` : 高可用性機能の使用有無
-   `user_name` : MSSQL ユーザーアクセス ID
-   `user_password` : MSSQL ユーザーアクセスパスワード
-   `image_product_code` : 生成する MSSQL インスタントサーバーイメージスペックのイメージ製品番号
-   `product_code` : 生成する MSSQL インスタンスサーバイメージスペックの製品番号
-   `data_storage_type` : データストレージの種類 - `SSD`, `HDD`
-   `backup_file_retention_period` : バックアップファイルの保管期間、バックアップは日単位で行われ、バックアップファイルは別のストレージに保存 - `1~30`
-   `enable_automatic_backup` : バックアップの自動化を使用するかどうかの可否
-   `port` : MySQL 接続ポート - `3306~20000`
-   `character_set_name` : DB 文字セット。ハングルと英語の組み合わせから選択可能
