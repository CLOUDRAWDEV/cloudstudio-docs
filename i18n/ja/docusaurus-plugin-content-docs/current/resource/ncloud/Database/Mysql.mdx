## リソース概要

Mysql はリレーショナルデータベース MySQL を利用した完全管理型クラウドデータベースサービスで、MySQL データベースを簡単に構築、安定的に運営し、障害発生時に自動的に復旧するサービスです。

## リソース設定値

-   `server_name_prefix` : MySQL サーバー名
-   `user_name` : MySQL ユーザーアクセス ID
-   `user_password` : MySQL のユーザーパスワード
-   `host_ip` : 接続しようとする client IP
-   `database_name` : 生成しようとしているデータベース名
-   `image_product_code` : MySQL 構成インスタンススペックで生成しようとする MySQL サーバーイメージの製品コード
-   `product_code` : 生成する MySQL インスタンスのサーバー仕様を決定する製品コード
-   `data_storage_type` : データストレージの種類 - `SSD`, `HDD`
-   `enable_high_availiability` : 高可用性機能の使用有無
-   `enable_backup` : バックアップ機能の使用有無
-   `backup_file_retention_period` : バックアップファイルの保管期間、バックアップは日単位で行われ、バックアップファイルは別のストレージに保存 - `1~30`
-   `enable_automatic_backup` : バックアップの自動化機能の使用有無
-   `port` : MySQL 接続ポート - `3306~20000`
