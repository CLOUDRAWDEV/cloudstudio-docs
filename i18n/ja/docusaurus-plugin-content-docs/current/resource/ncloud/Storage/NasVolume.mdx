## リソース概要

多数のユーザーがデータを共有するように、複数のサーバーをネットワークに接続したストレージ サービスです。ネットワークに接続するストレージ容量で、多数のユーザーがデータを共有するのに適しています。

## リソース設定値

-   `volume_name_postfix` : NAS ボリュームの名前、英数字が許可されており、3〜20 文字以内
-   `volume_size` : ボリュームサイズ(100GB)、500 から 10,000 の範囲
-   `volume_allotment_protocol_type` : プロトコルの種類、マウント可能なサーバー: NFS-Linux/CIFS-Windows - `NFS`, `CIFS`
