## リソース概要

サーバーの基本ストレージに加えて、追加で提供する HDD または SSD ストレージサービスです。 短時間で作成してすぐにサーバーと接続でき、必要でない場合はすぐに返却できます。

-   [**Server**](https://docs.cstudio.app/ja/docs/resource/ncloud/Compute/Server)

## リソース設定値

-   `linked_server_name` : Block Storage を関連付けるサーバーの名前
-   `storage_size` : Block Storage のサイズ(GB)、10 から 16380 までの範囲
-   `storage_disk_type` : Block Storage のディスクタイプ、SSD または HDD - `SSD`, `HDD`
-   `snapshot_name` : 生成するスナップショット名
