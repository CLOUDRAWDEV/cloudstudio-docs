キャンバスはリソースを Drag&Drop してクラウドインフラを描き、バージョンを管理してデプロイまでできる <highlight color="#206FD6">CloudStudio のコアサービス</highlight>です。
キャンバスで構成されたインフラは単に視覚的アーキテクチャとして使用することもでき、希望する CSP 社に直接デプロイすることもできます。

✂️  自由にキャンバスを編集してみましょう！

<p>&nbsp;</p>

## ダイアグラムオプションツールボックスの使い方 {#toolbox}

上部にあるダイアグラムオプションツールボックスは、キャンバスを飾ることができる様々な機能を提供します。

| イメージ                                                              | 動作                   | 機能                                                                 |
| --------------------------------------------------------------------- | ---------------------- | -------------------------------------------------------------------- |
| <img width="40px" src={ undo } alt="실행 취소 이미지" />              | 取り消し               | 1 つ前に戻します。                                                   |
| <img width="40px" src={ redo } alt="다시 하기 이미지" />              | やり直し               | 1 つ先に進みます。                                                   |
| <img width="40px" src={ list } alt="리소스 리스트 이미지" />          | リソースリスト         | リソースリストの表示/非表示を切り替えるボタンです。                  |
| <img width="40px" src={ listName } alt="리소스 이름 리스트 이미지" /> | リソース名の表示       | 名前を含むリソース アイコンのリストを表示するボタンです。            |
| <img width="40px" src={ front } alt="뒤로 보내기 이미지" />           | 背面に移動             | 2 つのリソースが重なったときに、選択したリソースを後方に送信します。 |
| <img width="40px" src={ back } alt="앞으로 보내기 이미지" />          | 前面に移動             | 2 つのリソースが重なったときに、選択したリソースを前方に送信します。 |
| <img width="40px" src={ text } alt="텍스트 색상 변경 이미지" />       | テキスト色の変更       | テキスト文字のオプションを変更できます。                             |

| <img width="40px" src={ theme } alt="캔버스 테마 이미지" />           | キャンバステーマの変更 | キャンバスで該当の CSP に合わせてテーマを変更できます。              |

## キャンバスオプションツールボックス使い方 {#toolbox}

サイドメニューにあるキャンバスオプションツールボックスを使用して CloudStudio のさまざまな機能を利用できます。

| イメージ                                                                | 動作                     | 機能                                                                                                      |
| ----------------------------------------------------------------------- | ------------------------ | --------------------------------------------------------------------------------------------------------- |

| <img width="40px" src={ exporter } alt="내보내기 이미지" />             | エクスポート             | キャンバスに作成されたアーキテクチャのイメージと Terraform コードを圧縮ファイルとしてダウンロードします。 |

| <img width="40px" src={ versionSave } alt="버전 저장 이미지" />         | バージョン保存           | 現在構成されているキャンバスをバージョンごとに保存できます。                                              |
| <img width="40px" src={ versionCommit } alt="버전 커밋 이미지" />       | バージョン履歴           | 以前に記録されたバージョン履歴を確認して読み込むことができます。                                          |
| <img width="40px" src={ billing } alt="비용계산 이미지" />              | コスト計算               | 現在キャンバスに構成されているリソースのコストを予想して計算します。                                      |
| <img width="40px" src={ diagnose } alt="보안 진단 이미지" />            | セキュリティ診断         | キャンバスによって生成された Terraform コードの脆弱性を診断します。                                       |
| <img width="40px" src={ plan } alt="유효성 검사 이미지" />              | 有効性検査               | キャンバスによって生成された Terraform コードの有効性検査を行います。                                     |
| <img width="40px" src={ deploy } alt="배포 이미지" />                   | デプロイ                 | クラウド環境にキャンバスで構成されたリソースを実際にデプロイします。                                      |

| <img width="40px" src={ resourceOption } alt=" 리소스옵션 이미지" />    | リソースオプション       | キャンバスで構成されたリソースのオプションを確認できます。                                                |
| <img width="40px" src={ code } alt=" 테라폼 코드 이미지" />             | Terraform コード         | キャンバスで構成された Terraform コードを確認できます。                                                   |
| <img width="40px" src={ configuration } alt=" 캔버스 구성 이미지" />    | キャンバス構成           | キャンバス構成をツリー形式で確認できます。                                                                |

## リソース Drag & Drop {#drag-drop}

`Drag & Drop`はキャンバスを構成できる最も単純な機能です。
左側の`リソースリスト`から必要なリソースをドラッグしてキャンバスに Drag&Drop してみましょう！
互いに接続できないリソースは、赤い線とともに跳ね返ります。

    <img src={dragDrop} alt="드래그앤드롭 이미지" />
</div>

:::

### リソースタイプ

リソースは大きく以下の`4つ`タイプに分けられます。以下のリソースリストを参考にして、キャンバス リソースを接続します。

| 이미지                                            | 이름       | 설명                                                   |
| ------------------------------------------------- | ---------- | ------------------------------------------------------ |
| <img width="40px" src={ basic } alt="이미지" />   | 基本形     | グループ内に属するリソースを表示します。               |
| <img width="40px" src={ connect } alt="이미지" /> | 連結形     | そのリソースに接続できるリソースのリストを表示します。 |
| <img width="40px" src={ group } alt="이미지" />   | グループ型 | 該当グループに属するリソースのリストを表示します。     |
| <img width="40px" src={ tools } alt="이미지" />   | Tools      | キャンバス ツールのリストを表示します。                |

## リソースオプションの編集 {#option-edit}

リソースを`ダブルクリック`してリソース設定値を変更できる`編集ウィンドウ`を開きます。
`詳細を見る`アイコンをクリックしてそのリソースの説明を確認できます。
❓ `クェスチョンマーク`アイコンにマウスを合わせると各オプションに関する説明を確認できます。説明を参考にして、必要な値を設定してください。

    <img src={edit} alt="옵션 편집 이미지" />
</div>

## エラーを修正する {#error-correction}

🔴  赤い点はそのリソースにエラーがあることを示しています。 <highlight color="#FF7272">エラーがある場合、キャンバスを保存及びデプロイすることはできません。</highlight>
エラーを解決するために、キャンバスでエラーが発生したリソースを`ダブルクリック`してキャンバスエラーウィンドウを開きます。
キャンバス エラー ウィンドウで修正しようとしている特定のエラーをダブルクリックすると、エラーが発生した場所に移動します。エラーの原因を参考にしてエラーを修正します。

<div className={styles.imgWrap}>
    <img src={error} alt="오류 수정 이미지" />
</div>

<p>&nbsp;</p>

### エラーの種類

| 종류  | 설명                                                 |
| ----- | ---------------------------------------------------- |
| value | 入力値が正しくない場合に発生するエラー               |
| group | どのグループに属していない場合に発生するエラー       |
| line  | 必要なリソースが接続されていない場合に発生するエラー |
