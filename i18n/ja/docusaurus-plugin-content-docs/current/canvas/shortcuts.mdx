CloudStudio のキャンバスで使用できる`ショートカットキー`です。

| ショートカットキー                  | 説明               |
| ----------------------------------- | ------------------ |
| Ctrl + マウススクロール             | 画面の拡大及び縮小 |
| Space + マウス左クリック + ドラッグ | 画面移動           |
| Delete                              | 削除               |
| Ctrl + Z                            | 元に戻す           |
| Ctrl + C                            | コピー             |
| Ctrl + V                            | ペースト           |
