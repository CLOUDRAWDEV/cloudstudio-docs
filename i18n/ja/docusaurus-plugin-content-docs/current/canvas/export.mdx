CloudStudio からファイルをエクスポートすると 3 つの形式のファイルを取得できます。

## 形式

-   png ファイル
-   3 つの tf ファイル (main, output, variables)
-   state ファイル (管理中のインフラストラクチャ)
-   nodes.json ファイル

### 1. architecture.png ファイル

ユーザーが描いたキャンバス アーキテクチャの png ファイルです。

### 2. tf ファイル

ユーザーは 3 つの tf ファイルを取得できます。

`main.tf`: 生成したインフラストラクチャの骨組みを示す Terraform ファイルです。
`output.tf`: インフラストラクチャデプロイ後に関連情報を知らせる Terraform ファイルです。
`variables.tf`: 使用された変数を示す Terraform ファイルです。

### 3. terraform.state ファイル

デプロイ後のインフラストラクチャの状態を管理する Terraform ファイルです。デプロイには提供いたしません。

### 4. nodes.json ファイル

キャンバスを描画するための座標データです。
