CloudStudio は非会員でアクセスできません。ログイン前に会員登録を先に行ってください。

🔗  [CloudStudio 会員登録リンク](https://cstudio.app/signup)

✏️  <Link to="/docs/start/sign-up">会員登録の方法</Link>

会員登録が終わりましたら[CloudStudio ログインページ](https://cstudio.app/signin)에 접속합니다.

<p>&nbsp;</p>

1. 会員登録時に入力したメールアドレスから ID と一時パスワードを確認します。

<div className={styles.imgWrap}>
    <img width="100%" src={login01} width="450px" alt="img" />
</div>

<p>&nbsp;</p>

2. CloudStudio のログインページで`IDと一時パスワードを入力`します。

<img width="100%" src={login02} alt="img" />

<p>&nbsp;</p>

3. 新しいパスワードを設定します。

<img width="100%" src={otp06} alt="otp img" />

<p>&nbsp;</p>

:::info
最初にログインするときは、パスワードをリセットしていただく必要があります。

:::
