CloudStudio 会員登録のために、以下のリンクにアクセスします。

[Start CloudStudio](https://cstudio.app/signin)

1. 会員登録ボタンをクリックします。
2. 使用するメールアドレスを入力します。 例) `cloudstudio@cloudraw.kr`
3. `コード転送`ボタンをクリックし、入力したメールアドレスに送信された認証番号 6 桁を確認します。
4. CloudStudio に戻り、認証番号を入力し、`確認`ボタンをクリックしてメール認証を完了します。

:::info
認証番号は`3分以内`に入力する必要があります。認証時間が満了すると、認証番号再送信のために**再送信**をクリックしてください。
:::

5. 使用する ID を入力し、`重複検査`ボタンをクリックします。

:::info
ID は以下の条件を満たす必要があります。

-   英小文字で始まり
-   英数字のみを使用
-   6 桁以上 16 桁以内

:::

6. 利用規約に同意します。
7. `会員登録`をクリックします。
8. 登録したメールに一時パスワードが送信されます。
