完成したプロジェクトは`修正·コピー·削除`できます。

<br />

## プロジェクトの詳細情報を修正する

### プロジェクト名を修正する

1. プロジェクトの詳細情報を修正するためにテンプレートメニューで`修正を希望するプロジェクト`をクリックします。
2. ポップアップが表示されたらプロジェクト名に`編集アイコン`をクリックします。
3. `プロジェクト名`を修正します。
4. もう一度`編集アイコン`をクリックして修正を完了します。

1. プロジェクトの詳細を修正するためにテンプレートメニューで`修正を希望するプロジェクト`をクリックします。
2. ポップアップが表示されたらプロジェクトの詳細に`編集アイコン`をクリックします。
3. `プロジェクトの詳細`を修正します。
4. もう一度`編集アイコン`をクリックして修正を完了します。

## プロジェクトのコピー及び削除

### プロジェクトのコピー

1. プロジェクトをコピーするためにテンプレートメニューで修正を希望するプロジェクトの`三点リーダー`をクリックします。
2. `コピー`をクリックします。
3. ポップアップが表示されたらプロジェクトをコピーするフォルダーを選択します。
4. プロジェクト名を作成します。

> -   英語、ハングル、数字のみ許可
> -   2 ~ 20 桁

5. プロジェクトの説明を作成します。

> -   100 文字以内

6. `コピーする`ボタンをクリックします。

    <img src={modifyProject} width="800px" alt="이미지" />
</div>

<p>&nbsp;</p>

### プロジェクトの削除

1. プロジェクトを削除するためにテンプレートメニューで削除を希望するプロジェクトの`三点リーダー`をクリックします。
2. `削除`をクリックします。
3. ポップアップが表示されたらプロジェクト名をもう一度入力します。
4. `削除する`ボタンをクリックします。

## キャンバスの修正

1. キャンバスを修正するためにテンプレートから修正を希望するプロジェクトをクリックします。
2. ポップアップが表示されたら右側の`プロジェクトを修正する`ボタンをクリックします。
