## ID を探す

1. CloudStudio ログインページで **[ID を探す](https://cstudio.app/forgot/id)**をクリックします。
2. メールアドレスを入力し、コード転送をクリックします。 例) `cloudstudio@cloudraw.kr`
3. 入力したメールアドレスから送信された`6桁の認証番号`を確認します。
4. CloudStudio に戻り、認証番号を入力して`確認`をクリックします。
5. `次へ`ボタンをクリックします。
6. メールアドレスに送信された ID を確認します。

## パスワードを探す

1. CloudStudio ログインページで**[パスワードを探す](https://cstudio.app/forgot/pw)**をクリックします。
2. メールアドレスを入力し、コード転送をクリックします。 例) `cloudstudio@cloudraw.kr`
3. 入力したメールアドレスから送信された`6桁の認証番号`を確認します。
4. CloudStudio に戻り、認証番号を入力して`確認`をクリックします。
5. `パスワードのリセット`ボタンをクリックします。
6. 新しいパスワードを入力します。
7. 確認のため、もう一度新しいパスワードを入力します
8. `パスワードのリセット`ボタンを押して完了します。

<div className={styles.imgWrap}>
    <img src={findPassword} alt="find password img" />

<p>&nbsp;</p>

:::info
パスワードは、以下の条件を満たす必要があります。

-   英字の大文字·数字·特殊文字のみ使用
-   英小文字で始まり
-   8 桁以上 20 桁以内

:::
