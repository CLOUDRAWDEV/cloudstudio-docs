📁 フォルダは複数のプロジェクトを管理できるようにする機能です。
生成したフォルダに プロジェクトを入れて管理してみましょう！

<br />
<br />

<div className={styles.imgWrap}>
    <img src={createFolder} alt="이미지" />
</div>

1. 新しいフォルダを生成するために CloudStudio 内のテンプレートページで`フォルダの生成`アイコンをクリックします。
2. フォルダを追加できるポップアップが表示されたら`フォルダ名`과 `フォルダの説明`を作成します。フォルダの説明は省略できます。
   フォルダ名とフォルダの説明は、「以下の条件」を満たす必要があります。

    > フォルダ名

    - 英語、ハングル、数字を許可
    - 2 ~ 12 桁
    - 空白なしで

    > フォルダの説明

    - 最大 100 文字まで作成可能

3. ポップアップウィンドウの内容の作成が終わったら、`作成する`ボタンをクリックします。
4. 生成されたフォルダのリストは`テンプレート > マイフォルダ`で確認することができます。
