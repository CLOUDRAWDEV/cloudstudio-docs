## Register with Github

To manage the version of your canvas in Cloud Studio, integration with a Version Control System (VCS) is required.<br/>
Additionally, VCS account integration must precede the linking of a Terraform Cloud account for project validation and deployment.<br/>
This page explains how to register your Github account as a VCS

<br />

<details>
    <summary>What is Github?</summary>
    <h5>
        Github is a platform for hosting and managing software development
        projects.
    </h5>
</details>

<p>&nbsp;</p>

1. After signing up and logging into [Github](https://github.com/), click on the Profile Icon - Settings in the top right corner.

<img width="100%" src={github01} alt="otp img" />

<p>&nbsp;</p>

2. Click on `Developer settings`at the bottom of the left sidebar.

<img width="100%" src={github02} alt="otp img" />

<p>&nbsp;</p>

3. In the left sidebar, click on `Personal access tokens - Tokens (classic)`,
   then select `Generate new token - Generate new token (classic)` on the right.

<img width="100%" src={github03} alt="otp img" />

<p>&nbsp;</p>

4. In the `Select scopes` section at the bottom, choose the `repo` option to select the permissions for the Github token,
   and then click the `Generate token` button at the bottom to create the token.

<img width="100%" src={github04} alt="otp img" />

<p>&nbsp;</p>

:::caution

The issued token cannot be viewed again, so make sure to save it separately.

:::

<p>&nbsp;</p>

5. In Cloud Studio, click on `My Page - Plugin`, then click the `+` button to link your Github account.

<img width="100%" src={github05} alt="otp img" />

<p>&nbsp;</p>

6. Enter your Github `Username` and the issued `Token`, then click the `Add` button.

<img width="100%" src={github06} alt="otp img" />

<p>&nbsp;</p>

7. Once you have entered a valid Github `Username` and the issued `Token`, the Github integration will be successfully completed.

<img width="100%" src={github07} alt="otp img" />
<br />

:::caution

To integrate with the Github Enterprise version, additional inputs for the hosted `HTTP URL` and `API URL` are required.

:::
