## Register with Terraform Cloud

To validate or deploy the infrastructure drawn in the Cloud Studio canvas, integration with a Terraform Cloud account is required. Additionally, integration with a Terraform Cloud account must be preceded by linking either a [GitHub account](/docs/plugin/vcs/add-github) or a [GitLab account](/docs/plugin/vcs/add-gitlab).
<br />

<details>
    <summary>What is Terraform Cloud?</summary>
    <h5>
    Terraform Cloud is a managed service for Terraform that supports infrastructure automation and collaboration. 
    It centralizes the management of remote state storage, planning, and applying changes, while facilitating teamwork.
    </h5>
</details>

<p>&nbsp;</p>

After signing up and logging into [Terraform Cloud](https://app.terraform.io/session), click on the Profile Icon - 
Account settings in the left sidebar.

<img width="100%" src={terraformcloud01} alt="otp img" />

<p>&nbsp;</p>

2. Click on `Tokens` at the bottom of the left sidebar, then click the `Create an API token` button on the right.

<img width="100%" src={terraformcloud02} alt="otp img" />

<p>&nbsp;</p>

3. Enter a `Description` to identify the token later, set the `Token Expiration`, and then click the `Generate token` button.

<img width="100%" src={terraformcloud03} alt="otp img" />

<p>&nbsp;</p>

4. Review the issued token.

<img width="100%" src={terraformcloud04} alt="otp img" />

<p>&nbsp;</p>

:::caution

The issued token cannot be viewed again, so make sure to save it securely.

:::

<p>&nbsp;</p>

5. Return to the main screen of [Terraform Cloud](https://app.terraform.io/app/organizations) and click the `Create organization` button.

<img width="100%" src={terraformcloud05} alt="otp img" />

<p>&nbsp;</p>

6. Enter the `Organization name` and click the `Create organization` button.

<img width="100%" src={terraformcloud06} alt="otp img" />

<p>&nbsp;</p>

:::caution

The `Organization name` must be unique globally.

:::

<p>&nbsp;</p>

7. In Cloud Studio, click on `My Info - Plugin Management`, then click the `+` button to link your Terraform Cloud account.

<img width="100%" src={github07} alt="otp img" />

<p>&nbsp;</p>

:::caution

Integration with a Terraform Cloud account can only be done after linking either a GitHub account or a GitLab account in advance.

:::

<p>&nbsp;</p>

8. Enter the issued Terraform Cloud `Token`, then click the `Register` button.

<img width="100%" src={terraformcloud07} alt="otp img" />

<p>&nbsp;</p>

9. After integration, select the `Organization` created in Terraform Cloud, and all tasks will be completed.

<img width="100%" src={terraformcloud08} alt="otp img" />

<p>&nbsp;</p>
