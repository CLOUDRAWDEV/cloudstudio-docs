## Find User ID

1. Click the **[Forgot ID](https://cstudio.app/forgot/id)**button on the Cloud Studio signin page.
2. Enter your Email Address and click Send Code (e.g., cloudstudio@cloudraw.kr).
3. Check your Email for the `verification code` that has been sent to you.
4. Return to Cloud Studio, enter the verification code, and click `Confirm`.
5. Click the `Next` button.
6. Check your email for the User ID that has been sent to you.

## Find Password

1. Click the **[Forgot password](https://cstudio.app/forgot/pw)** button on the Cloud Studio signin page.
2. Enter your Email address and click Send Code (e.g., cloudstudio@cloudraw.kr).
3. Check your Email for the `verification code` that has been sent to you.
4. Return to Cloud Studio, enter the verification code, and click `Confirm`.
5. Click the `Reset Password` button.
6. Enter your New Password.
7. Re-Enter your New Password to confirm.
8. Click the `Reset Password` button to complete the process.

<div className={styles.imgWrap}>
    <img src={findPassword} alt="find password img" />
</div>

:::info
The Password needs to satisfy the following conditions:

-   Start with a lowercase letter.
-   Use only English letters and numbers.
-   The Password should be between 6 and 16 characters in length.

:::
