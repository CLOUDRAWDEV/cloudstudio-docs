Cloud Studio goes beyond just providing visual infrastructure architecture,it also offers the capability to
`deploy the infrastructure drawn on the canvas to an actual CSP`. This allows users to easily build their cloud infrastructure.

## Deploy Validation Check {#deploy-test}

This is a step where the syntax and settings of IaC are virtually validated to ensure correctness.
It does not involve deployment to an actual CSP.
(It functions similarly to the --dry-run option.)

1. To perform a deployment test, click `Deploy Validation Check > Run Deploy Validation Check`.

<div className={styles.imgWrap}>
    <img src={deploy03} width="200px" alt="배포이미지" />
</div>

<br />

2. Select `Credential`.
3. Click the `Apply Validation Check` button.

    <img src={deploy02} alt="배포이미지" />
</div>

## Deploy History {#tedeployst-history}

Deploy History shows records of `actually deployed infrastructure`.
You can view the `latest deployment history` under each tab: DEV (Development) / TEST (Testing) / PROD (Production).

1. To view the deployment history, click `Deploy > Deploy History`.

<div className={styles.imgWrap}>
    <img src={deploy01} width="500px" alt="배포이미지" />
</div>

<br />

2. Review the most recent deployment history. To delete a record, select the desired history and click the `Delete` button.
