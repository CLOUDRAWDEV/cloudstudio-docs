## Edit Project Detail

### Edit Project name

1. To edit the project details, `click the template you want to edit` in My Templates.
2. When the pop-up appears, click the `Edit Icon` next to Project Name.
3. Modify the `Project Name`.
4. Click the `Edit Icon` again to finalize the changes.

### Edit Project Description

1. To edit the Project Description, click the project you want to modify in `My Templates`.
2. When the pop-up appears, click the `Edit Icon` next to Project Description.
3. Edit the `Project Description`.
4. Click the Edit Icon again to finalize the changes.

## Copy and Delete Projects

### Copy a Project

1. To copy a project, click the `Add Icon` next to the project you want to modify in My Templates.
2. Click `Copy a Project`.
3. When the pop-up window appears, select the folder where you want to copy the project.
4. Enter the new project name.

> -   English, Korean, Numbers allowed.
> -   2 to 20 digits.

5. Enter Project Description.

> -   Within 100 characters.

6. Click Copy.

### Delete Project

1. To delete a project, click the `More Options Icon` next to the project you want to delete in My Templates.
2. Click `Delete Project`.
3. In the pop-up window, write the project name.
4. Click the `Delete` button.
