Click the link below to sign up for Cloud Studio membership.

[Start CloudStudio](https://cstudio.app/signin)

1. Click the SignUp button.
2. Enter your Email Address (e.g., cloudstudio@cloudraw.kr).
3. Click the `Send Code` button and check the `verification code` sent to your email.
4. Return to Cloud Studio, enter the verification code, and click the `Confirm` button to complete the email verification process.

:::info
You must enter your verification code within `3 minutes`.
If the verification time expires, please click the **Resend Icon** to receive a new code.
:::verification

5. Enter the User ID you wish to use, and click the `Check` button.

:::info
The User ID needs to satisfy the following conditions:

-   Start with a lowercase letter.
-   Use only English letters and numbers.
-   Use English (lowercase) and numbers, from 6 to 16 characters

:::

6. Agree to the Terms of Use.
7. Click the `SignUp` button.
8. A temporary password has been emailed to you.
