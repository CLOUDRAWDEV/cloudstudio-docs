Click the link below to sign up for Cloud Studio membership.

[Start CloudStudio](https://cstudio.app/signin)

1. Click the Sign Up button.
2. Enter your email address (e.g., cloudstudio@cloudraw.kr).
3. Click the 'Send Code' button and check the 'six-digit verification code' sent to your email.
4. Return to Cloud Studio, enter the verification code, and click the 'Verify' button to complete the email verification process.

:::info
You must enter your verification code within `3 minutes`.
If the verification time expires, please click the Resend Icon to receive a new code.
:::

5. Enter the User ID you wish to use, and click the 'Check for Duplicates' button.

:::info
The User ID needs to satisfy the following conditions:

-   Start with a lowercase letter.
-   Use only English letters and numbers.
-   The User ID should be between 6 and 16 characters in length.

:::

6. Agree to the Terms and Conditions.
7. Click the 'Sign Up' button.
8. A temporary password will be sent to your registered email.
