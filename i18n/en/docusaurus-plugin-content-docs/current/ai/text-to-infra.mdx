CloudStudio not only allows users to draw infrastructure directly
but also provides a feature that automatically generates infrastructure based on text input describing the desired infrastructure structure.

<p>&nbsp;</p>

## Text to Infra

1. To generate infrastructure using text, click the `Templates` - `Create Project` button, and then click the `AI Generate Template - Preview` button.

    <div className={styles.imgWrap}>
        <img src={textToInfra01} width="600px" alt="배포이미지" />
    </div>

<p>&nbsp;</p>
<p>&nbsp;</p>

2. After entering the information for the infrastructure you want to create in the text box at the bottom, click the `Generate` button.

    <div className={styles.imgWrap}>
        <img src={textToInfra02} width="600px" alt="배포이미지" />
    </div>
