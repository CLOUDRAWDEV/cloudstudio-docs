The Canvas is <highlight color="#206FD6">the core service of CloudStudio</highlight>
that allows you to `drag & drop` resources to design cloud infrastructure, manage versions, and deploy them.

✂️ Feel free to edit the Canvas as you like!

## Resource Drag & Drop {#drag-drop}

`Drag & Drop` is the simplest feature for configuring your canvas.
Drag the desired resource from the left Resource List and drop it onto the canvas!
Resources that cannot be connected will bounce back with a red line.

## Editing Resource Options {#option-edit}

Double-click a resource to open an edit panel where you can change the resource settings.
Click the Details Icon to view the description of the resource.
❓Hover over the Question Mark Icon to see descriptions for each option.
Use these descriptions to configure the settings to your preferences.

## Fixing Errors {#error-correction}

🔴 A red dot indicates that there is an error with the resource. <highlight color="#FF7272">
If there are errors, you cannot save or deploy the canvas.</highlight>
To resolve errors, double-click the resource with the error on the canvas to open the canvas error window.
In the canvas error window, double-clicking a specific error will navigate you to the location of the error.
Refer to the error cause and fix the issues accordingly.

<div className={styles.imgWrap}>
    <img src={error} alt="Error Correction Image" />
</div>

<p>&nbsp;</p>

### Types of Errors

| type  | description                                         |
| ----- | --------------------------------------------------- |
| value | Errors caused by incorrect input values             |
| group | Errors when a resource does not belong to any group |
| line  | Errors when required resources are not connected    |
