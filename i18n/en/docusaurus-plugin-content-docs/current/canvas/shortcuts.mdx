Here are the `keyboard shortcuts` available in CloudStudio Canvas.

| Shortcut                  | Description     |
| ------------------------- | --------------- |
| Ctrl + Mouse Scroll       | Zoom in and out |
| Space + Left Click + Drag | Pan the view    |
| Delete                    | Delete          |
| Ctrl + Z                  | Undo            |
| Ctrl + C                  | Copy            |
| Ctrl + V                  | Paste           |
