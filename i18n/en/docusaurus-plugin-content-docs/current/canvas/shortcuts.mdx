Here are the ```keyboard shortcuts``` available in Cloud Studio Canvas.

| Shortcut | Description           |
| ---------- | -------------- | 
| ctrl + Mouse Scroll	 | Zoom in and out |
| space key + Left Click + Drag	 | Pan the view
 |
| delete key | Delete |
| ctrl key + z | Undo |
| ctrl key + c | Copy |
| ctrl key + v | Paste |
