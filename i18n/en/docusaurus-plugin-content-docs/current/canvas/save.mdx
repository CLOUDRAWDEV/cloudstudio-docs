## Save {#save}

There are `two ways` to save the canvas.

-   [Save](#save)
-   [Save Version](#version)

Saving refers to saving the current state of the canvas. Saving the canvas is similar to saving a project. 
The image displayed in the project thumbnail reflects the canvas after saving.

<div className={styles.imgWrap}>
    <img src={save05} width="300px" alt="Save image" />
</div>

<br />

The process is simple. Click `File > Save`.

<div className={styles.imgWrap}>
    <img src={save01} width="200px" alt="저장하기 이미지" />
</div>

<p>&nbsp;</p>

:::caution

If there are 🔴 errors in the canvas, only the image will be saved. Be sure to resolve all errors before saving. For instructions on how to resolve errors, please refer to [here](/docs/canvas/edit-canvas).

:::

## Version Save and Version History {#version}

Version saving is a way to save the canvas at a specific point in time. 
By saving a version, you can return to the desired state of the canvas using version history.


### Version Save {#version-save}

1. To save a version, click `Version Management > Save Version`.


<div className={styles.imgWrap}>
    <img src={save02} width="200px" alt="Version save image" />
</div>

<br />

2. Enter a `Tag Message`. The tag message you enter will appear as the `version title` in the version history.
<div className={styles.imgWrap}>
    <img src={save03} alt="Version save image" />
</div>

<br />

3. Click `Save`.

<p>&nbsp;</p>

### Version History {#version-history}

1. To view version history, click `Version Management > Version History`.

<div className={styles.imgWrap}>
    <img src={save06} width="200px" alt="Version history image" />
</div>

<br />

2. Review the saved versions in the version history.

:::note

-   <highlight color="#206FD6">Latest</highlight>: The most recently saved history.
-   <highlight color="#AD28DC">Manual</highlight>: The version save history.
-   <highlight color="#02CDE8">Planed</highlight>: The pre-deployment inspection history.
-   <highlight color="#5C4AF6">Applied</highlight>: The completed deployment history.

:::

<br />

3. Click on the canvas version you want to revert to.

<div className={styles.imgWrap}>
    <img src={save04} alt="Version history image" />
</div>
