## Resource Overview

A resource group is a container that holds resources related to an Azure solution. 
A resource group can contain all resources for a solution or only those resources that you want to manage as a group.
Decide how to assign resources to resource groups based on what best fits your organization.
In general, you add resources that share the same lifecycle for the same resource group so that you can easily deploy, update, and delete them as a group.
