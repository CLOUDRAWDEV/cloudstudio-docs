## Resource Overview

Log Analytics is a tool used within the Azure Portal to edit and execute log queries against data in Azure Monitor log stores. 
You can write simple queries that return record sets and then use Log Analytics features to sort, filter, and analyze the data. 
Alternatively, you can create advanced queries to perform statistical analysis and visualize results in charts to identify specific trends. 
Whether handling query results interactively or using them in conjunction with other Azure Monitor features like query alerts or workbook logs, Log Analytics is used for query creation and testing.

## Resource Setting Values

- `retention_in_days`: The workspace data retention in days - `30` ~ `730`

- `internet_query_enabled`: Enable querying over the public Internet - `true`, `false`

- `internet_ingestion_enabled`: Enable ingestion over the public Internet - `true`, `false`

- `linked_automation_account_names`: List of readable resource name that will be linked to the workspace
