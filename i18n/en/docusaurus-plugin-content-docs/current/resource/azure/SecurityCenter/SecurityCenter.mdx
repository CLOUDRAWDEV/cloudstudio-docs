## Resource Overview

Azure Security Center is a cloud-based service provided by Microsoft that helps organizations prevent, detect, and respond for security threats across hybrid cloud workloads. 
It offers a centralized dashboard that provides visibility into the security posture of resources hosted in both Microsoft Azure and on-premises environments. 
Azure Security Center continuously monitors your environment and provides security recommendations, vulnerability assessments, threat intelligence, and security alerts to help protect against threats and identify potential vulnerabilities.

-   `linked_log_analytics_workspace`: Name of the Log Analytics Workspace where data will be stored

-    `assessment_policy`
      -    `display_name`: The display name of the Security center assessment
      -    `severity`: The severity level of the Security center assessment
      -    `description`: The description of the Security center assessment

-   `assessment`

-   `linked_advanced_threat_protection`: The name of Security Center Advanced Threat Protection
