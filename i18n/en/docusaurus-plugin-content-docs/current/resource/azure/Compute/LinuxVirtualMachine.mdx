## Resource Overview

An Azure Linux Virtual Machine is a virtual machine (VM) running a Linux operating system on Microsoft's Azure cloud computing platform. 
By running a Linux VM on Azure, users can leverage the scalability, reliability, and global reach of Microsoft's cloud platform while using their preferred Linux distribution.

## Resource Configuration Values

- `included_vnet_name`: The name of the Virtual Network that includes the subnet.
- `size`: The size configuration of the Virtual Machine for running workloads.
- `source_image`:
  - `enabled_custom_image`: Whether to use a custom image ID, `true` or `false`.
  - `publisher`: The publisher of the image used for VM creation.
  - `offer`: The ID of the image used for VM creation.
  - `sku`: The pricing tier of the image used for VM creation.
- `enable_ssh_key_authentication`: Whether to use an SSH key for administrator account authentication.
- `admin_username`: The administrator username for the VM.
- `admin_password`: The administrator password for the VM.
- `on_disk_caching`: The type of caching for the internal OS disk.
- `storage_account_type`: The type of storage used for backing up the internal OS disk.
- `disk_size_gb`: The size of the internal OS disk in GB, ranging from `1` ~ `32767`.
- `disk_attachment`: The name of the disk attachment.
- `linked_network_interface_names`: The names of the Network Interfaces.
- `primary_network_interface_name`: The name of the Network Interface connected to the Virtual Machine.
- `tag`: A tag used to categorize resources, up to 512 characters, in the format `key:value`.
