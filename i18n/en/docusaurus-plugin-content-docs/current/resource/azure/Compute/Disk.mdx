## Resource Overview

Azure Virtual Machines are virtual servers running in the cloud. 
A virtual machine includes one or more disks, which are used to store the operating system (OS) and data. 
Disks serve as persistent storage for the virtual machine.

## Resource Setting Values

- `storage_type`: The type of storage account to back the Disk
- `size_gb`: The size of the Disk in GB - `1` to `32767`
- `enabled_public_network_access`: Allow public network access for the Disk - `true`, `false`
- `tag` : Tag to categorize resources - up to 512 characters, `key:value`
