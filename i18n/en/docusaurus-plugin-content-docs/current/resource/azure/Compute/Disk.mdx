## Resource Overview

Azure Virtual Machines are virtual servers running in the cloud. 
Each virtual machine includes one or more disks, which are used to store the operating system (OS) and data. 
These disks serve as permanent storage for the virtual machine.

## Resource Configuration Values

- `storage_type`: The type of storage for backing up the disk
- `size_gb`: The size of the disk in GB - `1` to `32767`
- `enabled_public_network_access`: Allow public network access to the disk - `true`, `false`
- `tag`: A tag to categorize the resource - up to 512 characters, `key`:`value`
