## Resource Overview

Azure WindowsServer Virtual Machine은 Microsoft의 Azure 클라우드 컴퓨팅 플랫폼에서 WindowsServer 운영 체제를 실행하는 가상 머신(VM)입니다.
Azure에서 WindowsServer 가상 머신을 실행함으로써 사용자는 선호하는 Linux 배포를 사용하면서 Microsoft 클라우드 플랫폼의 확장성, 안정성 및 글로벌 범위를 활용할 수 있습니다.

## Resource Configuration Values

- `included_vnet_name`: The name of the Virtual Network that includes the subnet.
- `size`: The size configuration of the Virtual Machine for running workloads.
- `source_image`:
  - `enabled_custom_image`: Whether to use a custom image ID, `true` or `false`.
  - `publisher`: The publisher of the image used for creating the VM.
  - `offer`: The offer ID of the image used for creating the VM.
  - `sku`: The pricing tier of the image used for creating the VM.
- `computer_name`: The host name of the Virtual Machine.
- `admin_username`: The administrator username for the VM.
- `admin_password`: The administrator password for the VM.
- `on_disk_caching`: The type of caching for the internal OS disk.
- `storage_account_type`: The type of storage used for backing up the internal OS disk.
- `disk_size_gb`: The size of the internal OS disk in GB, ranging from `1` to `32767`.
- `disk_attachment`: The name of the disk attachment.
- `linked_network_interface_names`: The names of the network interfaces.
- `primary_network_interface_name`: The name of the primary network interface connected to the Virtual Machine.
- `tag`: A tag used to categorize resources, up to 512 characters, in the format `key:value`.
