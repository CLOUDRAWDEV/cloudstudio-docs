## Resource Overview

Azure Container Registry is a managed registry service based on the open-source Docker Registry 2.0. 
It provides capabilities to create and maintain a container registry in Azure for storing and managing container images and related artifacts.

### Parent Resource

-    `sku`: SKU type for performance and scalability of the Container Registry - `Basic`, `Standard`, `Premium`

-    `admin_enabled`: Admin enabled - `true`, `false`

-    `public_network_access_enabled`: Allow public network access - `true`, `false`

-    `network_rule_set`: Can be created if the SKU type is Premium
      - `default_action`: Action for requests that do not match any rules - `Deny`, `Allow`
      - `ip_ranges`: IP address CIDR for requests that match the rule
-    `tag`: Tag for categorizing resources - up to 512 characters, `key`:`value`
