## Resource Overview

Azure Database for MySQL - Flexible Server is a fully managed production-ready database service designed to offer greater control and flexibility over database management features and configuration settings. Flexible Server is most suitable for:

- Ease of deployment for features like backups, high availability, security, and monitoring, along with inherent scalability and reduced database management overhead.
- Application development requiring a community edition of MySQL with better control and customization.
- Production workloads with high availability through zone redundancy, zonal deployments, and managed maintenance windows.
- Streamlined development environments.
- Enterprise-grade security, compliance, and privacy.

## Resource Configuration Values

-   `version` : MySQL version to use - `5.7`, ``8.0.21`

-   `sku_name` : Pricing tier name for MySQL, a combination of Tier + compute generation + number of cores - `azureMysqlFlexibleSku`

-   `storage_gb` : Capacity of the MySQL Server (GB) - `storageGB`

-   `iops` : Storage IOPS for MySQL Flexible Server - `360` ~ `20000`

-   `admin_username` : Administrator username for the Server when the creation mode is default

-   `admin_password` : Administrator login password for the Server when the creation mode is default

-   `auto_grow_enabled` : Enable automatic storage growth - `true` , `false`

-   `availability_zone` : Specifies the availability zone. Possible values are 1, 2, 3. - `1` ~ `3`

-   `enable_high_availability` : Enable high availability for MySQL Flexible Server - `true` , `false`

-   `high_availability_mode` : High availability mode for MySQL Flexible Server - `SameZone` , `ZoneRedundant`

-   `standby_availability_zone` : Specifies the availability zone for the standby server. 
Possible values are 1, 2, 3. Must be different from the primary availability zone. - `1` ~ `3`

-   `backup_retention_days` : Number of days to retain backups for MySQL Flexible Server. Possible values are 1 to 35 days. - `1` ~ `35`

-   `public_network_access_enabled` : Allow public network access to the Server - `true` , `false`

-   `configuration`

    -   `name` : Name of the configuration for MySQL Flexible Server
    -   `param_value` : Value of the configuration for MySQL Flexible Server

-   `database`

    -   `charset` : Character set for the MySQL Database - `azureMysqlChartSet`
    -   `collation` : Collation for the MySQL Database - `azureMysqlCol`

-   `firewall_rule`
    -   `start_ip_address` : Starting IP address for the firewall rule
    -   `end_ip_address` : Ending IP address for the firewall rule
-   `tag` : Tag for categorizing resources - maximum 512 characters, `key`:`value`
