## Resource Setting Values

-   `included_subnet_name` : The name of the subnet that includes the Network Interface
-   `ip_configuration`
    -   `is_primary` : Whether the IP configuration is set as primary - `true`, `false`
    -   `private_ip_address_allocation` : Method of private IP address allocation
    -   `enabled_public_ip` : Whether to use a public IP - `true`, `false`
    -   `linked_public_ip_name` : Name of the public IP to be used
-   `tag` : Tag used to categorize the resource - up to 512 characters, `key`:`value`
