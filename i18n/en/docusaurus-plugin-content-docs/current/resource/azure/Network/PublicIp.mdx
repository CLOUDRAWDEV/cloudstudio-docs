## Resource Overview

Azure Public IP(공용 IP)는 인터넷에서 직접 액세스할 수 있는 고유한 IP 주소입니다. 
Azure 리소스, 예를 들어 가상 머신, Load Balancer, VPN Gateway 등과 같이 인터넷으로 노출되는 리소스에 사용됩니다. 
이를 통해 Azure 리소스에 대한 인터넷 트래픽을 보낼 수 있고, 인터넷에서 해당 리소스에 접근할 수 있게 됩니다.

## Resource Setting Values

-   `sku`: Pricing tier for the public IP - `Standard`, `Basic`
-   `allocation_method`: Address allocation method for the public IP - `Static`, `Dynamic`
-   `tag`: A tag for categorizing the resource - maximum 512 characters, `key`: `value`
