## Resource Overview

Azure Public IP is a unique IP address that is accessible directly from the internet. 
It is used for Azure resources that are exposed to the internet, such as virtual machines, Load Balancers, and VPN Gateways. 
This allows internet traffic to be directed to Azure resources and enables access to those resources from the internet.

## Resource Setting Values

-   `sku`: The SKU of Public Ip - `Standard`, `Basic`
-   `allocation_method`: Public Ip address allocation method - `Static`, `Dynamic`
-   `tag` : Tag to categorize resources - up to 512 characters, `key`:`value`
