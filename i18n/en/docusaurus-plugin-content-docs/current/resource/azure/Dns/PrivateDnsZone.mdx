## Resource Overview

Azure Private DNS provides a reliable and trustworthy DNS service that allows you to manage and resolve domain names within a virtual network without the need to add a custom DNS solution. 
By using private DNS zones, you can use your own custom domain names instead of the default names provided by Azure. Records in a private DNS zone cannot be resolved from the internet. 
DNS resolution for private DNS zones only works within the connected virtual networks.
