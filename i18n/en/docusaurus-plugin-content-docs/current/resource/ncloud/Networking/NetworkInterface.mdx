## Resource Overview

This service creates network interfaces that can be assigned to servers. Servers with applied network interfaces can use subnets to enable independent network communication.

## Associated Resources

### Parent Resource

-   [**Subnet**](https://docs.cstudio.app/en/docs/resource/ncloud/Networking/Subnet)

### Connected Resources

-   [**Access Control Group**](https://docs.cstudio.app/en/docs/resource/ncloud/Security/AccessControlGroup)
-   [**Server**](https://docs.cstudio.app/en/docs/resource/ncloud/Compute/Server)

## Resource Setting Values

-   `included_subnet_name` : The name of Subnet to included with the Network interface
-   `linked_acg_names` : The names of ACG to apply the Network interface is located
