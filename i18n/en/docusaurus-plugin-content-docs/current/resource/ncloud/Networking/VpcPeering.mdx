## Resource Overview

This resource sets up VPC peering for communication between VPCs.

## Associated Resources

### Parent Resource

-   [**Region**](https://docs.cstudio.app/en/docs/resource/ncloud/Base/Region)

### Connected Resources

-   [**Vpc**](https://docs.cstudio.app/en/docs/resource/ncloud/Networking/Vpc)

## Resource Setting Values

-   `linked_vpc_name_01` : The name of VPC from which the request is sent
-   `linked_vpc_name_02` : The name of VPC to receive requests
