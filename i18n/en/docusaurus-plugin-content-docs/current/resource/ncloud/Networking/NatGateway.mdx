## Resource Overview

This service allows multiple servers to connect through a single public IP while blocking direct external access. It enables servers with private IPs to communicate with external servers while maintaining security by preventing direct external access.

## Associated Resources

### Parent Resource

-   [**Vpc**](https://docs.cstudio.app/docs/resource/ncloud/Networking/Vpc)

## Resource Setting Values

-   `included_vpc_name` : The name of VPC to create the Nat gateway
-   `zone` : Available zone where the Nat gateway will be placed physically - `KR-1`, `KR-2`
