## Resource Overview

A VPC subnet that allows for detailed segmentation based on network purposes.

-   [**Network Acl**](https://docs.cstudio.app/docs/resource/ncloud/Network/NetworkAcl)

## Resource Setting Values

-   `subnet_cidr` : VPC 주소 범위 내의 서브넷 IP 주소 범위
-   `zone` : 서브넷이 위치할 물리적 지역 - `KR-1`, `KR-2`
-   `subnet_type` : 퍼블릭 혹은 프라이빗 서브넷 설정 - `PUBLIC`, `PRIVATE`
-   `usage_type` : 로드 밸런서 사용 여부 설정 - `GEN`, `BM`
-   `included_vpc_name` : 서브넷이 위치할 VPC 이름
-   `linked_nacl_name` : 서브넷에 대한 접근 제어를 위한 Network access control list
