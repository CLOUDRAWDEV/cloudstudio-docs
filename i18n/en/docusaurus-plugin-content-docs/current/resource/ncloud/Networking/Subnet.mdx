## Resource Overview

A VPC subnet that allows for detailed segmentation based on network purposes.

## Resource Setting Values

-   `subnet_cidr` : The subnet address ranges within the range of VPC addresses
-   `zone` : Available zone where the Subnet will be placed physically - `KR-1`, `KR-2`
-   `subnet_type` : Internet connectivity, public or private - `PUBLIC`, `PRIVATE`
-   `usage_type` : Enable or disable Load balancer - `GEN`, `BM`
-   `included_vpc_name` : The name of VPC where the Subnet will be located
-   `linked_nacl_name` : Network access control list for controlling access to Subnet
