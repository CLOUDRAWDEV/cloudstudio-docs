## Resource Overview

This is a storage service that connects multiple servers to a network to enable data sharing among numerous users. It provides network-attached storage space that is ideal for sharing data among multiple users.

## Resource Setting Values

-   `volume_name_postfix` : The name of NAS volume, alphanumeric characters are allowed, within 3 to 20 characters
-   `volume_size` : The size of volume in 100GB, From 500 to 10,000
-   `volume_allotment_protocol_type` : The type of protocol, Mountable server: NFS-Linux/CIFS-Windows - `NFS`, `CIFS`
