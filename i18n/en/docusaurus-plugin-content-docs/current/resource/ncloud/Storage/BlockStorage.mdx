## Resource Overview

This is an HDD or SSD storage service offered in addition to the server's primary storage. It can be quickly created and instantly attached to the server, and when no longer needed, it can be returned immediately.

## Associated Resources

### Parent Resource

-   [**Subnet**](https://docs.cstudio.app/en/docs/resource/ncloud/Networking/Subnet)

### Connected Resources

-   [**Server**](https://docs.cstudio.app/en/docs/resource/ncloud/Compute/Server)

## Resource Setting Values

-   `linked_server_name` : The name of server to associate with the Block storage
-   `storage_size` : The size of Block storage in GB, From 10 to 16380
-   `storage_disk_type` : The type of Block storage disk, SSD or HDD - `SSD`, `HDD`
-   `snapshot_name` : Name of the snapshot to be created
