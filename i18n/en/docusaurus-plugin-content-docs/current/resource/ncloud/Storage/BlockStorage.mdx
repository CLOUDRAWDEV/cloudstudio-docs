## Resource Overview

서버의 기본 스토리지 외에 추가로 제공하는 HDD 또는 SSD 스토리지 서비스입니다. 단시간에 생성하여 즉시 서버와 연결할 수 있고 필요하지 않을 경우 바로 반납할 수 있습니다.

-   [**Server**](https://docs.cstudio.app/en/docs/resource/ncloud/Compute/Server)

## Resource Setting Values

-   `linked_server_name` : The name of server to associate with the Block storage
-   `storage_size` : The size of Block storage in GB, From 10 to 16380
-   `storage_disk_type` : The type of Block storage disk, SSD or HDD - `SSD`, `HDD`
-   `snapshot_name` : Name of the snapshot to be created
