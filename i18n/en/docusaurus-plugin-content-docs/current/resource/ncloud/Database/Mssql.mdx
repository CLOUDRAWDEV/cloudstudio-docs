## Resource Overview

Cloud DB for MSSQL allows you to easily build and automatically manage MSSQL databases. It operates reliably with optimized settings from Naver and is a fully managed cloud database service that automatically recovers from failures.

## Resource Setting Values

-   `enable_high_availability` : Whether to use high availability
-   `user_name` : The MSSQL access user ID
-   `user_password` : The MSSQL access user password
-   `image_product_code` : The Image product code to determine the MSSQL instance server image specification to create
-   `product_code` : The product code to determine the MSSQL instance server image specification to create
-   `data_storage_type` : The data storage type  - `SSD`, `HDD`
-   `backup_file_retention_period` : The backup file retention period. The backups are performed daily and backup files are stored in separate backup storage - `1~30`
-   `enable_automatic_backup` : Whether to use automatic backup
-   `port` : The TCP port number to access MSSQL - `3306~20000`
-   `character_set_name` : The DB character set. The DB character set can be selected from Korean and English collation
