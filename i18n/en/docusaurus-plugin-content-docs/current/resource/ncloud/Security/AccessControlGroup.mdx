## Resource Overview

This is an IP/Port-based firewall service that allows for network access control and management between servers.

## Associated Resources

### Parent Resource

-   [**VPC**](https://docs.cstudio.app/en/docs/resource/ncloud/Networking/Vpc)

### Connected Resources

-   [**Network Interface**](https://docs.cstudio.app/en/docs/resource/ncloud/Networking/NetworkInterface)
-   [**Auto Scaling**](https://docs.cstudio.app/en/docs/resource/ncloud/Compute/AutoScaling)

## Resource Setting Values

-   `included_vpc_name` : ACG를 생성할 VPC 이름
-   `acg_description` : ACG에 대한 설명
-   `inbound_rule`
    -   `name` : inbound rule에 대한 이름
    -   `protocol` : 인바운드 트래픽의 프로토콜 - `TCP`, `UDP`, `ICMP`
    -   `rule_apply_method` : 인바운드 규칙 적용 방법 - `ip_block`, `source_name`
    -   `ip_block` : 인바운드 트래픽 목적지, CDIR 형식의 IP 대역
    -   `port_range` : 인바운드 규칙을 적용할 포트 범위, 1 ~ 65535의 범위(범위 설정시: 100-200)

-   `outbound_rule`
    -   `name` : outbound rule에 대한 이름
    -   `protocol` : 아웃바운드 트래픽의 프로토콜 - `TCP`, `UDP`, `ICMP`
    -   `rule_apply_method` : 아웃바운드 규칙 적용 방법 - `ip_block`, `source_name`
    -   `ip_block` : 인바운드 트래픽 목적지, CDIR 형식의 IP 대역
    -   `port_range` : 인바운드 규칙을 적용할 포트 범위, 1 ~ 65535의 범위(범위 설정시: 100-200)
