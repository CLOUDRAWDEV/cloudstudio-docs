## Resource Overview

This resource is used to specify the region where Naver Cloud resources will be deployed. It supports Korea (KR), Japan (JPN), and Singapore (SGN).

## Associated Resources

### Parent Resource

-   [**Base**](https://docs.cstudio.app/docs/resource/ncloud/Base/)

### Child Resource

-   [**Network Vpc**](https://docs.cstudio.app/docs/resource/ncloud/Network/Vpc)
-   [**Network Vpc Peering**](https://docs.cstudio.app/docs/resource/ncloud/Network/VpcPeering)

## Resource Setting Values

-   `region` : Deployment region
