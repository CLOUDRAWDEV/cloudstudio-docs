## Resource Overview

This computing resource provides the flexibility of virtualization without the need to manage physical hardware. It is efficient in terms of time and cost by creating servers in a cloud environment, making it suitable for scenarios where infrastructure management is burdensome or when temporary access to a large number of server resources is needed.

## Resource Setting Values

-   `include_subnet_name` : The name of Subnet to which to create the Server
-   `server_image_product_code` : The code of OS image to use
-   `server_product_code` : The code for hardware configuration on the server
-   `login_key_name` : The Login key name to use
-   `init_script` : The name of Init script to apply
-   `is_encrypted_base_block_storage_volume` : Whether to encrypt basic Block storage if server image is RHV
-   `linked_network_interface_name` : The Network interface to apply with the server
