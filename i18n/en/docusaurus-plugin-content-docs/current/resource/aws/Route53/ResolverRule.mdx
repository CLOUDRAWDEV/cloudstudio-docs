## Resource Overview

Rules used by the Resolver to forward queries for specified domain names over the network

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### Connected Resources

-   [**Vpc**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Vpc)
-   [**ResolverEndpoint**](https://docs.cstudio.app/en/docs/resource/aws/Route53/ResolverEndpoint)

## Resource Setting Values

-   `rule_type` : Rule type - `Forward`, `system`
-   `domain_name` : Domain name to which DNS queries are forwarded
-   `linked_vpc_names` : VPC name using rules
-   `linked_resolver_endpoint_name` : Outbound endpoint name used by Resolver
-   `target_ip` : Target IP address
-   `target_ip.name` : Name of the target IP address
-   `target_ip.ip` : IP of Target IP address
-   `target_ip.port` : Port of Target IP address - `1`~`65535`
