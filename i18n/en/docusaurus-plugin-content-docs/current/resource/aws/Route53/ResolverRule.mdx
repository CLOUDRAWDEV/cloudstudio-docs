## 리소스 개요

Resolver가 지정된 도메인 이름의 쿼리를 네트워크로 전달할 때 사용하는 규칙

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### 연결 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)
-   [**ResolverEndpoint**](https://docs.cstudio.app/docs/resource/aws/Route53/ResolverEndpoint)

## 리소스 설정 값

-   `rule_type` : 규칙 유형 - `Forward`, `system`
-   `domain_name` : DNS 쿼리가 전달되는 도메인 이름
-   `linked_vpc_names` : 규칙을 사용하는 VPC 이름
-   `linked_resolver_endpoint_name` : Resolver가 사용하는 아웃바운드 엔드포인트 이름
-   `target_ip` : Target IP 주소
-   `target_ip.name`
-   `target_ip.ip` : Target IP 주소의 IP
-   `target_ip.port` : Target IP 주소의 포트 - `1~65535`
