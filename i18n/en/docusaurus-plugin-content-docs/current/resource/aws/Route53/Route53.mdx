## Resource Overview

Amazon Route 53 is a highly available and scalable Domain Name System (DNS) web service.

### Parent Resources

-   [**Base**](https://docs.cstudio.app/docs/resource/aws/Base/)

### Connected Resources

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)

## Resource Setting Values

-   `linked_vpc_names` : Names of the VPCs to which the Route 53 zone will be associated
-   `record` : Name of the record to be created in the Route 53 zone
    -   `type` : Type of the record - `A`, `CAA`, `CNAME`, `DS`, `NAPTR`, `NS`, `PTR`, `SOA`, `SPF`, `SRV`, `TXT`
    -   `ttl` : TTL of the record
    -   `records` : List of strings for the record
