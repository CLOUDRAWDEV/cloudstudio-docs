## Resource Overview

Endpoint service for DNS queries sent to and received from Amazon VPC

### Parent Resources

-   [**Region**](https://docs.cstudio.app/en/docs/resource/aws/Base/Region)

### Connected Resources

-   [**Vpc**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Vpc)
-   [**SecurityGroup**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/SecurityGroup)
-   [**ResolverRule**](https://docs.cstudio.app/en/docs/resource/aws/Route53/ResolverRule)

## Resource Setting Values

-   `direction` : DNS query direction - `INBOUND`, `OUTBOUND`
-   `linked_vpc_name` : VPC name to associate with the endpoint
-   `linked_security_group_names` : Security group name for the endpoint
-   `ip_address` : Name to IP Address
