## Resource Overview

A logical networking component representing a virtual network card within a VPC

## Related Resources

### Parent Resources

-   [**Subnet**](https://docs.cstudio.app/docs/resource/aws/Vpc/Subnet)

### Connected Resources

-   [**Eip**](https://docs.cstudio.app/docs/resource/aws/Ec2/Eip)

## Resource Configuration Values

-   `private_ips`: List of private IP addresses
-   `linked_security_group_names`: Names of the security groups to be used with the network interface
-   `linked_eip_names`: Names of the Elastic IPs to be used
