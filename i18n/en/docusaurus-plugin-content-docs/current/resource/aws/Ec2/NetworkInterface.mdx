## 리소스 개요

VPC에서 가상 네트워크 카드를 나타내는 논리적 네트워킹 구성 요소

## 연관 리소스

### 상위 리소스

-   [**Subnet**](https://docs.cstudio.app/docs/resource/aws/Vpc/Subnet)

### 연결 리소스

-   [**Eip**](https://docs.cstudio.app/docs/resource/aws/Ec2/Eip)

## 리소스 설정 값

-   `private_ips` : private ips 목록
-   `linked_security_group_names` : 네트워크 인터페이스에 사용할 Security Group 이름
-   `linked_eip_names` : 사용할 Elastic Ip 이름
