## Resource Overview

A logical networking component representing a virtual network card within a VPC

## Associated Resources

### Parent Resources

-   [**Subnet**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Subnet)

### Connected Resources

-   [**Eip**](https://docs.cstudio.app/en/docs/resource/aws/Ec2/Eip)

## Resource Setting Values

-   `private_ips`: List of private IP addresses
-   `linked_security_group_names`: The name of the Security Group to use for the Network interface
-   `linked_eip_names`: The name of Elastic Ip to use
