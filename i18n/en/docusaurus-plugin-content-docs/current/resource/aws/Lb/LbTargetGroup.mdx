## 리소스 개요

지정한 프로토콜과 포트 번호를 사용하여 EC2 인스턴스 같은 하나 이상의 등록된 대상으로 요청을 라우팅하는 서비스

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)
-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)

### 연결 리소스

-   [**Ec2 Instance**](https://docs.cstudio.app/docs/resource/aws/Ec2/Instance)
-   [**Lambda Function**](https://docs.cstudio.app/docs/resource/aws/Lambda/LambdaFunction)
-   [**Lb**]https://docs.cstudio.app/docs/resource/aws/Lb)

## 리소스 설정 값

-   `target_type` : LB 대상의 종류 - `Instance`, `Ip`, `Lambda`, `Alb`
-   `protocol` : 트래픽을 대상으로 라우팅하는 데 사용할 프로토콜 - `HTTP`,`HTTPS`,`TCP`,`TCP_UDP`,`TLS`
-   `port` : 트래픽을 수신하는 포트 - `1~65535`
-   `included_vpc_name` : 대상 그룹을 생성할 VPC의 이름
-   `linked_instance_names` : Target Instance의 이름 목록
