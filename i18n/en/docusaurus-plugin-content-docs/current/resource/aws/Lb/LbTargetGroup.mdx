## Resource Overview

A service that routes requests to one or more registered targets, such as EC2 instances, using the specified protocol and port number.

## Resource Setting Values

-   `target_type` : The type of LB target - `Instance`, `Ip`, `Lambda`, `Alb`
-   `protocol` : Protocol to use for routing traffic to the targets - `HTTP`, `HTTPS`, `TCP`, `TCP_UDP`, `TLS`
-   `port` : Port on which targets receive traffic - `1~65535`
-   `included_vpc_name` : Name of the VPC in which to create the target group
-   `linked_instance_names` : List of the target Instance names
