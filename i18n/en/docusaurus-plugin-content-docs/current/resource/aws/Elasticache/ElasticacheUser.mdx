## Resource Overview

An AWS ElastiCache User is an IAM user required to work with ElastiCache clusters. 
The ElastiCache User has permissions to create, modify, and delete ElastiCache clusters. 
This user is created in AWS IAM and is typically assigned to applications or developers who use ElastiCache. 
The ElastiCache User is responsible for security and access control to the ElastiCache clusters, helping to protect applications and data. 
The ElastiCache User can also be used alongside other resources within the AWS account.

## Resource Setting Values

-   `user_id`: Each user's unique ID
-   `access_string`: A string containing the permission to be applied to the user
-   `authentication_type`: Authentication method for users to access the cluster
