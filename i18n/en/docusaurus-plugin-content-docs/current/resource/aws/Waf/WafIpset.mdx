## 리소스 개요

AWS WAF IPSet은 Amazon Web Services의 Web Application Firewall에서 사용하는 IP 주소 목록입니다. 이를 통해, 사용자는 정의한 IP 주소들에 대한 액세스 규칙을 생성하고 관리할 수 있습니다. IPSet을 활용하면, 특정 IP 주소들로부터의 요청을 허용하거나 차단함으로써 웹 애플리케이션 보안을 강화할 수 있습니다.

## 리소스 설정 값

-   `scope` : AWS CloudFront 배포용인지 아니면 리전 애플리케이션용인지 지정 - `REGIONAL`, `CLOUDFRONT`
-   `ip_address_version` : IPV4 또는 IPV6 지정 - `IPV4`, `IPV6`
-   `addresses` : CIDR 표기법으로 하나 이상의 IP 주소 또는 IP 주소 블록을 지정하는 문자열 배열을 입력
