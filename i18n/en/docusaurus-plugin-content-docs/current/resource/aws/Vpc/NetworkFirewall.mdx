## Resource Overview

An AWS service that allows you to configure protection across AWS accounts and resources within your organization

## Resource Configuration Values

-   `description`: Description of the firewall
-   `included_vpc_name`: Name of the VPC where the firewall will be located
-   `linked_subnet_names`: Public subnets dedicated to firewall endpoints
-   `linked_network_firewall_policy_name`: Name of the firewall policy to be associated
-   `enabled_delete_protection`: Enable protection to prevent the firewall from being deleted - `false`, `true`
-   `enabled_subnet_change_protection`: Enable protection to prevent changes to subnet associations - `false`, `true`
-   `encryption_configuration.enabled_custom_configuration`: Encrypt data using AWS managed keys - `false`, `true`
