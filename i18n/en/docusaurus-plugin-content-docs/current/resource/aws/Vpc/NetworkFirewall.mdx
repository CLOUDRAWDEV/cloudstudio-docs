## 리소스 개요

조직 내 AWS 계정 및 리소스 전반에 걸쳐 보호를 구성할 수 있는 AWS 서비스

## 연관 리소스

### 상위 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)

### 연결 리소스

-   [**Network Firewall Policy**](https://docs.cstudio.app/docs/resource/aws/Vpc/NetworkFirewallPolicy)
-   [**Subnet**](https://docs.cstudio.app/docs/resource/aws/Vpc/Subnet)


## 리소스 설정 값

-   `description` : 방화벽 설명
-   `included_vpc_name` : 방화벽이 위치할 VPC 이름
-   `linked_subnet_names` : 방화벽 엔드포인트 전용 퍼블릭 서브넷
-   `linked_network_firewall_policy_name` : 연결할 방화벽 정책 이름
-   `enabled_delete_protection` : 방화벽이 삭제되지 않도록 보호 활성화 - `false`,`true`
-   `enabled_subnet_change_protection` : 서브넷 연결이 변경되지 않도록 보호 활성화 - `false`,`true`
-   `encryption_configuration.enabled_custom_configuration` : AWS 관리형 키를 사용하여 데이터를 암호화 - `false`,`true`
