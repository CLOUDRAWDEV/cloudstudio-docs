## Resource Overview

An AWS service that allows you to configure protection across AWS accounts and resources within your organization

## Associated Resources

### Parent Resources

-   [**Vpc**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Vpc)

### Connected Resources

-   [**Network Firewall Policy**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/NetworkFirewallPolicy)
-   [**Subnet**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Subnet)


## Resource Setting Values

-   `description`: A description of the firewall policy
-   `included_vpc_name`: VPC name where the firewall will be located
-   `linked_subnet_names`: Public subnet dedicated to firewall endpoints
-   `linked_network_firewall_policy_name`: Firewall policy name to connect to
-   `enabled_delete_protection`: Enable firewall protection from deletion - `false`, `true`
-   `enabled_subnet_change_protection`: Enable protection against subnet connectivity changes - `false`, `true`
-   `encryption_configuration.enabled_custom_configuration`: Encrypt data using AWS-managed key - `false`, `true`
