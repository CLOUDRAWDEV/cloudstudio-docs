## Resource Overview

AWS Transit Gateway connects Amazon Virtual Private Cloud (VPC) and on-premises networks through a central hub. 
This connection simplifies the network by eliminating complex peering relationships. 
Transit Gateway serves as a highly scalable cloud router.
