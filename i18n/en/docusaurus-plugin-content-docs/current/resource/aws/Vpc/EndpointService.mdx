## Resource Overview

A service that allows customers to privately connect to supported AWS services and VPC endpoint services

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### Child Resources

-   [**LB**](https://docs.cstudio.app/docs/resource/aws/Lb/Lb)
-   [**Endpoint**](https://docs.cstudio.app/docs/resource/aws/Vpc/Endpoint)

## Resource Setting Values

-   `acceptance_required` : Whether the service owner needs to accept VPC endpoint connection requests for the service - `false`, `true`
-   `loadbalancer_type` : Type of load balancer to connect with the endpoint service - `network`, `gateway`
-   `linked_lb_names` : Names of one or more load balancer resources connected to the endpoint service
-   `manual_lb_arns` : Amazon Resource Names (ARNs) of one or more load balancing devices for the endpoint service
-   `supported_ip_address_types` : Supported IP address types - `ipv4`
-   `activate_private_dns_name` : Private DNS name for the service
