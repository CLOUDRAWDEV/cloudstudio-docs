## Resource Overview

A service that allows customers to privately connect to supported AWS services and VPC endpoint services

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### Child Resources

-   [**LB**](https://docs.cstudio.app/docs/resource/aws/Lb/Lb)
-   [**Endpoint**](https://docs.cstudio.app/docs/resource/aws/Vpc/Endpoint)

## Resource Setting Values

-   `acceptance_required` : Whether or not VPC endpoint connection requests to the service must be accepted by the service owner - `false`, `true`
-   `loadbalancer_type` : Type of load balancer to connect from Endpoint service - `network`, `gateway`
-   `linked_lb_names` : Names of Lb for the endpoint service
-   `manual_lb_arns` : Amazon Resource Names(ARNs) of one or more Load Balancers for the endpoint service
-   `supported_ip_address_types` : The supported IP address types - `ipv4`
-   `activate_private_dns_name` : The private DNS name for the service
