## Resource Overview

A subnet is a range of IP addresses within a VPC. AWS resources, such as EC2 instances, can be created within a specific subnet.

## Resource Configuration Values

-   `included_vpc_name`: The name of the VPC where the subnet will be deployed
-   `cidr_block`: The IP range (CIDR) of the subnet
-   `availability_zone`: The availability zone where the subnet will be deployed
-   `pip_enable`: Public IP assignment
