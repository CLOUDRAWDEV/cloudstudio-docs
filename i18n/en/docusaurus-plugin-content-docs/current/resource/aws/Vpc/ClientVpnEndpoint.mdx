## Resource Overview

A managed client-based VPN service that provides secure access to AWS resources and on-premises network resources.

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)
-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)

## Resource Setting Values

-   `description`: A brief description of the client VPN endpoint
-   `client_cidr_block`: The IP address range assigned to client IP addresses
-   `server_acm_certificate_name`: The name of the server certificate
-   `authentication`: The authentication method used
-   `authentication.option_types`: The authentication methods available - `certificate-authentication`, `directory-service-authentication`, `federated-authentication`
-   `connection_logging.enabled`: Whether logging details for client connections are enabled - `false`, `true`
-   `transport_protocol`: The transport protocol used in the TLS session - `tcp`, `udp`
-   `enabled_split_tunnel`: Whether split tunneling is enabled - `false`, `true`
-   `vpn_port`: The VPN port
-   `enabled_self_service_portal`: Whether the self-service portal is enabled - `false`, `true`
-   `session_timeout_hours`: The session timeout period - `8`, `10`, `12`, `24`
-   `included_vpc_name`: The VPC name
-   `linked_security_group_names`: The names of the security groups applied to the endpoint
-   `linked_subnet_names`: The names of the subnets to connect
