## Resource Overview

A managed client-based VPN service that provides secure access to AWS resources and on-premises network resources.

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/en/docs/resource/aws/Base/Region)
-   [**Vpc**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Vpc)

## Resource Setting Values

-   `description`: A brief description of the client VPN endpoint
-   `client_cidr_block`: The IP address range from which client IP addresses are allocated
-   `server_acm_certificate_name`: The server certificate name
-   `authentication`: Authentication methods to use
-   `authentication.option_types`: Authentication methods to use - `certificate-authentication`, `directory-service-authentication`, `federated-authentication`
-   `connection_logging.enabled`: Enable log details on client connections - `false`, `true`
-   `transport_protocol`: Transport protocol used by the TLS sessions - `tcp`, `udp`
-   `enabled_split_tunnel`: Enable split-tunnel - `false`, `true`
-   `vpn_port`: VPN port
-   `enabled_self_service_portal`: Enable self-service portal - `false`, `true`
-   `session_timeout_hours`: Session timeout hours - `8`, `10`, `12`, `24`
-   `included_vpc_name`: VPC name
-   `linked_security_group_names`: Security group names to be applied to the endpoint
-   `linked_subnet_names`: Subnet names to be connected to the endpoint
