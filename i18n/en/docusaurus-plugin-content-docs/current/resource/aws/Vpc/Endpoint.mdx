## Resource Overview

A service that allows customers to privately connect to supported AWS services and VPC endpoint services.

## Resource Configuration Values

-   `service_range`: The category of the service for using the endpoint service - `awsService`, `endpointService`
-   `aws_service_name`: The name of the AWS service using the endpoint service
-   `vpc_endpoint_type`: The type of VPC endpoint service to use - `gateway`, `interface`, `gatewayloadbalancer`
-   `included_vpc_name`: The name of the VPC where the endpoint will be created
-   `route_table_names`: One or more route table IDs to associate with the endpoint
-   `policy_access`: The policy access scope to control service access for the endpoint - `Full`, `Custom`
