## Resource Overview

A service that allows customers to privately connect to supported AWS services and VPC endpoint services.

## Associated Resources

### Parent Resources

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)

### Child Resources

-   [**Endpoint Service**](https://docs.cstudio.app/docs/resource/aws/Vpc/EndpointService)
-   [**Security Group**](https://docs.cstudio.app/docs/resource/aws/Vpc/SecurityGroup)
-   [**Subnet**](https://docs.cstudio.app/docs/resource/aws/Vpc/Subnet)

## Resource Setting Values

-   `service_range` : The range of AWS service to use endpoint - `awsService`, `endpointService`
-   `aws_service_name` : The name of AWS service to use endpoint
-   `vpc_endpoint_type` : The name of other endpoint service to use endpoint - `gateway`, `interface`, `gatewayloadbalancer`
-   `included_vpc_name` : The name of VPC to use endpoint
-   `route_table_names` : One or more route table IDs to associate with endpoint
-   `policy_access` : Policy access range to attach to the endpoint that controls access to the service - `Full`, `Custom`
