## Resource Overview

A service that can be used to connect a VPC to on-premises networks

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### Connected Resources

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/Vpc)
-   [**TransitGateway**](https://docs.cstudio.app/docs/resource/aws/Vpc/TransitGateway)
-   [**TransitGatewayAttachmentAccepter**](https://docs.cstudio.app/docs/resource/aws/Vpc/TransitGatewayAttachmentAccepter)


## Resource Setting Values

-   `linked_transit_gateway_name` : Name of the Transit Gateway connecting to the VPC
-   `is_manual` : Whether to enter the VPC ID to connect directly - `false`, `true`
-   `linked_vpc_name` : VPC name to connect the Transit Gateway to
-   `subnet_names` : Subnet name to connect the Transit Gateway
