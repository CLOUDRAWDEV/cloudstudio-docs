## Resource Overview

A NAT gateway is a Network Address Translation (NAT) service. 
It allows instances in a private subnet to connect to services outside the VPC while preventing external services from initiating connections to those instances.

## Associated Resources

### Parent Resources

-   [**Subnet**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/Subnet)

## Resource Setting Values

-   `included_subnet_name` : The name of Subnet to create the Nat gateway
-   `connectivity_type` : Connection type of NAT Gateway - `public`, `private`
-   `eip_name` : The name of the Elastic IP (EIP)
