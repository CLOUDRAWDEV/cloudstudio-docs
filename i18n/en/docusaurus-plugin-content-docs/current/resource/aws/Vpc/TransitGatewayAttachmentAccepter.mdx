## Resource Overview

A resource that determines whether to allow the sharing of resources across different accounts

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/en/docs/resource/aws/Base/Region)

### Connected Resources

-   [**TransitGatewayAttachment**](https://docs.cstudio.app/en/docs/resource/aws/Vpc/TransitGatewayAttachment)

## Resource Setting Values

-   `is_manual` : Whether to enter the Transit gateway Attachment ID to connect directly - `false`, `true`
-   `linked_transit_gateway_attachment_name` : Transit Gateway Attachment's Name that to enable share
