## Resource Overview

A resource that determines whether to allow the sharing of resources across different accounts

## Related Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### Connected Resources

-   [**TransitGatewayAttachment**](https://docs.cstudio.app/docs/resource/aws/Vpc/TransitGatewayAttachment)

## Resource Configuration Values

-   `is_manual` : Whether to manually enter the Transit Gateway Attachment ID for the connection - `false`, `true`
-   `linked_transit_gateway_attachment_name` : Name of the other Transit Gateway Attachment for which sharing is allowed
