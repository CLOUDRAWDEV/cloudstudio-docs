## Resource Overview

Using the ACM Certificate resource allows you to request and manage certificates from Amazon Certificate Manager.

### Parent Resource

-   [**Region**](https://docs.cstudio.app/en/docs/resource/aws/Base/Region)

## Associated Resources


## Resource Setting Values

-   `enabled_import`: Import certificate
-   `certificate_body`: PEM-encoded certificate body
-   `certificate_private_key`: PEM-encoded certificate private key
-   `certificate_chain`: PEM-encoded certificate chain
