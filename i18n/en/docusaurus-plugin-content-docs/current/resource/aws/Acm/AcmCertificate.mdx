## 리소스 개요

ACM Certificate 리소스를 사용하면 Amazon Certificate Manager에서 인증서를 요청하고 관리할 수 있습니다.

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

## 연관 리소스


## 리소스 설정 값

-   `enabled_import` : 인증서 불러오기
-   `certificate_body` : 불러올 인증서의 PEM 인코딩된 본문
-   `certificate_private_key` : 불러올 인증서의 PEM 인코딩된 프라이빗 키
-   `certificate_chain` : 불러올 인증서의 PEM 인코딩된 인증서 체인
