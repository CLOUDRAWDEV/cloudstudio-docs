## Resource Overview

Using the ACM Certificate resource allows you to request and manage certificates from Amazon Certificate Manager.

## Resource Overview

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

## Related Resources


## Resource Configuration Values

-   `enabled_import`: Import Certificate
-   `certificate_body`: PEM-encoded body of the certificate to be imported
-   `certificate_private_key`: PEM-encoded private key of the certificate to be imported
-   `certificate_chain`: PEM-encoded certificate chain of the certificate to be imported
