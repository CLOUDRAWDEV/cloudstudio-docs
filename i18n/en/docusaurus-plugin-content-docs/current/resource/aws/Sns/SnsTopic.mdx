## 리소스 개요

AWS SNS에서 커뮤니케이션 채널 역할을 하는 논리적 액세스 포인트

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### 연결 리소스

-   [**Metric**](https://docs.cstudio.app/docs/resource/aws/Cloudwatch/Metric)
-   [**Config**](https://docs.cstudio.app/docs/resource/aws/Config/Config)

## 리소스 설정 값

-   `display_name` : SNS Topic의 표시이름
-   `type` : SNS Topic의 유형 - `FIFO`, `Standard`
-   `kms_key_name` : SNS Topic에 사용할 KMS Key 이름
