## Resource Overview

A logical access point that serves as a communication channel in AWS SNS

## Associated Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

-   [**Metric**](https://docs.cstudio.app/docs/resource/aws/Cloudwatch/Metric)

## Resource Setting Values

-   `display_name` : Display name of the SNS Topic
-   `type` : Type of the SNS Topic - `FIFO`, `Standard`
-   `kms_key_name` : KMS Key name to use for SNS Topic
