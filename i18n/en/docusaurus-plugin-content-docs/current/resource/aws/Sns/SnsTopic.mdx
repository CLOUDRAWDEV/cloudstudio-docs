## Resource Overview

A logical access point that serves as a communication channel in AWS SNS

## Resource Configuration Values

-   `display_name`: The display name of the SNS topic
-   `type`: The type of the SNS topic - `FIFO`, `Standard`
-   `kms_key_name`: The name of the KMS key used for the SNS topic
