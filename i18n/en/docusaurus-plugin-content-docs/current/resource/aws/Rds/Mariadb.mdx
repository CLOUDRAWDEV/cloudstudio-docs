## Resource Overview

Here is the improved English version:

MariaDB is a popular open-source relational database created by the original developers of MySQL. 
With Amazon RDS, you can easily set up, operate, and scale MariaDB servers in the cloud. 
Amazon RDS allows you to deploy a scalable MariaDB cloud database with cost-effective and resizable hardware capacity in just a few minutes.

### Connected Resources

-   [**MariaDB**](https://docs.cstudio.app/docs/resource/aws/Rds/Mariadb)

 values

-   `database_name`: Initial database name
-   `engine_version`: MariaDB version to be used with RDS
-   `instance_class`: Pricing tier for the RDS DB instance
-   `storage_type`: Storage type for the RDS instance
-   `allocated_storage`: Capacity of the RDS instance storage (in Gigabytes)
-   `max_allocated_storage`: Maximum storage limit for automatic scaling by Amazon RDS
-   `multi_az_enabled`: Enable Multi-AZ deployment - `true`, `false`
-   `username`: Username for the RDS instance
-   `password`: Password for the RDS instance
-   `included_subnet_names`: List of subnet names to be associated with the RDS subnet group
-   `subnet_group_name`: Name of the subnet group to be used with RDS
-   `publicly_accessible`: Whether the instance is publicly accessible - `true`, `false`
-   `monitoring`
    -   `performance_insights_enabled`: Enable Performance Insights
-   `encryption`
    -   `storage_encrypted`: Whether the DB instance is encrypted - `true`, `false`
    -   `kms_key_name`: KMS key name used for encryption
-   `backup`
    -   `backup_retention_period`: Duration for retaining backup data - `1` to `35` days
    -   `final_snapshot_enabled`: Whether to create a final DB snapshot before the instance is deleted - `true`, `false`
    -   `final_snapshot_name`: Name of the final snapshot
