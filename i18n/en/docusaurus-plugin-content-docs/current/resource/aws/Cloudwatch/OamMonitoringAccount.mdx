## Resource Overview

AWS CloudWatch OAM (Observability Access Management) Monitoring Accounts is a service that allows you to set up a central monitoring account to monitor applications across multiple accounts.

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/en/docs/resource/aws/Base/Region)

### Connected Resources

-   [**Oam Source Account**](https://docs.cstudio.app/en/docs/resource/aws/Cloudwatch/OamSourceAccount)

## Resource Setting Values

-   `oam_source_accounts`: List the accounts, organization paths, or organization IDs of the accounts that will share data with the monitoring account
-   `resource_types`: The data types that you want to share
    -   `AWS::Logs::LogGroup`
    -   `AWS::CloudWatch::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
