## 리소스 개요

AWS CloudWatch Oam Monitoring Accounts는  중앙 모니터링 계정을 설정하여 여러 계정의 애플리케이션을 모니터링하는 서비스입니다.

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### 연결 리소스

-   [**Oam Source Account**](https://docs.cstudio.app/docs/resource/aws/Cloudwatch/OamSourceAccount)

## 리소스 설정 값

-   `oam_source_accounts` : 모니터링 계정과 데이터를 공유할 계정의 계정, 조직 경로 또는 조직 ID
-   `resource_types` : 공유하려는 데이터 유형 
    -   `AWS::Logs::LogGroup`
    -   `AWS::Cloudwatch::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
