## Resource Overview

AWS CloudWatch OAM Source Accounts configures the source accounts that will be shared with the central monitoring account.

## Associated Resources

### Parent Resources

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### Connected Resources

-   [**Oam Monitoring Account**](https://docs.cstudio.app/docs/resource/aws/Cloudwatch/OamMonitoringAccount)

## Resource Setting Values

-   `is_manual`: Whether to manually input the ARN of the OAM Source Account to be linked - `false`, `true`
-   `linked_oam_monitoring_account_name`: The name of the OAM Monitoring Account to be linked
-   `resource_types`: Types of data to be shared with the monitoring account
    -   `AWS::Logs::LogGroup`
    -   `AWS::CloudWatch::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
-   `label_template`: A label to help identify the source account
