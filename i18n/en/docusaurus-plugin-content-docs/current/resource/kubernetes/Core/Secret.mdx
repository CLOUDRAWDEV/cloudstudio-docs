## 리소스 개요

개별 속성이나 전체 파일 또는 JSON Blob과 같은 대략적인 항목으로 중요한 정보를 저장하는 데 사용되는 쿠버네티스 리소스입니다.

## 리소스 설정 값

-   `service_name` : Secret 리소스 이름
-   `namespace` : Secret 리소스가 배포될 네임스페이스
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `type` : Secret 타입 - `Opaque`,`kubernetes.io/basic-auth`,`kubernetes.io/service-account-token`,`kubernetes.io/dockerconfigjson`
-   `immutable` : 보안 비밀에 저장된 데이터를 업데이트할 수 없도록 설정 (객체 메타데이터만 수정가능) - `false`,`true`

-   `data` : secret에 들어갈 data
    -   `data.name` : secret에 들어갈 data 이름 - `Opaque`,`kubernetes.io/asic-auth` 
        -   `data.secret` : secret에 들어갈 data 값 - `Opaque`,`kubernetes.io/basic-auth`         
    -   `data.name` : private docker registry 이름 - `kubernetes.io/dockerconfigjson`
        -   `data.username` : private docker registry 인증 사용자 이름 - `kubernetes.io/dockerconfigjson`
        -   `data.password` : private docker registry 인증 사용자 비밀번호 - `kubernetes.io/dockerconfigjson`
