## 리소스 개요

쿠버네티스 클러스터 내에서 애플리케이션 및 서비스가 클러스터 자원에 접근하고 사용할 수 있도록 권한을 부여하는 계정입니다.

### 연결 리소스


## 리소스 설정 값

-   `service_name` : ServiceAccount 리소스 이름
-   `namespace` : ServiceAccount 리소스가 배포될 네임스페이스
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `secret` : 이 서비스 계정을 참조하는 파드(Pod)에서 사용하도록 허용된 시크릿 목록
-   `image_pull_secrets` : 이 서비스 계정을 참조하는 파드(Pod)의 이미지를 가져오는 데 사용할 동일한 네임스페이스의 시크릿 목록
