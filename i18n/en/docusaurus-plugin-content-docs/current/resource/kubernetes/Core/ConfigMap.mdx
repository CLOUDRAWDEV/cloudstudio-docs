## 리소스 개요

애플리케이션 설정 정보를 분리하여 저장하고 파드 내의 컨테이너에 전달하는 데 사용되는 쿠버네티스 리소스입니다.

## 리소스 설정 값

-   `service_name` : ConfigMap 리소스 이름
-   `namespace` : ConfigMap 리소스가 배포될 네임스페이스
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `data` : key-value 쌍으로 일반적인 데이터 - 설정파일,환경변수 등과 같은 텍스트 형식의 구성정보를 저장하는데 사용
    -   `data.name` : data의 key 값
    -   `data.value` : data의 value 값
-   `binary_data` : 특정 키에 대한 바이너리 데이터 - 바이너리 파일 또는 기타 이진형식의 데이터를 저장하는데 사용
    -   `data.name` : binary data의 key 값
    -   `data.value` : binary data의 value 값
