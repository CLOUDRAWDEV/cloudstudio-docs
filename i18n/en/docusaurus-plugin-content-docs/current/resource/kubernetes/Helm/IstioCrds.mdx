## Resource Setting Values

-   `service_name` : Istio crds 헬름 차트 릴리즈 이름
-   `namespace` : Istio crds 헬름 차트가 배포될 네임스페이스 -  `istio-system`으로 지정
-   `version` : 사용할 Istio crds 헬름 차트 버전
-   `set` : Istio crds 기본 차트의 값을 동적으로 변경
