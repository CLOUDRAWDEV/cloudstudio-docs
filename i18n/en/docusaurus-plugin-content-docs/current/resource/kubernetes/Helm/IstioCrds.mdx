## Resource Overview

Helm chart for Custom Resource Definitions (CRDs) for Istio.

### Connected Resources




## Resource Setting Values

-   `service_name` : Istio CRDs Helm Chart Release Name
-   `namespace` : Namespace where the CertManager Helm Chart resource will be deployed -  `istio-system`으로 지정
-   `version` : CertManager Helm Chart version to use
-   `set` : Dynamically modify values for the Istio CRDs base chart
