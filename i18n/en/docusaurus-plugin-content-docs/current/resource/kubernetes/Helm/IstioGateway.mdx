## Resource Setting Values

-   `service_name` : Istio gateway Vault Helm chart release name
-   `namespace` : Namespace where Istio Ingress Helm Chart resources will be deployed
-   `version` : Specifies the version of the Istio Ingress Helm char
-   `set` : Dynamically change values of Istio Ingress default chart
