## Resource Overview

A resource that allows you to deploy the default Helm chart with your desired values.

## Associated Resources

-   [**Cluster**](https://docs.cstudio.app/en/docs/resource/kubernetes/Base/Cluster)

### Connected Resources



## Resource Setting Values

-   `service_name` : Helm Custom Chart Release Name
-   `namespace` : Namespace where custom Helm Chart resources will be deployed
-   `repository` : Repository address of the Helm chart to be customized
-   `chart` : Specify a name for the Helm chart you want to customize
-   `version` : Specifies the version of the Helm chart to be customized
-   `set` : Dynamically change values in main chart
