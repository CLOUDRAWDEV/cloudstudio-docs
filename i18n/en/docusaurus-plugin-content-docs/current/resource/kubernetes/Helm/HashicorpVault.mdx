## Resource Setting Values

-   `service_name` : HashiCorp Vault Helm Chart Release Name
-   `namespace` : Namespace where Hashicorp Vault Helm Chart resources will be deployed
-   `version` : Specifies the version of the Hashicorp Vault Helm chart
-   `set` : Dynamically change values of Hashicorp Vault default chart
