## Resource Setting Values

-   `service_name` : Istio Vault Helm chart release name
-   `namespace` : Namespace where Istiod Helm Chart resources will be deployed - typically set to `istio-system`
-   `version` : Specifies the version of the Istiod Helm chart
-   `set` : Dynamically change values of Istiod default chart
