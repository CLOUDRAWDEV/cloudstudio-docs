## Resource Setting Values

-   `service_name` : Cert-Manager 헬름 차트 릴리즈 이름
-   `namespace` : Namespace where the CertManager Helm Chart resource will be deployed
-   `version` : CertManager Helm Chart version to use
-   `installCRDs` : Check whether custom resource definition related to Cert Manager is installed
