## 리소스 개요

AWS CloudWatch Oam Source Accounts는 중앙 모니터링 계정과 공유될 소스 계정을 구성합니다.

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### 연결 리소스

-   [**Oam Monitoring Account**](https://docs.cstudio.app/docs/resource/aws/Cloudwatch/OamMonitoringAccount)

## 리소스 설정 값

-   `is_manual` : 연결할 Oam Source Account Arn를 직접 입력할지 여부 - `false`,`true`
-   `linked_oam_monitoring_account_name` : 연결할 Oam Monitoring Account 이름 
-   `resource_types` : 모니터링 계정과 공유하려는 데이터 유형
    -   `AWS::Logs::LogGroup`
    -   `AWS::Cloudwatch::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
    -   `AWS::Logs::LogGroup`
-   `label_template` : 소스 계정을 식별하는 데 도움이 되는 레이블
