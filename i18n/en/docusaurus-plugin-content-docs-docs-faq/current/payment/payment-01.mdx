# 사용 중인 구독을 업그레이드 하고싶어요

<hr />
<br />

1. 클라우드 스튜디오 메인화면에서 좌측의 `내 정보` 메뉴를 선택합니다.

2. `구독 관리`를 클릭하여 구독플랜 페이지로 이동합니다.

3. 현재 구독을 `PRO 플랜`으로 변경하여 구독을 업그레이드 할 수 있습니다.

:::info

구독 업그레이드 시에는 과금(정기결제)이 진행됩니다.

:::
