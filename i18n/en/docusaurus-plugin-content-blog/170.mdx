#### Kubernetes Import Feature Added

<div className={styles.releaseWrap} >
<div className={styles.tag} data-type="feature">
    Feature
</div>

1. Kubernetes Import Feature Added
    - The import feature has been added for Kubernetes Providers, enabling visualization of deployed Kubernetes resources.
        - Supported: AKS (Microsoft Azure), EKS (AWS), NKS (Naver Cloud)
    - Imported Kubernetes resources can be converted into Terraform code.
    - You can generate a Terraform state file using the Create tfstate feature.
