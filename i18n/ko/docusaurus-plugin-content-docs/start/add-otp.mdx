1. `내 정보 - 계정 관리`에서 `OTP 등록 버튼`을 클릭한 후, 로그인에 필요한 `비밀번호`를 입력합니다.

<img width="100%" src={otp08} alt="otp img" />

<p>&nbsp;</p>

2. 링크를 통해 `google authenticator 앱을 설치`해줍니다.

<img width="100%" src={otp01} alt="otp img" />

<p>&nbsp;</p>

3. google authenticator 앱을 실행한 후 `QR코드를 스캔`해줍니다.

<img width="100%" src={otp02} alt="otp img" />

<p>&nbsp;</p>

4. 한 번 발급된 인증키는 `다시 찾을 수 없으니` 반드시 적어두도록 합니다.

<img width="100%" src={otp03} alt="otp img" />

<p>&nbsp;</p>

5. google authenticator 앱에 등록된 `6자리 인증코드`를 입력한 후 `인증완료 버튼`을 클릭합니다.

<img width="100%" src={otp04} alt="otp img" />
