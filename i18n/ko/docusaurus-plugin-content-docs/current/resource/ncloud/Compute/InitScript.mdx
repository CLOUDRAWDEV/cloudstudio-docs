## 리소스 개요

Init Script는 미리 작성해 둔 스크립트를 서버 생성 시 자동으로 실행시킬 수 있는 기능입니다. 같은 용도의 서버를 여러 대 일괄 생성하거나, 같은 환경의 서버를 주기적으로 생성하거나 용도별로 서버 초기 환경 관리가 필요한 경우 등 여러 가지 경우에 활용할 수 있습니다.

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/ncloud/Base/Region)


## 리소스 설정 값

-   `OS_type` : 사용할 서버 인스턴스의 종류 - `LNX`, `WND`
-   `Content` : 서버가 생성될 때 자동으로 실행할 스크립트 내용, 예) #!/bin/bash yum install -y httpd && systemctl start httpd
