## 리소스 개요

가상 프라이빗 게이트웨이는 Site-to-Site VPN 연결의 Amazon 측에 있는 VPN 집선기입니다. 가상 프라이빗 게이트웨이를 생성하여 Site-to-Site VPN 연결을 생성할 VPC에 연결합니다.

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

### 연결 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/)

-   [**Vpc Customer Gateway**](https://docs.cstudio.app/docs/resource/aws/Vpc/CustomerGateway)

## 리소스 설정 값

-   `linked_vpc_name` : VPN Gateway가 포함될 VPC 이름
-   `vpn_connection` : VPN Connection 이름
    -   `linked_customer_gateway` : VPN Gateway와 연결할 Customer Gateway 이름
    -   `static_routes_only` : Gateway간 라우팅 시 정적 라우팅만 사용할지 지정 - `true`, `false`
