## 리소스 개요

AWS EC2 인스턴스에 연결할 때 자격 증명 입증에 사용하는 보안 자격증명 집합

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/aws/Base/Region)

## 리소스 설정 값

-   `public_key` : 사용자가 생성한 키페어의 Public key
