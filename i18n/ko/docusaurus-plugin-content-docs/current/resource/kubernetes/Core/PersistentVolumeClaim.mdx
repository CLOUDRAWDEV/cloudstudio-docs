## 리소스 개요

## 리소스 설정 값

-   `service_name` : PersistentVolumeClaim 리소스 이름
-   `namespace` : PersistentVolume 리소스가 배포될 네임스페이스
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `spec` : PVC의 구체적인 스펙을 정의
    -   `spec.storageclass_name` : 사용할 storageclass 이름 지정
    -   `spec.volume_name` : PVC로 연결될 PV 이름
    -   `spec.request_storage` : PV에 요구할 스토리지 용량
    -   `spec.access_modes` : PV에 있어야 하는 원하는 액세스 모드 - (PV의 액세스 모드를 포함하거나 일치해야 합니다.) - `ReadOnlyMany`,`ReadWriteMany`,`ReadWriteOnce`
