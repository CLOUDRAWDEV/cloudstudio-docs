# 기존 인프라를 캔버스에 불러올 수 있나요?
<hr/>
<br />

클라우드 인프라를 크레덴셜 및 파일을 통해 불러올수 있습니다.

1. 캔버스 상위 메뉴 ```파일 > 가져오기```를 선택합니다.
2. ```인프라에서 불러오기``` 혹은 ```파일에서 불러오기```를 선택합니다.

<Tabs>
  <TabItem value="infraImport" label="인프라에서 불러오기" default>
    등록된 크레덴셜을 선택하고 리소스 그룹을 조회(Azure 기준)하여 리소스 그룹 단위로 불러올 수 있습니다.
  </TabItem>
  <TabItem value="fileImport" label="파일에서 불러오기">
    테라폼 파일을 직접 업로드하여 클라우드 스튜디오에 맞게 변환되어 제공합니다.
  </TabItem>
</Tabs>

:::tip

배포된 클라우드 인프라를 불러오기 위해서는 크레덴셜이 등록되어 있어야 합니다.

:::
