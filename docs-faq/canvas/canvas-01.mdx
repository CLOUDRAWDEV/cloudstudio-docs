# 저장과 버전저장의 차이
<hr/>
<br />

<Tabs>
  <TabItem value="save" label="저장" default>
    저장은 현재 캔버스의 상태를 저장하는 것입니다. 저장을 한다고 이전 버전의 캔버스로 돌아갈 수 있는 것은 아닙니다. 이전의 상태로 캔버스를 돌리고 싶다면 버전 저장 기능을 이용해야 합니다.
  </TabItem>
  <TabItem value="versionSave" label="버전저장">
    버전 저장은 캔버스를 원하는 시점의 상태로 돌리는 기능을 말합니다. 버전 저장을 통해 특정 시점의 캔버스 상태를 저장하면 나중에 해당 시점을 캔버스로 불러올 수 있습니다.
  </TabItem>
</Tabs>
