# 캔버스 배포는 어떻게 하나요?
<hr/>
<br />

1. 캔버스 배포는 배포관리 메뉴의 ```배포테스트```를 통해 배포 전 검사를 진행하고 이후 검사 성공시 ```배포하기```를 클릭하여 CSP에 배포 할 수 있습니다.
2. 캔버스 상위 메뉴 ```배포관리 > 배포테스트```를 선택합니다.
3. 배포테스트 모달에서 배포 환경 및 크레덴셜을 선택하고 태그 메세지를 입력 후 배포테스트 버튼을 클릭합니다.
4. 좌측 배포테스트 결과 메뉴에서 배포 테스트 진행 및 결과를 확인합니다.
5. 배포테스트 완료 이후 배포하고자 하는 ```환경을 선택```하고 배포하기를 클릭하여 CSP에 배포 할 수 있습니다.

:::tip

캔버스에 그린 클라우드 인프라를 CSP에 배포하기 위해서는 크레덴셜이 등록되어 있어야 합니다.

:::
