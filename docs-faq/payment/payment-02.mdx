# 구독을 취소하고 싶어요
<hr/>
<br />

1. 클라우드 스튜디오 메인화면에서 좌측의 ```프로필``` 메뉴를 선택합니다.

2. 구독정보에서 ```구독변경```을 클릭하여 구독플랜 페이지로 이동합니다.

3. 현재 구독을 ```FREE 플랜```으로 변경하여 구독을 취소할 수 있습니다.

:::info

구독 취소는 취소 다음달부터 적용됩니다.

:::
