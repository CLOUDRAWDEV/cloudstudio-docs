# Overview
##### 2022.07.15 / 읽는 데 1분 걸림(소요시간)

*****
Cloud Studio는 3개의 Cloud Service Provier(이하, CSP)인 Azure, AWS, Naver Cloud Platform을 대상으로 인프라 아키텍처의 시각화 및 자동화 서비스를 제공합니다. 
서비스를 편하게 사용할 수 있도록 각 CSP별 Well-Architecture를 Best Pracetice 템플릿으로 제공합니다.
사용자는 Cloud Studio에서 프로젝트를 만들 때, Best Pracetice 템플릿을 import해서 아키텍처를 쉽게 구성할 수 있습니다.
