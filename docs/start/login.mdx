클라우드 스튜디오는 비회원으로 접속할 수 없습니다. 로그인 전 `회원가입`을 먼저 진행해주세요.

🔗  [클라우드 스튜디오 회원가입 링크](https://cloudstudio.cloudraw.kr/signup)

✏️  <Link to="/docs/start/sign-up">회원가입하는 방법</Link>

회원가입이 끝났다면 [클라우드 스튜디오 로그인 페이지](https://cloudstudio.cloudraw.kr/signin)에 접속합니다.

<p>&nbsp;</p>

1.  회원가입 시 입력했던 아이디와 비밀번호를 입력합니다.

:::tip

아이디와 비밀번호가 기억나지 않는다면 아이디 찾기와 비밀번호 찾기를 통해 찾습니다.

<Link to="/docs/start/fine-id-password">자세히 보기</Link>
:::

2.  로그인 버튼을 클릭합니다.

<img width="100%" src={login} alt="login img" />

<p>&nbsp;</p>

:::caution

회원가입 후 owner가 admin 페이지에서 가입승인을 해줘야 서비스 로그인을 할 수 있습니다.

:::

<p>&nbsp;</p>

3. 최초 로그인 시 OTP 등록을 먼저 해줘야합니다.
