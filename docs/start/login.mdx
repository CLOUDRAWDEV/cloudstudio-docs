클라우드 스튜디오는 비회원으로 접속할 수 없습니다. 로그인 전 `회원가입`을 먼저 진행해주세요.

🔗  [클라우드 스튜디오 회원가입 링크](https://cstudio.app/signup)

✏️  <Link to="/docs/start/sign-up">회원가입하는 방법</Link>

회원가입이 끝났다면 [클라우드 스튜디오 로그인 페이지](https://cstudio.app/signin)에 접속합니다.

<!--:::caution

회원가입 후 관리자 승인이 필요합니다.

:::-->

<p>&nbsp;</p>

1. 회원가입 시 입력했던 이메일을 통해 아이디와 임시 패스워드를 확인합니다.

</div>

<p>&nbsp;</p>
