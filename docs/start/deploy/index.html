배포관리

last update: 2023.02.01

클라우드 스튜디오는 단순히 시각적인 인프라 아키텍처를 제공하는 것을 넘어서, 캔버스에 그린 인프라를 실제 CSP에 배포할 수 있는 기능을 제공합니다. 이를 통해 사용자들은 쉽게 클라우드 인프라를 구축할 수 있습니다.

 

배포 점검​

IaC의 문법적 오류나 설정이 올바르게 되었는지 가상으로 검증하는 단계입니다. 실제 CSP에 배포되지 않습니다. (--dry-run option과 같은 역할을 합니다.)

  1. 배포테스트를 진행하기 위해서 배포관리 > 배포 점검를 클릭합니다.

  1. 크레덴셜을 선택합니다.
  2. 태그 메세지를 입력합니다.
  3. 배포 점검하기 버튼을 클릭합니다.

 

loader가 돌아가고 있으면 배포 점검 중이라는 뜻입니다. loader가 사라질 때까지 기다려주세요.

 

info

배포테스트를 진행하기 전 크레덴셜을 필수적으로 등록해야 합니다. 크레덴셜을 등록하는 방법은 여기 를 참고해주세요.

 

 

배포 점검 결과​

배포 점검 결과를 확인할 수 있습니다. 배포가 완료된 환경은 각 CSP 페이지에서 확인할 수 있습니다.

  1. 배포 점검 결과를 확인하기 위해서 배포관리 > 배포 점검 결과를 클릭합니다.
  1. 최근 배포 점검 이력을 확인합니다.
  2. 배포하고자하는 이력을 클릭한 후 배포하기 버튼을 클릭합니다.

 

 

배포이력​

배포이력은 실제 배포된 인프라에 대한 이력을 보여줍니다. DEV(개발) / TEST(테스트) / PROD(운영) 각 탭마다의 최근 배포 이력을 확인할 수 있습니다.

  1. 배포 이력을 확인하기 위해서 배포관리 > 배포이력을 클릭합니다.

  1. 최근 배포 이력을 확인합니다. 이력을 삭제하려면 원하는 이력을 선택한 후 삭제하기 버튼을 클릭합니다.
