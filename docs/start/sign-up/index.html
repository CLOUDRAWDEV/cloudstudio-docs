회원가입

last update: 2022.10.24

클라우드 스튜디오를 시작하기 위해서 회원가입을 진행해야 합니다.아래의 링크에 접속합니다.

Start CloudStudio

클라우드 스튜디오 계정을 생성하는 방법은 두 가지가 있습니다.

  • 자체 계정 생성하기
  • 소셜 아이디로 계정을 생성하기

자체 계정 생성하기​

  1. 회원가입 버튼을 클릭합니다.
  2. 사용할 이메일 주소를 입력합니다. 예) cloudstudio@cloudraw.com
  3. 인증번호 전송 버튼을 클릭한 후 입력한 이메일 주소로 보내진 인증번호 여섯 자리를 확인합니다.
  4. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 확인 버튼을 클릭하여 이메일 인증을 완료합니다.
info

인증번호 재전송은 3분마다 가능합니다. 인증번호 재전송을 위해 재전송 아이콘을 클릭합니다.

  1. 사용할 아이디를 입력합니다. 아이디를 다음 항목을 만족해야 합니다.
  • 중복되지 않는 아이디
  • 영문(소문자) 및 숫자 사용, 6자리 이상 16자리 이내
  1. 비밀번호를 입력합니다. 비밀번호는 다음 항목을 만족해야 합니다.
  • 영어,숫자,특수문자 3종류 모두 포함
  • 8자리 이상 20자리 이내
  1. 비밀번호를 확인하기 위해 다시 한번 입력합니다. 비밀번호가 일치한다면 다음 단계로 넘어갑니다.
  2. 클라우드 스튜디오 이용약관과 개인정보 수집 및 이용동의를 꼼꼼하게 읽어본 후 체크박스를 클릭해 체크합니다.
  3. 다음 버튼을 클릭하여 회원가입을 완료합니다.

 

소셜 아이디로 계정 생성하기​

 

  • Naver 계정으로 회원가입
  • Kakao 계정으로 회원가입
  • Google 계정으로 회원가입
1. 클라우드 스튜디오 로그인 페이지에서 네이버 아이콘을 클릭합니다.
2. 회원가입 페이지가 나오면 사용할 네이버 이메일을 입력합니다.
3. 아이디를 입력한 후 중복검사를 클릭합니다.
4. 클라우드 스튜디오 이용약관과 개인정보 수집 및 이용 동의 내용을 꼼꼼하게 읽은 후 체크박스를 클릭하여 체크합니다.
5. 다음 버튼을 클릭하여 클라우드 스튜디오 네이버 계정 회원가입을 완료합니다.
1. 클라우드 스튜디오 로그인 페이지에서 카카오 아이콘을 클릭합니다.
2. 개인정보제공에 동의하기를 클릭합니다.
3. 회원가입 페이지가 나오면 사용할 카카오 이메일을 입력합니다.
3. 아이디를 입력한 후 중복검사를 클릭합니다.
4. 클라우드 스튜디오 이용약관과 개인정보 수집 및 이용 동의 내용을 꼼꼼하게 읽은 후 체크박스를 클릭하여 체크합니다.
5. 다음 버튼을 클릭하여 클라우드 스튜디오 카카오 계정 회원가입을 완료합니다.
1. 클라우드 스튜디오 로그인 페이지에서 구글 아이콘을 클릭합니다.
2. 회원가입 페이지가 나오면 사용할 구글 이메일을 입력합니다.
3. 아이디를 입력한 후 중복검사를 클릭합니다.
4. 클라우드 스튜디오 이용약관과 개인정보 수집 및 이용 동의 내용을 꼼꼼하게 읽은 후 체크박스를 클릭하여 체크합니다.
5. 다음 버튼을 클릭하여 클라우드 스튜디오 구글 계정 회원가입을 완료합니다.
