<br />

## 프로젝트 상세정보 수정하기

### 프로젝트명 수정하기

1. 프로젝트 상세정보를 수정하기 위해 내 템플릿에서 ```수정을 원하는 프로젝트를 클릭```합니다. 
2. 팝업이 뜨면 프로젝트명에 ```편집 아이콘```을 클릭합니다.
3. ```프로젝트명```을 수정합니다.
4. 다시 ```편집 아이콘```을 클릭하여 수정을 마무리 합니다.

## 프로젝트 복사 및 삭제하기

### 프로젝트 복사

1. 프로젝트를 복사하기 위해 내 템플릿에서 수정을 원하는 프로젝트의 ```더보기 아이콘```을 클릭합니다. 
2. ```프로젝트 복사```를 클릭합니다.
3. 팝업창이 뜨면 프로젝트를 복사할 폴더를 선택합니다.
4. 프로젝트 명을 작성합니다.

> - 영문, 한글, 숫자 허용
> - 2 ~ 20 자리

5. 프로젝트 설명을 작성합니다.

> - 100자 이내

6. 복사하기를 클릭합니다.

</div>

<p>&nbsp;</p>

### 프로젝트 삭제

1. 프로젝트를 삭제하기 위해 내 템플릿에서 삭제를 원하는 프로젝트의 ```더보기 아이콘```을 클릭합니다.
2. ```프로젝트 삭제```를 클릭합니다.
3. 팝업창이 뜨면 프로젝트명을 다시 한번 입력해줍니다.
4. ```삭제하기``` 버튼을 클릭합니다.

## 캔버스 수정

1. 캔버스를 수정하기 위해 내 템플릿에서 수정을 원하는 프로젝트를 클릭합니다. 
2. 팝업이 열리면 우측의 프로젝트 수정하기를 클릭합니다.
