# 캔버스 편집하기
<hr />
 
캔버스는 리소스를 Drag & Drop 하여 클라우드 인프라를 그리고 버전을 관리하고 배포까지 할 수 있는 <highlight color="#206FD6">클라우드 스튜디오의 핵심 서비스</highlight>입니다.
캔버스로 구성한 인프라는 단순히 시각적 아키텍처로 사용할 수도 있고 원하는 CSP 사에 직접 배포까지 할 수 있습니다.

✂️ 자유롭게 캔버스를 편집해보세요! 


## 리소스 Drag & Drop {#drag-drop}
Drag & Drop은 캔버스를 구성할 수 있는 가장 단순한 기능입니다.
좌측 리소스 리스트에서 원하는 리소스를 끌어다 캔버스에 Drag & Drop 해보세요!

    <img src={ dragDrop } alt="드래그앤드롭 이미지" />
</div>


## 리소스 연결하기 {#resource-connect}
캔버스에 Drag & Drop 한 리소스들을 연결 가능한 리소스들끼리 선으로 연결시켜줍니다.

<div className={styles.imgWrap}>
    <img src={ line } alt="리소스 연결 이미지" />
</div>

<br />

리소스는 크게 아래의 ```세 가지``` 유형으로 나눠집니다. 아래의 리소스 목록을 참고하여 캔버스 리소스들을 연결시켜줍니다.

> - 기본형 : 기본 리소스들을 보여줍니다.
> - 연결형 : 해당 리소스에 연결될 수 있는 리소스 목록을 보여줍니다.
> - 그룹형 : 해당 그룹에 속할 수 있는 리소스 목록을 보여줍니다.


## 리소스 옵션 편집 {#option-edit}
리소스를 ```더블 클릭```하여 리소스 설정값을 변경할 수 있는 ```편집창```을 열어줍니다.
❓물음표 아이콘에 마우스를 가져다 대면 각 항목에 대한 설명을 확인할 수 있습니다. 설명을 참고하여 원하는 값들을 설정해 줍니다.

<div className={styles.imgWrap}>
    <img src={ edit } alt="옵션 편집 이미지" />
</div>

## 오류 수정하기 {#error-correction}
🔴 빨간 점은 해당 리소스에 오류가 있다는 표시입니다.  <highlight color="#FF7272">오류가 있으면 캔버스를 저장 및 배포할 수 없습니다.</highlight>
오류를 해결하기 위해 해당 리소스를 ```더블 클릭```한 후 오류창을 클릭합니다.
오류가 난 원인을 확인한 후 캔버스를 수정합니다.

    <img src={ error } alt="오류 수정 이미지" />
</div>

## 툴 박스 사용법 {#toolbox}
상단에 위치한 툴 박스는 캔버스를 꾸밀 수 있는 다양한 기능을 제공합니다.

1.  실행 취소 : 한 칸 이전으로 되돌립니다.
2.  다시 하기 : 한 칸 앞으로 돌아갑니다.
3.  리소스 리스트 : 리소스 리스트를 껐다 켜는 버튼입니다.
4.  폰트 설정 : 폰트의 종류와 사이즈를 설정할 수 있습니다.
5.  굵기 : 폰트의 굵기를 조정합니다.
6.  기울이기 : 폰트의 기울기를 조정합니다.
7.  밑줄 : 폰트에 밑줄을 추가합니다.
8.  색상 : 폰트의 색상을 조정합니다. 
9.  연결선 : 리소스 연결선의 굵기와 종류를 조정합니다.
10.  화면비율: 캔버스 화면비율을 조정합니다.
11.  그리드 : 캔버스 그리드를 해제할 수 있습니다.
