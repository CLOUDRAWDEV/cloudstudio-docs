캔버스는 리소스를 ```Drag & Drop``` 하여 클라우드 인프라를 그리고 버전을 관리하고 배포까지 할 수 있는 <highlight color="#206FD6">클라우드 스튜디오의 핵심 서비스</highlight>입니다.
캔버스로 구성한 인프라는 단순히 시각적 아키텍처로 사용할 수도 있고 원하는 CSP 사에 직접 배포까지 할 수 있습니다.


✂️ 자유롭게 캔버스를 편집해보세요!

## 리소스 Drag & Drop {#drag-drop}

```Drag & Drop```은 캔버스를 구성할 수 있는 가장 단순한 기능입니다.
좌측 ```리소스 리스트```에서 원하는 리소스를 끌어다 캔버스에 Drag & Drop 해보세요!
서로 연결될 수 없는 리소스들은 빨간선과 함께 튕깁니다.

리소스는 크게 아래의 ```네 가지``` 유형으로 나눠집니다. 아래의 리소스 목록을 참고하여 캔버스 리소스들을 연결시켜줍니다.

| 이미지       | 이름         | 설명         |
| ---------- | -------------- | ------------ |

## 리소스 옵션 편집 {#option-edit}
리소스를 ```더블 클릭```하여 리소스 설정값을 변경할 수 있는 ```편집창```을 열어줍니다.
```자세히 보기 아이콘```을 클릭하여 해당 리소스에 대한 설명을 확인할 수 있습니다.
❓ ```물음표 아이콘```에 마우스를 가져다 대면 각 옵션에 대한 설명을 확인할 수 있습니다. 설명을 참고하여 원하는 값들을 설정해 줍니다.

    <img src={ edit } alt="옵션 편집 이미지" />
</div>

## 오류 수정하기 {#error-correction}
🔴 빨간 점은 해당 리소스에 오류가 있다는 표시입니다.  <highlight color="#FF7272">오류가 있으면 캔버스를 저장 및 배포할 수 없습니다.</highlight>
오류를 해결하기 위해 캔버스에서 오류가 난 리소스를 ```더블 클릭```하여 캔버스 오류창을 열어줍니다.
캔버스 오류창에서 수정하려는 특정 오류를 더블 클릭하면 오류가 난 곳으로 이동합니다. 오류 원인을 참고하여 오류들을 수정합니다.

<div className={styles.imgWrap}>
    <img src={ error } alt="오류 수정 이미지" />
</div>

<p>&nbsp;</p>

### 오류의 종류

| 종류       | 설명        | 
| ---------- | -------------- | 
|  value | 입력값이 잘못된 경우 발생하는 오류     |     
| group | 어떠한 그룹에 속해있지 않은 경우 발생하는 오류    | 
| line | 필요한 리소스가 연결되어 있지 않은 경우 발생하는 오류     |
