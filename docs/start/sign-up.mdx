클라우드 스튜디오 회원가입을 위해 아래의 링크에 접속합니다.

[Start CloudStudio](https://cstudio.app/signin)

1. 회원가입 버튼을 클릭합니다.
2. 사용할 이메일 주소를 입력합니다. 예) `cloudstudio@cloudraw.kr`
3. `코드 전송` 버튼을 클릭한 후 입력한 이메일 주소로 보내진 `인증번호 여섯 자리`를 확인합니다.
4. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 `확인` 버튼을 클릭하여 이메일 인증을 완료합니다.

:::info
인증번호는 `3분 이내`에 입력해야 합니다. 인증시간이 만료되면 인증번호 재전송을 위해 **재전송 아이콘**을 클릭해주세요.
:::

5. 사용할 아이디를 입력한 후 `중복검사` 버튼을 클릭합니다.

:::info
아이디는 아래의 조건을 만족해야합니다.

-   영소문자로 시작
-   영문 및 숫자만을 사용
-   6자리 이상 16자리 이내

:::

6. 이용약관에 동의합니다.
7. `회원가입`을 클릭합니다.
8. 등록한 이메일로 임시 비밀번호가 전송됩니다.
