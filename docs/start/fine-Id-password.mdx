# 아이디/비밀번호 찾기
<hr/>

## 아이디 찾기

1. 클라우드 스튜디오 로그인 페이지에서 **[아이디 찾기](https://cloudstudio.cloudraw.kr/forgot/id)**를 클릭합니다.
2. 이메일 주소를 입력한 후 인증번호 전송을 클릭합니다. 예) ```cloudstudio@cloudraw.com```
3. 입력한 이메일 주소를 통해 전송된 ```여섯 자리의 인증번호```를 확인합니다.
4. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 ```인증번호 확인```을 클릭합니다.
5. 인증번호 확인이 인증 완료로 바뀌었다면 ```다음``` 버튼을 클릭합니다.
6. 이메일 주소로 전송된 아이디를 확인합니다.

## 비밀번호 찾기

1. 클라우드 스튜디오 로그인 페이지에서 **[비밀번호 찾기](https://cloudstudio.cloudraw.kr/forgot/pw)**를 클릭합니다.
2. 가입했던 아이디를 입력합니다.
3. 이메일 주소를 입력한 후 ```인증번호 전송```을 클릭합니다. 예) ```cloudstudio@cloudraw.com``` 
4. 입력한 이메일 주소를 통해 전송된 ```여섯 자리의 인증번호```를 확인합니다.
5. 클라우드 스튜디오로 돌아와 인증번호를 입력한 후 ```인증번호 확인```을 클릭합니다.
6. 인증번호 확인이 인증 완료로 바뀌었다면 ```다음``` 버튼을 클릭합니다.

<div className={styles.imgWrap}>
    <img src={findPassword} alt="find password img" />
</div>

7. ```비밀번호 재설정``` 버튼을 클릭합니다.
8. 새 비밀번호를 입력합니다. 새 비밀번호는 다음 항목을 만족해야 합니다.

>  - 영어,숫자,특수문자 3종류 모두 포함
>  - 8자리 이상 20자리 이내

9. 비밀번호 확인을 위해 다시 한번 입력합니다.
10. 비밀번호 재설정을 클릭하여 비밀번호 변경을 완료합니다.
