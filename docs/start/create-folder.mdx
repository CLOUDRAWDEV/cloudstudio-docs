# 새 폴더 생성하기 
<hr />

📁폴더는 여러 개의 프로젝트들을 관리할 수 있게 해주는 기능입니다. 생성한 폴더에 프로젝트들을 담아 정리해 보세요!

1. 새로운 폴더를 생성하기 위해 클라우드 스튜디오 내 템플릿 페이지에서 ```폴더 추가 아이콘```을 클릭합니다.
2. 폴더를 추가할 수 있는 팝업이 뜨면 ```폴더명```과 ```폴더 설명```을 작성합니다. 폴더 설명은 생략할 수 있습니다.
    폴더명과 폴더 설명은 ```아래의 조건```을 만족해야 합니다.
    > 폴더명
    -  영문, 한글, 숫자 허용
    -  2 ~ 12자리
    -  공백 없이

    > 폴더 설명
    - 최대 100자 까지 작성 가능
3. 팝업창 내용 작성이 끝나면 ```생성하기 버튼```을 클릭합니다.
4. 생성된 폴더 목록은 ```내 템플릿 > 내 폴더``` 에서 확인할 수 있습니다.
