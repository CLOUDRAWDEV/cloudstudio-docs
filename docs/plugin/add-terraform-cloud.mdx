## Terraform Cloud 등록하기

클라우드 스튜디오 캔버스에 그린 인프라의 유효성 검사나 배포를 하기 위해서는 Terraform Cloud 계정 연동이 필요합니다.
또한 Terraform Cloud 계정 연동을 하기 위해서는 [Github 계정 연동](/docs/plugin/add-github)이 선행되어야 합니다.

<br />

<details>
    <summary>Terraform Cloud란?</summary>
    <h5>
        Terraform Cloud는 Terraform의 관리형 서비스로, 인프라 자동화 및 협업을
        지원합니다. 원격 상태 저장, 계획 및 적용 작업을 중앙에서 관리하고,
        팀과의 협업을 용이하게 합니다.
    </h5>
</details>

<p>&nbsp;</p>

1. [Terraform Cloud](https://app.terraform.io/session) 회원가입 및 로그인 후, 좌측 사이드바의 `프로필 아이콘 - Account settings`를 클릭합니다.

<img width="100%" src={terraformcloud01} alt="otp img" />

<p>&nbsp;</p>

2. 좌측 사이드바 하단의 `Tokens`를 클릭한 후, 우측의 `Create an API token` 버튼을 클릭합니다.

<img width="100%" src={terraformcloud02} alt="otp img" />

<p>&nbsp;</p>

3. 추후에 이 토큰을 식별할 수 있게 할 `Description`을 입력하고 토큰 유효기간을 설정한 후, `Generate token` 버튼을 클릭합니다.

<img width="100%" src={terraformcloud03} alt="otp img" />

<p>&nbsp;</p>

4. 발급된 토큰을 확인합니다.

<img width="100%" src={terraformcloud04} alt="otp img" />

<p>&nbsp;</p>

:::caution

발급된 토큰은 다시 확인할 수 없으니, 반드시 따로 저장해두도록 합니다.

:::

<p>&nbsp;</p>

5. 클라우드 스튜디오에서 `내 정보 - 플러그인 관리` 를 클릭한 후, Terraform Cloud 계정연동 `+` 버튼을 클릭합니다.

<img width="100%" src={github07} alt="otp img" />

<p>&nbsp;</p>

:::caution

사전에 [Github 계정 연동](/docs/plugin/add-github)이 선행되어야 Terraform Cloud 계정 연동이 가능합니다.

:::

<p>&nbsp;</p>

6. 발급받은 Terraform Cloud `Token` 을 입력한 후, `등록하기` 버튼을 클릭합니다.

<img width="100%" src={terraformcloud05} alt="otp img" />

<p>&nbsp;</p>

7. 유효한 Terraform Cloud 계정의 `Token` 값 입력이 완료되면, 정상적으로 Terraform Cloud 연동이 완료됩니다.

<img width="100%" src={terraformcloud06} alt="otp img" />
