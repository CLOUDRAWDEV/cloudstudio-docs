## Github 등록하기

클라우드 스튜디오에서 캔버스의 버전 관리를 하기 위해서는 Github 계정 연동이 필요합니다. 또한 프로젝트의 유효성 검사와 배포를 하기 위한 Terraform Cloud 계정 연동을
위해서는 Github 계정 연동이 선행되어야 합니다.

<br />

<details>
    <summary>Github란?</summary>
    <h5>
        GitHub는 소프트웨어 개발 프로젝트를 호스팅하고 관리할 수 있는
        플랫폼입니다.
    </h5>
</details>

<p>&nbsp;</p>

1. [Github](https://github.com/) 회원가입 및 로그인 후, 우측상단의 `프로필 아이콘 - Settings`를 클릭합니다.

<img width="100%" src={github01} alt="otp img" />

<p>&nbsp;</p>

2. 좌측 사이드바 하단의 `Developer settings`를 클릭합니다.

<img width="100%" src={github02} alt="otp img" />

<p>&nbsp;</p>

3. 좌측 사이드바에서 `Personal access tokens - Tokens (classic)`를 클릭한 후,
   우측의 `Generate new token - Generate new token (classic)`을 선택합니다.

<img width="100%" src={github03} alt="otp img" />

<p>&nbsp;</p>

4. 하단에 Github 토큰이 접근할 수 있는 권한을 선택하는 옵션인 Select scopes에서 `repo`를 선택하고 하단에 `Generate token` 버튼을 클릭하여 토큰을 생성합니다.

<img width="100%" src={github04} alt="otp img" />

<p>&nbsp;</p>

:::caution

발급된 토큰은 다시 확인할 수 없으니, 반드시 따로 저장해두도록 합니다.

:::

<p>&nbsp;</p>

5. 클라우드 스튜디오에서 `내 정보 - 플러그인 관리` 를 클릭한 후, Github 계정연동 `+` 버튼을 클릭합니다.

<img width="100%" src={github05} alt="otp img" />

<p>&nbsp;</p>

6. Github 계정의 `UserName` 과 발급받은 `Token` 을 입력한 후, `등록하기` 버튼을 클릭합니다.

<img width="100%" src={github06} alt="otp img" />

<p>&nbsp;</p>

7. 유효한 Github 계정의 `UserName` 과 발급받은 `Token` 의 입력이 완료되면, 정상적으로 Github 연동이 완료됩니다.

<img width="100%" src={github07} alt="otp img" />
