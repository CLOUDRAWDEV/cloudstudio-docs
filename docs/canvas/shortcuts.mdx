클라우드 스튜디오 캔버스에서 사용할 수 있는 `단축키`입니다.

| 단축키                       | 설명              |
| ---------------------------- | ----------------- |
| Ctrl + 마우스 스크롤         | 화면 확대 및 축소 |
| Space + 마우스 클릭 + 드래그 | 화면 이동         |
| Delete                       | 삭제하기          |
| Ctrl + Z                     | 되돌리기          |
| Ctrl + C                     | 복사              |
| Ctrl + V                     | 붙여넣기          |
