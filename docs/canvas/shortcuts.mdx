클라우드 스튜디오 캔버스에서 사용할 수 있는 ```단축키```입니다.

| 단축키 | 설명           |
| ---------- | -------------- | 
| ctrl 키 + 마우스 스크롤 | 화면 확대 및 축소 |
| space 키 + 마우스 좌클릭 + 드래그 | 화면 이동 |
| delete 키 | 삭제하기 |
| ctrl 키 + z | 되돌리기 |
| ctrl 키 + c | 복사 |
| ctrl 키 + v | 붙여넣기 |
