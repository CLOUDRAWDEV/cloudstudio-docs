# 내보내기
<hr />
<br />

클라우드 스튜디오에서 파일 내보내기 시 세 가지 형식의 파일을 얻을 수 있습니다.

## 형식
- png 파일
- 세 가지의 tf 파일 (main, output, variables)
- nodes.json 파일

### 1. architecture.png 파일
사용자가 그린 캔버스 아키텍처의 png 파일입니다.

### 2. tf 파일
사용자는 세 가지의 tf 파일을 얻을 수 있습니다.

```main.tf```: main.tf은 생성한 인프라의 뼈대를 보여주는 테라폼 파일입니다.  
```output.tf```: 인프라 배포 후 관련 정보를 알려주는 테라폼 파일입니다.  
```variables```: variables.tf는 변수들을 보여주는 테라폼 파일입니다.

### 3. nodes.json 파일
캔버스를 그리기 위한 좌표 데이터입니다.
