## 리소스 개요

클러스터 수준과 모든 네임스페이스에서 적용되는 역할을 사용자에게 참조하는 쿠버네티스 리소스

## 연관 리소스

### 상위 리소스

-   [**Cluster**](https://docs.cstudio.app/docs/resource/kubernetes/Base/Cluster)


### 연결 리소스

-   [**ClusterRole**](https://docs.cstudio.app/docs/resource/kubernetes/Rbac/ClusterRole)



## 리소스 설정 값

-   `service_name` : ClusterRoleBinding 리소스 이름
-   `annotations` : 쿠버네티스 시스템에서 필요한 정보들을 표시해 주기 위해서 사용하는 key-value
-   `labels` : 객체를 구분하기 위해서 사용자가 임의로 원하는 값을 지정
-   `role_name` : 주체를 바인딩할 Role 이름을 지정
-   `subject` : Role을 부여할 주체 설정 
    -   `name` : 바인딩할 주체의 이름을 입력
    -   `namespace` : 바인딩할 주체의 네임스페이스를 입력
    -   `api_group` : 승인 결정을 내리는 API Group - `rbac.authorization.k8s.io`,`User`,`Group`
    -   `kind` : 바인딩할 주체의 유형을 선택- `ServiceAccount`,`User`,`Group`
