## 리소스 개요

기본 헬름 차트를 사용자가 원하는 값으로 설정하여 배포할 수 있는 리소스입니다.

## 연관 리소스

### 상위 리소스

-   [**Cluster**](https://docs.cstudio.app/docs/resource/kubernetes/Base/Cluster)

### 연결 리소스



## 리소스 설정 값

-   `service_name` : Helm Custom 차트 릴리즈 이름
-   `namespace` : Helm Custom가 배포될 네임스페이스
-   `repository` : 헬름차트를 저장하고 공유하기 위한 저장소 (커스텀할 차트가 저장된 저장소를 지정)
-   `chart` : 커스텀할 헬름 차트 이름을 지정
-   `version` : 커스텀할 헬름 차트의 버전을 지정
-   `set` : 기본 차트의 값을 동적으로 변경
