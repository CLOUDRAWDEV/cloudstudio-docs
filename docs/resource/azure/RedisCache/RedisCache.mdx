## 리소스 개요

Azure Redis Cache는 오픈 소스 Redis Cache 엔진을 기반으로 하는 완전히 관리되는 메모리 내 데이터 저장소입니다.
Microsoft Azure 클라우드 플랫폼에서 서비스로 제공되며 확장성이 뛰어나고 응답성이 뛰어난 애플리케이션을 구축하는 데 사용됩니다.
Azure Redis Cache는 자주 그리고 빠르게 액세스해야 하는 세션 데이터, 사용자 기본 설정 및 애플리케이션 상태와 같은 데이터를 저장하고 검색하는 데 사용할 수 있습니다.
문자열, 해시, 목록, 세트 및 정렬된 세트와 같은 데이터 구조를 지원합니다.
Azure Redis Cache는 데이터 지속성, 고가용성 및 자동 크기 조정과 같은 기능을 제공하므로 빠르고 안정적인 데이터 액세스가 필요한 애플리케이션을 빌드하는 데 널리 사용됩니다.
Azure App Service, Azure Functions 및 Azure Logic Apps와 같은 다른 Azure 서비스와 통합할 수도 있습니다.

### 상위 리소스

-   [**Resource Group**](https://docs.cstudio.app/docs/resource/azure/Base/ResourceGroup)

## 리소스 설정 값

-   `sku_name` : Redis 캐시의 가격 책정 계층 이름 - `Basic`,`Standard`,`Premium`,`Enterprise`

-   `cache_type` : Redis 캐시의 가격 책정 그룹 및 배포를 위한 Redis Cache의 크기 - `azureRedisCacheSku`

-   `version` : Redis 버전 - `6`

-   `enable_non_ssl_port` : Redis에 non-SSL port를 적용할 것인지 - `true`,`false`

-   `minimum_tls_version` : Redis의 TLS 최소 버전 - `1.0`,`1.1`,`1.2`

-   `public_network_access_enabled` : 공용 IP 주소를 통해 공개적으로 Redis 캐시에 접근 허용 - `true`,`false`

-   `redis_firewall_rule`

    -   `start_ip` : 시작 IP 주소 범위
    -   `end_ip` : 종료 IP 주소 범위
    -   `included_redis_cache_name` : 방화벽 규칙을 적용할 Redis 캐시 이름

-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
