## 리소스 개요
Azure 구독은 고객이 Azure 클라우드 컴퓨팅 플랫폼 및 해당 서비스에 액세스하고 사용할 수 있도록 하는 지불 계획입니다. 
사용자에게 가상 머신, 스토리지, 데이터베이스 및 기타 클라우드 기반 서비스와 같은 다양한 서비스에 대한 액세스를 제공합니다. 
Azure 구독은 각기 다른 기능과 가격 계획이 있는 다양한 계층에서 사용할 수 있으며 사용자는 비즈니스 요구 사항에 따라 필요한 리소스 수준을 선택할 수 있습니다.
