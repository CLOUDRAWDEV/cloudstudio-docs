## 리소스 개요
 Microsoft Azure에서 제공하는 서버리스 컴퓨팅 플랫폼으로 개발자가 클라우드에서 함수라고 하는 작은 코드 조각을 실행할 수 있도록 합니다.   
 이를 통해 개발자는 인프라 관리에 대한 걱정 없이 확장 가능하고 유연한 플랫폼에서 이벤트 기반 애플리케이션을 구축하고 배포할 수 있습니다.

## 리소스 설정 값
- ```kind``` : App Service에 적용할 운영체제 유형 ```Linux```
- ```sku_tier``` : App Service 요금제의 가격 책정 계층
- ```delegated_subnet_name``` : 가상 네트워크 위임을 위한 Subnet 이름
- ```function_app``` : 앱서비스의 이름
    -  ```storage_account_name``` : Function App의 사용을 위한 범용 Storage Account 이름
    -  ```https_only``` : 암호화 및 인증된 보안 연결을 위해 Function App에 HTTPS를 통한 접근만 허용 - ```true```, ```false```
    -  ```detailed_error_messages_enabled``` : HTTP 오류 발생 시, 오류 페이지에 자세한 오류 메시지를 포함 - ```true```, ```false```
    -  ```auth_settings_enabled``` : Function App의 권한 활성화 - ```true```, ```false```
    -  ```http2_enabled``` : 보안을 위해 최신 버전의 HTTP 활성화 - ```true```, ```false```
