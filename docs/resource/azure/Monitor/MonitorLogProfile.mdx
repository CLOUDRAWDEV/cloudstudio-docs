## 리소스 개요
Azure Monitor 로그 프로필은 로그 데이터를 저장할 위치와 처리 방법을 지정할 수 있는 Microsoft Azure의 기능입니다. 
단일 인터페이스를 사용하여 사용자 지정 로그 및 애플리케이션 로그를 비롯한 다양한 Azure 서비스 및 기타 소스에서 로그를 수집하고 분석할 수 있습니다.
로그 프로필은 Azure Monitor가 로그 데이터를 처리하는 방법을 결정하는 설정 모음입니다. 
특정 Azure 구독 또는 리소스 그룹에 대해 캡처해야 하는 스토리지 계정, 보존 기간 및 로그 범주를 정의합니다. 
로그 프로필에는 관련 데이터만 수집되도록 필터 및 데이터 샘플링을 포함하여 데이터 보존 및 집계에 대한 설정도 포함될 수 있습니다.

## 연관 리소스
### 연결 리소스
- [**Storage Account**](https://docs.cstudio.app/docs/resource/azure/Storage/StorageAccount)



## 리소스 설정 값
- ```categories``` : 로그의 유형 목록 -  ```Action```,```Delete```, ```write```

- ```locations``` : 활동 로그가 저장되거나 스트리밍되는 Azure 위치 목록

- ```retention_enabled``` : 보존 정책 활성화 - ```true```, ```false```

- ```retention_day``` : 보존 정책을 적용한 데이터 보존 일 수(0일 경우 무기한)

- ```linked_storage_account_name``` : 활동 로그가 저장되는 Storage 계정의 이름
