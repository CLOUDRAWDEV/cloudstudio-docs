## 리소스 개요

Azure Container Registry는 오픈 소스 Docker Registry 2.0을 기반으로 하는 관리형 레지스트리 서비스입니다.
Azure 컨테이너 레지스트리를 만들고 유지 관리하여 컨테이너 이미지와 관련 아티팩트를 저장하고 관리합니다.

## 리소스 설정 값

-   `sku` : Container Registry의 성능 및 확장성을 위한 SKU 유형 - `Basic`, `Standard`, `Premium`

-   `admin_enabled` : 관리자 활성화 - `true`, `false`

-   `public_network_access_enabled` : Public 네트워크 접근 허용 - `true`, `false`

-   `network_rule_set` : SKU 유형이 Premium일 경우 생성 가능
    -   `default_action` : Rule과 일치하지 않는 요청에 대한 동작 - `Deny`, `Allow`
    -   `ip_ranges` : Rule과 일치하는 요청의 IP 주소 CIDR
-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
