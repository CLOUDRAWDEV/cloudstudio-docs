## 리소스 개요

Azure Virtual Machines(가상 머신)은 클라우드에서 실행되는 가상 서버입니다.
가상 머신에는 하나 이상의 디스크가 포함되며, 이들 디스크는 가상 머신의 운영 체제(OS)와 데이터를 저장하는 데 사용됩니다.
디스크는 가상 머신의 영구 저장소 역할을 합니다.

### 연결 리소스

## 리소스 설정 값

-   `storage_type` : 디스크를 백업하기 위한 스토리지 유형
-   `size_gb` : 디스크의 크기(GB) - `1` ~ `32767`
-   `enabled_public_network_access` : 디스크에 퍼블릭 네트워크 접근 허용 - `true`, `false`
-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
