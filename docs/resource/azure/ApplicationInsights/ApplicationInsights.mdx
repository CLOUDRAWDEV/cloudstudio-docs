## 리소스 개요

Azure Application Insights는 개발자가 라이브 애플리케이션을 모니터링하고 분석하는 데 도움이 되는 Microsoft에서 제공하는 클라우드 기반 서비스입니다.
이를 통해 개발자는 애플리케이션의 문제를 감지 및 진단하고 애플리케이션의 성능에 대한 실시간 통찰력을 얻을 수 있습니다.
Application Insights를 사용하여 개발자는 초당 요청, 응답 시간 및 오류율을 포함하여 애플리케이션의 성능 및 사용량을 추적할 수 있습니다.
또한 애플리케이션이 사용하는 종속성 및 외부 서비스에 대한 가시성을 얻을 수 있습니다. 이를 통해 문제를 빠르고 효율적으로 해결할 수 있습니다.

-   [**Log Analytics Workspace**](https://docs.cstudio.app/docs/resource/azure/LogAnalytics/LogAnalyticsWorkspace)

## 리소스 설정 값

-   `type` : 생성할 애플리케이션 Insight의 플랫폼 유형 - `ios`,`java`,`Node.js`,`other`,`MobileCenter`,`phone`,`store`,`web`
-   `retention_in_days` : 데이터 보관 일 수 - `30`,`60`,`90`,`120`,`180`,`270`,`365`,`550`,`730`
-   `linked_workspace_name` : Application Insights가 수집한 데이터를 저장할 Log Analytics Workspace 이름
-   `tag` : 리소스를 카테고리로 분류하기 위한 태그 - 최대 512자, `key`:`value`
