## 리소스 개요
Azure Private Endpoint는 가상 네트워크에서 비공개로 Azure 서비스(예: Azure Storage, Azure SQL Database 및 Azure Cosmos DB)에 액세스할 수 있게 해주는 Azure의 기능입니다. 
이를 통해 인터넷이 아닌 가상 네트워크의 개인 끝점을 통해 이러한 서비스에 안전하게 액세스할 수 있습니다.
Azure 프라이빗 엔드포인트를 사용하면 액세스하는 서비스가 가상 네트워크의 프라이빗 IP 주소에 매핑되고 가상 네트워크와 서비스 간의 트래픽이 프라이빗 링크를 통해 라우팅됩니다. 
이렇게 하면 트래픽이 가상 네트워크의 경계 내에 유지되고 공용 인터넷에 노출되지 않습니다.

## 리소스 설정 값
- ```security_rules``` 
    + ```included_subnet_name``` : Private Endpoint에 할당될 Private IP 주소를 가진 Subnet 이름
    + ```linked_connection_resource_name``` : Private Endpoint가 연결될 리소스 이름
    + ```subresource_names``` : Private Endpoint가 연결될 수 있는 Subresource 이름 목록
    + ```dns_zone_group_name``` : Private Endpoint와 연결할 Private DNS Zone의 그룹 이름
    + ```private_dns_zone_names``` : Private Endpoint와 리소스로 연결할 리소스의 Private DNS Zone 이름 목록
    + ```connection_name``` : Private Endpoint와 리소스의 연결 이름  
    + ```manual_connection``` : 외부 리소스 소유자의 Manual Approval 필요
