## 리소스 개요
Azure 경로 테이블은 VNet(가상 네트워크) 내의 서브넷 간 및 VNet 간에 네트워크 트래픽을 라우팅하는 데 도움이 되는 Microsoft Azure에서 제공하는 서비스입니다. 
경로 테이블에는 서브넷 또는 VNet에서 대상으로의 네트워크 트래픽 경로를 정의하는 경로라는 규칙 집합이 포함되어 있습니다.
경로 테이블은 가상 네트워크의 서브넷 간 트래픽 흐름을 제어하고 가상 머신에 대한 사용자 지정 라우팅 경로를 제공하는 데 유용합니다. 
경로 테이블에서 경로를 정의하면 특정 네트워크 인터페이스, 게이트웨이 또는 서비스 엔드포인트를 통해 특정 대상으로 트래픽을 보낼 수 있습니다.
Azure에서 경로 테이블을 서브넷과 연결할 수 있으며 해당 서브넷의 모든 리소스는 연결된 경로 테이블에 정의된 경로를 자동으로 사용합니다. 
경로 테이블을 가상 네트워크 게이트웨이와 연결하여 가상 네트워크 간의 트래픽 라우팅을 제어할 수도 있습니다.

## 리소스 설정 값
- ```disable_bgp_route_propagation``` : Subnet의 네트워크 인터페이스로 온-프레미스 경로 전파를 방지하기 위해 게이트웨이 경로 전파 비활성화 - ```true```, ```false```
- ```included_subnet_name``` : 경로 테이블과 연결할 Subnet 이름
- ```route``` 
    + ```route_address_prefix``` : 경로를 나타내는 IP 접두사
    + ```next_hop_type``` : 경로의 일치하는 패킷을 처리하기 위해 다음 홉 형식 지정 - ```VirtualNetworkGateway```, ```VnetLocal```, ```Internet```, ```VirtualAppliance```, ```None```
