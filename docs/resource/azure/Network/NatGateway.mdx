## 리소스 개요
Azure NAT(네트워크 주소 변환) 게이트웨이는 Microsoft Azure에서 제공하는 관리형 네트워킹 서비스로, 소스 IP 주소를 숨긴 상태에서 VNet(가상 네트워크) 내의 리소스가 인터넷에 액세스할 수 있도록 합니다.
Azure NAT Gateway를 사용하여 개인 IP 주소를 인터넷에 노출하지 않고 VNet 내의 리소스에 인터넷 연결을 제공할 수 있습니다. 이렇게 하면 인터넷에서 리소스에 직접 액세스하는 것을 방지하여 보안을 개선하는 데 도움이 될 수 있습니다. 
또한 VNet 내의 리소스에 액세스하는 데 필요한 공용 IP 주소의 수를 줄이는 데 도움이 될 수 있습니다.

## 리소스 설정 값
- ```linked_subnet_name``` : Nat Gateway를 연결할 Subnet의 이름

- ```pip_name``` : 공용 IP의 이름 

- ```idle_timeout_in_minutes``` : Nat Gateway에서 유휴 흐름을 제거하는 시간 제한(분)  - ```4```~  ```120```
