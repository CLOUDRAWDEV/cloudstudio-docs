## 리소스 개요
Azure DDoS Protection은 DDoS(Distributed Denial of Service) 공격으로부터 Azure 리소스를 보호하는 데 도움이 되는 Microsoft Azure에서 제공하는 서비스입니다. 
Azure DDoS Protection은 기본, 표준 및 프리미엄의 세 가지 보호 계층을 제공합니다. 
기본 계층은 일반적인 네트워크 계층 공격에 대한 보호를 제공하는 반면 표준 및 프리미엄 계층은 보다 정교한 애플리케이션 계층 공격에 대한 추가 보호를 제공합니다.
Azure DDoS Protection 플랜은 고객이 Azure 구독의 보호된 공용 IP 주소 수에 따라 DDoS Protection 서비스 비용을 지불할 수 있는 가격 책정 플랜입니다. 
이 계획은 고객이 보호 서비스에 대한 추가 비용을 들이지 않고 DDoS 공격으로부터 Azure 리소스를 보호할 수 있는 비용 효율적인 방법을 제공합니다.
