## 리소스 개요

NAT 게이트웨이는 NAT(네트워크 주소 변환) 서비스입니다. 프라이빗 서브넷의 인스턴스가 VPC 외부의 서비스에 연결할 수 있지만 외부 서비스에서 이러한 인스턴스와의 연결을 시작할 수 없도록 NAT 게이트웨이를 사용할 수 있습니다.

## 연관 리소스

### 상위 리소스

-   [**Subnet**](https://docs.cstudio.app/docs/resource/aws/Vpc/Subnet)

## 리소스 설정 값

-   `connectivity_type` : NAT Gateway의 연결 유형 - `public`, `private`
-   `included_subnet_name` : NAT Gateway를 연결할 Subnet의 이름
-   `eip_in_vpc` : EIP가 VPC에 있는지 여부 - `true`, `false`
