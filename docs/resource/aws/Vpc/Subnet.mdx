## 리소스 개요

서브넷은 VPC의 IP 주소 범위입니다. 특정 서브넷에서 EC2 인스턴스와 같은 AWS 리소스를 생성할 수 있습니다.

## 리소스 설정 값

-   `included_vpc_name` : Subnet을 배포할 VPC의 이름
-   `cidr_block` : Subnet의 IP 대역(CIDR)
-   `availability_zone` : Subnet을 배포할 가용성 영역
-   `pip_enable` : 퍼블릭 IP 할당
