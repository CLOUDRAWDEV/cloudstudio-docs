## 리소스 개요

VPC 흐름 로그는 VPC의 네트워크 인터페이스에서 전송되고 수신되는 IP 트래픽에 대한 정보를 수집할 수 있는 기능입니다. 흐름 로그 데이터는 Amazon CloudWatch Logs, Amazon S3 또는 Amazon Kinesis Data Firehose에 게시될 수 있습니다. 흐름 로그를 생성하면 구성한 로그 그룹, 버킷 또는 전송 스트림의 흐름 로그 레코드를 검색하고 볼 수 있습니다.

## 연관 리소스

### 상위 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/aws/Vpc/)

### 연결 리소스

-   [**S3 Bucket**](https://docs.cstudio.app/docs/resource/aws/S3/S3Bucket)
-   [**Cloudwatch Log**](https://docs.cstudio.app/docs/resource/aws/Cloudwatch/Log)

## 리소스 설정 값

-   `linked_log_destination_type` : 로그 데이터를 저장할 저장소 타입
-   `linked_log_destination_name` : 로그 데이터를 저장할 저장소 이름
-   `traffic_type` : 수집할 트래픽 종류
-   `included_vpc_name` : Flow Log를 연결할 VPC 이름
