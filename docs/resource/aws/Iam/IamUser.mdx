## 리소스 개요

AWS의 IAM (Identity and Access Management) 리소스 중 하나인 IAM 사용자는 AWS 계정에 액세스하는 개별 사용자를 나타냅니다. IAM 사용자를 생성하면 해당 사용자에게 AWS 리소스에 대한 액세스 권한을 부여할 수 있습니다. IAM 사용자는 비밀번호 및 MFA (Multi-Factor Authentication) 설정, AWS 리소스에 대한 액세스 권한 관리 등 다양한 기능을 제공합니다. 또한 IAM 정책을 사용하여 특정 리소스에 대한 액세스 권한을 부여할 수 있습니다. IAM 사용자는 필요한 경우 삭제하여 보안성을 높일 수 있습니다.

## 연관 리소스

### 상위 리소스

-   **Iam Group**

## 리소스 설정 값

-   `path` : IAM User 생성되는 경로
-   `user_policy_action` : 접근 가능한 리소스에 가능한 동작을 설정
-   `user_policy_effect` : IAM User의 Action을 허가할지 거부할지 설정 - `Allow`, `Deny`
-   `group_policy_resource` : IAM Group가 접근 가능한 리소스
-   `included_group_name` : IAM User가 포함된 Group Name
