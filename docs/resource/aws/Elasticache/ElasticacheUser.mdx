## 리소스 개요

AWS Elasticache User는 Elasticache 클러스터를 사용하기 위해 필요한 IAM 사용자입니다. Elasticache User는 Elasticache 클러스터를 생성, 수정 및 삭제할 수 있는 권한을 갖습니다. Elasticache User는 AWS IAM에서 생성되며, 일반적으로 Elasticache를 사용하는 애플리케이션 또는 개발자에게 할당됩니다. Elasticache User는 Elasticache 클러스터에 대한 보안 및 액세스 제어를 담당하며, 이를 통해 애플리케이션과 데이터를 보호할 수 있습니다. Elasticache User는 AWS 계정에서 다른 리소스와 함께 사용될 수 있습니다.

## 연관 리소스

### 상위 리소스

-   - [**Subnet**](https://docs.cstudio.app/docs/resource/azure/Network/Subnet)

## 리소스 설정 값

-   `user_id` : 사용자의 ID, 영문자, 하이픈(-) 사용 가능, 문자로 시작, 하이픈으로 끝나거나 하이픈을 두 개 연속으로 포함할 수 없음, 최대 40자
-   `access_string` : 사용자에게 적용할 권한을 입력한 string
-   `passwords` : 사용자의 PW, 최대 2개까지 설정 가능, 16자 이상 최대 128자까지, 입력 가능한 문자 제한 없음
