## 리소스 개요

접근할 AWS region을 저장하는 리소스입니다. Amazon EC2는 전 세계의 여러 곳에서 호스팅되고 있습니다.

## 연관 리소스

### 상위 리소스

-   **base**

### 하위 리소스

-   **cloudtrail-cloudtrail**
-   **cloudwatch-log**
-   **cloudwatch-metric**
-   **direct_connect-dx_connection**
-   **dynamodb-dynamodb**
-   **kms-kms_key**
-   **lambda-lambda_function**
-   **ecr-ecr_repository**
-   **servicecatalog-servicecatalog_portfolio**
-   **vpc-customer_gateway**
-   **vpc-vpc**
-   **vpc-vpn_gateway**
-   **af-waf_ipset**
-   **waf-waf_rule_group**
-   **waf-waf_web_acl**

## 리소스 설정 값

-   `Region` : 접근할 AWS region
