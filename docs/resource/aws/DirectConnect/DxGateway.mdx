## 리소스 개요

Direct Connect 연결을 통해 AWS 리전의 VPC에 엑세스하도록 도와주는 DxGateway

## 연관 리소스

### 상위 리소스

-   [**Base**](https://docs.cstudio.app/docs/resource/aws/Base)

### 하위 리소스

-   [**Transit Gateway**](https://docs.cstudio.app/docs/resource/aws/Vpc/TransitGateway)
-   [**Vpn Gateway**](https://docs.cstudio.app/docs/resource/aws/Vpc/VpnGateway)


## 리소스 설정 값

-   `side_asn` : Amazon 측 ASN(자율 시스템 번호) - `64512 ~ 4294967294`
