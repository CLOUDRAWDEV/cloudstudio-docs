## 리소스 개요

AWS Direct Connect를 사용하면 표준 이더넷 광섬유 케이블을 통해 내부 네트워크를 AWS Direct Connect 위치에 연결할 수 있습니다. 케이블의 한쪽 끝을 사용자의 라우터에 연결하고 다른 쪽 끝을 AWS Direct Connect 라우터에 연결합니다. 이 연결을 통해 다음을 만들 수 있습니다.가상 인터페이스퍼블릭AWS네트워크 경로에서 인터넷 서비스 공급자를 우회하여 서비스 (예: Amazon S3) 또는 Amazon VPC로 전송합니다. 원래 요청 ping에 대한AWS Direct Connect위치는 다음에 대한 액세스를 제공합니다.AWS해당 항목이 연결된 지역에서 퍼블릭 리전의 단일 연결을 사용하거나AWS GovCloud (US)퍼블릭에 대한 액세스AWS기타 모든 퍼블릭 리전의 서비스입니다.

## 리소스 설정 값

-   `bandwidth` : Direct Connection의 Bandwidth - `50Mbps`,`100Mbps`,`200Mbps`,`300Mbps`,`400Mbps`,`500Mbps`,`1Gbps`,`2Gbps`,`5Gbps`,`10Gbps`,`100Gbps`
-   `location` : 연결이 위치한 AWS Direct Connect 위치
