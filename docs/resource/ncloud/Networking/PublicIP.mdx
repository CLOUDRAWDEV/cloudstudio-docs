## 리소스 개요

서버를 서비스 목적으로 이용하거나 생성한 서버에 접속하는데 도와주는 서비스입니다.

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/ncloud/Base/Region)

### 연결 리소스

-   [**Server**](https://docs.cstudio.app/docs/resource/ncloud/Compute/Server)

## 리소스 설정 값

-   `target_server_name` : Public IP를 적용할 server instance 이름
