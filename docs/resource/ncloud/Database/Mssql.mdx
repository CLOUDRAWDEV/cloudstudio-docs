## 리소스 개요

Cloud DB for MSSQL은 MSSQL 데이터베이스를 손쉽게 구축하고 자동으로 관리할 수 있습니다. 네이버의 최적화 설정을 통해 안정적으로 운영하며, 장애 발생 시 자동 복구하는 완전 관리형 클라우드 데이터베이스 서비스입니다.

## 리소스 설정 값

-   `enable_high_availability` : 고가용성 기능 사용 여부
-   `user_name` : MSSQL 사용자 접근 ID
-   `user_password` : MSSQL 사용자 접근 비밀번호
-   `image_product_code` : 생성할 MSSQL 인스턴트 서버 이미지 스펙에 대한 이미지 제품 번호
-   `product_code` : 생성할 MSSQL 인스턴스 서버 이미지 스펙에 대한 제품 번호
-   `data_storage_type` : 데이터 스토리지 종류  - `SSD`, `HDD`
-   `backup_file_retention_period` : 백업 파일 보관 기간, 백업은 매일 단위로 진행되며, 백업 파일은 별도의 스토리지에 저장 - `1~30`
-   `enable_automatic_backup` : 백업 자동화 사용 여부
-   `port` : MySQL 접속포트 - `3306~20000`
-   `character_set_name` : DB 문자 세트. 한글과 영문 조합 중에서 선택 가능
