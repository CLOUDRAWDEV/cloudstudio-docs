## 리소스 개요

VPC 내 리소스의 네트워크 경로를 정의하는 서비스입니다.

## 연관 리소스

### 상위 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/ncloud/Network/Vpc)

### 연결 리소스

-   [**Subnet**](https://docs.cstudio.app/docs/resource/ncloud/Network/Subnet)

## 리소스 설정 값

-   `included_vpc_name` : 연결할 VPC 이름
-   `supported_subnet_type` : 지원하는 서브넷 설정, 퍼블릭 혹은 프라이빗 - `PUBLIC`, `PRIVATE`
-   `route_table_description` : Route Table 설명
-   `linked_subnet_names` : 연결할 서브넷 이름 목록
-   `route`
    -   `target_type` : 목적지 유형, Nat gateway 혹은 VPC 피어링 - `NATGW`, `VPCPEERING`, `VGW`
    -   `target_name` : 목적지 이름
    -   `destination_cidr_block` : CIDR 형식의 목적지 IP 범위
