## 리소스 개요

VPC 간 통신을 위한 VPC 피어링을 설정하는 리소스입니다.

## 연관 리소스

### 상위 리소스

-   [**Region**](https://docs.cstudio.app/docs/resource/ncloud/Base/Region)

### 연결 리소스

-   [**Vpc**](https://docs.cstudio.app/docs/resource/ncloud/Network/Vpc)

## 리소스 설정 값

-   `linked_vpc_name_01` : 요청을 보내는 VPC 이름
-   `linked_vpc_name_02` : 요청을 보내는 VPC 이름
