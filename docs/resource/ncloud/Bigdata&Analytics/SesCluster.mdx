## 리소스 개요

elasticsearch 및 OpenSearch 클러스터를 편리하게 관리할 수 있는 환경을 제공하고, 다양한 서버 유형 및 스토리지 용량을 선택하여 클러스터를 구성하고 손쉽게 배포, 운영 및 확장할 수 있는 리소스입니다.

## 리소스 설정 값

-   `os_image_code` : 사용할 OS 타입
-   `search_engine` : Search Engine 노드 주요 항목
    -   `version_code` : 사용할 Search Engine 버전 코드
    -   `user_name` : Search Engine 사용자 이름
    -   `user_password` : Search Engine 사용자 비밀번호
    -   `dashboard_port` : Search Engine 대시보드의 포트 (1024 ~ 65535)
-   `manager_node` : manager node 주요 항목
    -   `enable_dual_manager` : manager node 이중화 여부
    -   `manager_node_product_code` : 사용할 manager node 제품 코드
    -   `manager_node_subnet_name` : manager node를 위치시킬 subnet 이름
-   `data_node` : data node 주요 항목
    -   `data_node_product_code` : 사용할 data node 제품 코드
    -   `data_node_subnet_name` : data node를 위치시킬 subnet 이름
    -   `data_node_count` : data node의 수 (3 ~10)
    -   `data_node_storage_size` : data node 스토리지 용량(100~2000)
-   `enable_master_node` : master node 사용 여부
-   `login_key_name` : 암호화를 위한 공용 로그인 키 이름
