## 리소스 개요

실시간 스트리밍 데이터 파이프라인 및 애플리케이션 빌드를 위한 오픈 소스 플랫폼인 Apache Kafka 클러스터를 손쉽게 배포, 보호, 운영 및 확장할 수 있는 서비스입니다.

## 리소스 설정 값

-   `kafka_version_code` : CDSS에서 사용할 Kafka 버전
-   `linked_config_group_name` : 사용할 CDSS config group 이름
-   `os_image` : 사용할 OS 이미지 타입
-   `cmak` : CMAK(Cluster Manager for Apache Kafka)의 구성정보 
    -   `user_name` : CMAK 접근 ID
    -   `user_password` : CMAK 접근 ID
-   `manager_node` : CMAK를 통해 Apache Kafka 클러스터를 관리하는 노드 구성 정보
    -   `manager_node_subnet_name` : 니저 노드가 위치할 subnet 이름
    -   `manager_node_product_name` : 매니저 노드 하드웨어 제품 코드
-   `broker_node` : 실제 데이터가 저장되는 서버 노드에 대한 구성 정보
    -   `broker_node_subnet_name` : 브로커 노드가 위치할 subnet 이름
    -   `broker_node_product_code` : 브로커 노드 하드웨어 제품 코드
    -   `broker_node_count` : 브로커 노드 수
    -   `broker_node_storage_size` : 브로커 노드 저장 용량 - `100~2000`
