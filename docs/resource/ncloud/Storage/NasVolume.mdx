## 리소스 개요

다수의 사용자가 데이터를 공유하도록, 여러 서버를 네트워크에 연결한 스토리지 서비스입니다. 네트워크로 연결하는 저장 공간으로 다수의 사용자가 데이터를 공유하는 데 적합합니다.

## 리소스 설정 값

-   `volume_name_postfix` : NAS 볼륨 이름, 영숫자 허용, 3 ~ 20자
-   `volume_size` : 볼륨 사이즈(100GB), 500 ~ 10,000의 범위
-   `volume_allotment_protocol_type` : 프로토콜 타입, 마운트 가능 서버: NFS-Linux/CIFS-Windows - `NFS`, `CIFS`
