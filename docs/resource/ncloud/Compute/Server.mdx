## 리소스 개요

실제 하드웨어를 관리하지 않아도 가상화의 유연성을 제공하는 컴퓨팅 리소스입니다. 클라우드 환경에 서버를 생성하여 시간과 비용 측면에서 효율적이며 인프라 운영이 부담되거나 일시적으로 많은 서버 자원이 필요한 경우에 적합합니다.

## 연관 리소스

### 상위 리소스
-   [**Subnet**](https://docs.cstudio.app/docs/resource/ncloud/Network/Subnet)

### 연결 리소스

-   [**BlockStorage**](https://docs.cstudio.app/docs/resource/ncloud/Compute/BlockStorage)
-   [**PublicIp**](https://docs.cstudio.app/docs/resource/ncloud/Compute/PublicIp)
-   [**TargetGroup**](https://docs.cstudio.app/docs/resource/ncloud/Network/TargetGroup)

## 리소스 설정 값

-   `includ_subnet_name` : 서버가 위치하는 Subnet 이름
-   `server_image_product_code` : 사용할 운영체제 이미지 코드
-   `server_product_code` : 서버의 하드웨어 구성을 위한 코드
-   `login_key_name` : 사용할 login_key 이름
-   `init_script` : 사용할 init_script 이름
-   `is_encrypted_base_block_storage_volume` : 스토리지 암호화 여부
-   `linked_network_interface_name` : 서버에 할당하는 네트워크 인터페이스
