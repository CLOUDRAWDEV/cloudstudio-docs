## 리소스 개요

서버의 기본 스토리지 외에 추가로 제공하는 HDD 또는 SSD 스토리지 서비스입니다. 단시간에 생성하여 즉시 서버와 연결할 수 있고 필요하지 않을 경우 바로 반납할 수 있습니다.

## 연관 리소스

### 상위 리소스

-   [**Subnet**](https://docs.cstudio.app/docs/resource/ncloud/Network/Subnet)

### 연결 리소스

-   [**Server**](https://docs.cstudio.app/docs/resource/ncloud/Compute/Server)

## 리소스 설정 값

-   `linked_server_name` : 블록 스토리지를 연결할 서버 이름
-   `storage_size` : 블록 스토리지의 크기(GB), 10 ~ 2000의 범위
-   `storage_disk_type` : 블록 스토리지의 디스크 종료, SSD 혹은 HDD - `SSD`, `HDD`
-   `snapshot_name` : 생성할 스냅샷 이름
