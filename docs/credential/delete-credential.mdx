크레덴셜 관리 페이지에서는 크레덴셜 정보를 보고 삭제할 수 있습니다.
등록한 크레덴셜을 삭제하려면 ```아래의 절차```를 따라주세요.

<div className={styles.imgWrap}>
    <img src={ deleteCredential } />
</div>

<br /><br />

1. ```프로필 > 크레덴셜 관리```를 클릭합니다. 
2. 설정에서 ```점 세개 아이콘```을 클릭합니다.
3. ```삭제하기```를 클릭합니다.
4. 등록했던 ```크레덴셜명을 입력```한 후 삭제하기 버튼을 클릭하여 삭제를 완료합니다.

<br />

:::tip

내가 등록한 크레덴셜을 더욱 빨리 찾기위해 크레덴셜 검색 기능을 이용하세요.

:::
