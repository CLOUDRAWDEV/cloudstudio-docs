CloudStudio는 직접 인프라를 그리는 방식뿐만 아니라, 사용자가 원하는 인프라 구조를 텍스트로 입력하면 이를 바탕으로 자동으로 인프라를 생성하는 기능도 제공합니다.

<p>&nbsp;</p>

## 텍스트로 인프라 생성

1. 텍스트로 인프라를 생성하기 위해서 `템플릿 - 프로젝트 생성하기` 버튼을 누른 후, `AI 템플릿 생성-Preview` 버튼을 클릭합니다.

    <div className={styles.imgWrap}>
        <img src={textToInfra01} width="600px" alt="배포이미지" />
    </div>

<p>&nbsp;</p>
<p>&nbsp;</p>

2. 하단 텍스트박스에 생성하기를 원하는 인프라의 정보를 입력한 후, `생성하기` 버튼을 클릭합니다.
