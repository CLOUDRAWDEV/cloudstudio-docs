Remote Inspection

Remote Inspection Permission Inquiry​

To inquire about console inspection permissions, click the ⓘ button.

How to Execute​

  1. Click the Inspection > Inspection Management > Remote Inspection tab to access remote inspection.
  2. Select the group you want to inspect from the group list.
info

Remote inspections are conducted based on the selected group. Please create a group first before starting remote inspections. How to Create a Group

  1. Click the Remote Inspection tab.

  2. Click the Start Inspection Button.

  3. Enter a inspection title (to specify what the inspection is about) and click the Next Button.

    5-1. If using an existing server, enter the information of the server you want to use.

  4. Enter a inspection title (indicating what the inspection is about), then click the Start Inspection button.

 

Inspection Result Details​

You can view detailed results for each column in the table. Click on the item you want to review.

 

Detailed Results Table
3-1. Title
3-2. Item Name
3-3. Detailed Description
3-4. Inspection Result
3-5. Inspection Method

caution

If you navigate away from the page without clicking the Save Changes button after editing the details, the changes will not be applied.

 

Report Generation​

Once the inspection is complete, you can view the results in a table. The table includes six columns: Scope, Item Name, Inspection Criteria, Priority, Inspection Result, and Inspection Result Details. For more detailed information about Inspection Criteria, refer to here. CS Checker provides two types of reports: Excel and Word.

Report​

  1. To download the report file, click the Report Button.

  2. Select the compliance and click the Generate Button.

  3. Open the Excel file to review the inspection details.
